# 1) Client One-Pager: How to Review Clips + Request Revisions (Copy/Paste)

**Clip Factory — Review & Revisions Guide (10 clips / 48 hours)**

Thanks for using Clip Factory. To keep turnaround fast and revision rounds efficient, please review the clips using the steps below.

## What you’ll receive
You’ll receive:
- **10 ready-to-post vertical clips** (9:16)
- **Burned-in captions** + clean framing
- **Platform-formatted exports** for TikTok / IG Reels / YouTube Shorts
- A **notes file** with any b-roll cues, on-screen text intent, and posting suggestions

If you ever need to verify we’re a real service, here’s our website: https://6ff73b98-dcff-4f53-a10d-9ad392f63342-00-1n4h4nuiq6w8j.picard.replit.dev/sites/3

## How to review (fastest method)
1) **Watch each clip once without stopping** (does it “feel” like a scroll-stopper?).
2) Watch again and check:
   - **Hook lands in the first ~1.5 seconds**
   - Captions are readable (mobile) and **match the spoken words**
   - No awkward jump cuts / weird pauses
   - Ending feels intentional (CTA or clean wrap)
3) Put your feedback into the revision format below.

## Revision policy (what’s included)
To protect the **48-hour delivery** promise and keep the service predictable:
- **Included:** up to **1 revision round per clip** (within scope) if requested within the revision window.
- **Revision window:** request revisions within **72 hours of delivery**.
- **In-scope revisions (included):**
  - Caption text corrections (typos, missed words)
  - Caption styling tweaks (size/position within the preset)
  - Trim 0.5–2 seconds for pacing
  - Swap hook line from the same moment (where possible)
  - Light reframing adjustments (keep speaker centered)
- **Out-of-scope (new work):**
  - New clip selection from totally different timestamps not previously considered
  - Full re-edit with new story arc
  - Custom motion graphics packages beyond the chosen preset
  - Heavy manual b-roll sourcing

If you need out-of-scope changes, email us and we’ll quote it before doing any work.

# 4) Lightweight Lead Tracker Schema (for daily distribution)

Use a Google Sheet/Airtable/Notion table with these fields:

## Required fields
- **Lead ID:** POD-0001 / COACH-0001
- **Segment:** Podcaster / Coach
- **Name:** First + Last
- **Brand/Show:** Podcast/Business name
- **Role:** Host / Founder / Coach
- **Email:**
- **Secondary channel:** LinkedIn URL / IG handle
- **Source:** YouTube / LinkedIn / Spotify / Website
- **Last content link:** YouTube episode / webinar
- **Personalization note:** Guest name, topic, quote, recent post

## Pipeline stages
- **Prospect** (identified)
- **Contacted** (email sent)
- **Replied**
- **Qualified** (has long-form + wants clips)
- **Timestamp picks sent**
- **Closed won** (paid)
- **Closed lost** / **No response**

## Activity tracking
- **Date contacted**
- **Follow-up #1 date**
- **Follow-up #2 date**
- **Next action**
- **Notes** (objections, preferences, style)

This schema keeps outreach consistent and makes it easy to hand off to a VA/SDR later.
