## Purpose
Deliver a high-quality vertical short (9:16) from a long-form video/podcast within 24 hours of receiving assets. This is designed to (1) prove quality fast, (2) shorten time-to-value, and (3) convert prospects into DFY clip packages that fund the later SaaS build (revenue gate: $5k).

## Offer Definition (Sample Clip)
**Deliverable (within 24 hours):**
- 1 vertical clip (20–45s recommended)
- Burned-in captions (clean, high-contrast)
- Strong hook headline (on-screen + first caption line)
- Basic formatting for TikTok/Reels/Shorts
- 1 alternate hook text option (A/B)

**What’s not included (to avoid scope creep):** custom motion branding package, complex b-roll sourcing/licensing, full episode clipping, multi-language captions.

## Tools (Free/low-cost; use what’s available)
- Transcript: Whisper (local or API if already available later; for now manual + free tooling acceptable)
- Editing: CapCut Desktop / DaVinci Resolve / Premiere (whatever is available)
- Captions: CapCut auto-captions + manual corrections
- Delivery: Google Drive / Dropbox link + email

## Intake Requirements (Client Asset Checklist)
Send this as soon as they say yes.

**Client sends:**
1) Long-form link: YouTube / Spotify video / Google Drive / Riverside
2) Episode title + target audience (1 sentence)
3) CTA to include (choose one):
   - Follow for more
   - Book a call
   - Download lead magnet
   - Comment keyword
4) Style preferences (pick 1–2):
   - Clean podcast captions
   - Bold hook + fast captions
   - Storytime / meme-y
   - Corporate clean
5) Brand kit (optional for sample): logo, brand colors, font (or “use default clean style”)
6) Any “must-include” timestamps (optional): up to 3

**If client is slow:** proceed with best-guess defaults (clean captions, bold hook, no logo) to preserve 24h promise.

## SOP: 24-Hour Timeline
### T+0 to T+30 min — Setup
- Confirm you have a working source link.
- Save source to /01_Source/ (download if possible).
- Create a notes doc: episode summary + potential hooks.

### T+30 to T+2.5 hrs — Find the moment
Pick 1 segment using these criteria:
- Clear value in first 1–2 seconds (strong claim, contrast, or curiosity)
- Standalone coherence (no heavy context needed)
- Emotional punch: surprise, disagreement, vulnerability, tactical list
- Avoid long preambles and inside jokes

Heuristics to spot highlights quickly:
- “Most people think X… but actually Y”
- “Here’s the mistake…”
- “Do this instead…”
- “I used to… until…”
- “3 steps…”

### T+2.5 to T+6 hrs — Edit (9:16)
- Create 9:16 sequence (1080x1920).
- Frame speaker (center, eyes ~ top third).
- Remove dead air; tighten pacing.
- Add dynamic zooms (subtle: 105–115%) on emphasis.
- Add hook text overlay (5–8 words max). Provide 2 options.
- Optional: add a simple progress bar or underline emphasis (keep minimal for sample).

### T+6 to T+10 hrs — Captions
- Generate auto-captions.
- Manually correct proper nouns and numbers.
- Caption style default:
  - White text, black shadow or box
  - 2 lines max
  - Emphasize 1–2 key words per sentence (bold/color) if tool supports

### T+10 to T+20 hrs — QA + Export
QA checklist:
- First 2 seconds: hook is clear + readable
- Captions: no major errors, not covering mouth
- Audio: balanced, no clipping
- No awkward jump cuts
- Safe areas: not too close to edges for TikTok UI

Export settings (typical):
- H.264, 1080x1920, 30fps, 10–20 Mbps

### T+20 to T+24 hrs — Deliver + Next Step
- Upload export to Drive.
- Send delivery email with:
  - Link
  - 2 hook options
  - Posting suggestion (caption + CTA)
  - Upsell to package (e.g., “Want 8 more like this from the same episode?”)
