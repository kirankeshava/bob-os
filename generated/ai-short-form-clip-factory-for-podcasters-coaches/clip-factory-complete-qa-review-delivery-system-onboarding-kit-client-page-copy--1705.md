# Clip Factory — What to Expect (Client-Facing Page Copy)

**Official site:** https://6ff73b98-dcff-4f53-a10d-9ad392f63342-00-1n4h4nuiq6w8j.picard.replit.dev/sites/3  
**Support:** agent_bob_replit+clip-factory@agentmail.to

## 1) What you’ll receive (Delivery Pack)
For each batch we deliver a structured folder containing:
1. **Final vertical clips (MP4)** — ready to post.
2. **Captions file (SRT)** — for native platform uploads.
3. **B-roll / on-screen cue sheet (Google Doc or PDF)** — time-coded suggestions for overlays, b-roll, emojis, titles, and callouts.
4. **Posting suggestions (text file or doc)** — recommended first-line caption + CTA + hashtag set per clip.

## 2) Quality Standards (QA checklist we run on every clip)
We QC every clip before it reaches you:
- **Safe zones / framing:** faces not cropped; key text stays within TikTok/IG “safe zones” (no UI overlap). Export is **9:16**.
- **Captions accuracy:** punctuation and speaker intent preserved; no missing/duplicated lines; profanity handled per your preferences.
- **Readability:** caption size/contrast works on mobile; highlights don’t obscure faces.
- **Audio leveling:** consistent loudness across clips; voice is clear; background noise reduced without artifacts.
- **Jump cuts & pacing:** cuts are intentional; no jarring mid-word cuts; dead air removed.
- **Hook clarity:** first 1–2 seconds contains a clear hook (text + spoken context).
- **Brand compliance:** your handles/CTA/links, tone, and banned words respected.
- **Export settings:** H.264 MP4, 1080x1920, 30fps (unless specified), high bitrate, clean audio.

## 3) Review process (timestamp feedback)
You’ll receive a **review link** (Frame.io / Vimeo / unlisted YouTube) or a **Drive folder** depending on your preference.

**How to request revisions:**
- Leave **timestamped comments** (example: “00:12 change caption wording” or “00:18 remove ‘um’”).
- Group feedback per clip when possible.

## 4) Revision window
To keep turnaround fast, we include a **24-hour revision window** after delivery.
- If you reply with timestamped notes inside 24 hours, we prioritize revisions.
- If we don’t hear back, we’ll mark the batch as approved and close the delivery.

## 5) What counts as a free revision vs a change request
**Free revisions (included):**
- Caption typo fixes; minor punctuation/wording corrections.
- Timing shifts for captions (within the same spoken line).
- Small audio level tweaks.
- Minor crop/reframe adjustments.
- Removing a short section that was clearly unintended (e.g., dead air, stutter).

**Change requests (out of scope / scheduled separately):**
- Switching topics, rewriting hooks for a different angle after delivery.
- Major structural re-edit, new story arc, different clip selections.
- New branding package after editing begins (fonts/colors/style overhaul).
- Adding new footage, new b-roll libraries, or new music direction beyond original notes.

If something is ambiguous, we’ll propose the fastest option that preserves your intent and turnaround.

# Internal QA Checklist (Operator Use)
Use this before any clip leaves the factory.

## A) Project setup
- [ ] Correct client folder + correct batch name
- [ ] Correct platform target (TikTok / IG Reels / YT Shorts)
- [ ] Correct brand kit applied (fonts/colors/logo/handles)
- [ ] Confirm banned words / topics

## B) Visual framing + safe zones
- [ ] 9:16 (1080x1920)
- [ ] Subject face never cropped; eyes not at extreme top
- [ ] No critical text under UI (top header, right-side icons, bottom caption areas)
- [ ] Any emojis/overlays don’t cover mouth/eyes

## C) Captions QA
- [ ] No missing words vs audio
- [ ] No duplicated lines
- [ ] Punctuation supports meaning
- [ ] Speaker labels correct (if used)
- [ ] Profanity policy followed (bleep/asterisk/omit)
- [ ] Captions don’t exceed readable speed (avoid walls of text)

## D) Audio QA
- [ ] Voice intelligible on phone speaker
- [ ] No pumping/metallic artifacts from noise reduction
- [ ] Loudness consistent across clips in the batch
- [ ] No peak clipping

## E) Edit/pacing QA
- [ ] Hook in first 1–2 seconds
- [ ] Cuts not mid-word
- [ ] No dead air > ~0.3–0.5s unless intentional
- [ ] B-roll cues match what’s said

## F) Export QA
- [ ] MP4 H.264, 1080x1920, 30fps (unless specified)
- [ ] File naming correct
- [ ] Plays on mobile, no black bars

# Revision Triage Rules (Operator Decision Tree)

**If request is within 24 hours AND is one of:** typo, caption timing, minor crop, minor audio, small removal → **Free revision**.

**If request changes the premise/angle/selection OR requires re-cutting the clip selection OR new brand kit after start → Change request** (schedule separately; in Week 1 free launch, we can offer one courtesy change per client but document it as “one-time courtesy”).

**If request is caused by our error** (missed safe zones, wrong handle, caption wrong vs audio) → **Always fix ASAP, priority**.
