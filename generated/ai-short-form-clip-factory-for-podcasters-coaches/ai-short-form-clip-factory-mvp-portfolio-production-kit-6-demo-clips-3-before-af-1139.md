1) Source selection (permission-friendly) + verification checklist
Goal: avoid rights risk while still demonstrating editing quality.
Use one longform video per niche, from sources that are (a) Creative Commons licensed on YouTube OR (b) explicitly permission-friendly (creator allows reuse) OR (c) public-domain/CC assets.
Verification checklist (do this before download):
- Confirm YouTube license shows “Creative Commons — Attribution” (preferred) OR the channel/site provides explicit reuse permission.
- Save evidence: screenshot the license/permission text + URL in a “00_Source-Proof” folder.
- Use only 30–60s excerpts; label all posts “Sample edit / demo” (not implying endorsement).

2) The 6 demo clips (2 per niche) — concepts + on-screen copy + edit direction
Export specs for every clip:
- Format: 9:16 vertical, 1080x1920
- Length: 25–45s (target), max 60s
- Captions: burned-in, high contrast, 2-line max, keyword emphasis
- Cuts: remove silences, add punch-ins (105–120%), add pattern breaks every 1.5–3.0s
- Audio: normalize speech; light music bed at -25 to -18 LUFS under voice
- End card (last 1.0s): “Need clips like this? https://6ff73b98-dcff-4f53-a10d-9ad392f63342-00-1n4h4nuiq6w8j.picard.replit.dev/sites/3 | agent_bob_replit+clip-factory@agentmail.to”
