Goal: Create a proof-heavy portfolio for the AI Short-Form Clip Factory that closes prospects fast. Output required: 12 vertical demo clips (4 per niche across Real Estate, Fitness, B2B) + 3 before/after comparisons (raw excerpt → polished clip). This document contains (1) sourcing rules to stay permission-friendly, (2) clip briefs you can execute immediately, (3) a consistent formatting spec, (4) before/after templates, and (5) Google Drive + Carrd organization.

A) Permission-friendly sourcing rules (non-negotiable)
1) Use Creative Commons / public-domain longform only unless you record your own.
2) For YouTube: filter by “Creative Commons” license. Save the URL and a screenshot of the license area for audit.
3) Keep each demo clip clearly labeled “Sample edit using Creative Commons source” in the caption/description.
4) Avoid using copyrighted music; use platform library music only (or no music) for the demo exports.

B) Universal clip formatting spec (apply to all 12)
- Canvas: 1080x1920 (9:16), 30fps.
- Safe margins: keep captions centered, 120px from bottom, 80px from sides.
- Caption style: 2-line max, high-contrast (white text + black stroke). Highlight 1-3 keywords per line in yellow.
- Pacing: cuts every 0.7–1.5s; add punch-in zooms on emphasis words.
- Hook window: first 1.0–1.5s must contain a bold on-screen hook.
- Visual layers: speaker video + captions + occasional b-roll overlay + simple progress bar/top label (“REAL ESTATE”, “FITNESS”, “B2B”).
- End CTA: 1.5s card: “Want 10 clips/week? DM ‘CLIPS’ • agent_bob_replit+clip-factory@agentmail.to • https://6ff73b98-dcff-4f53-a10d-9ad392f63342-00-1n4h4nuiq6w8j.picard.replit.dev/sites/3”

C) 12 clip briefs (4 per niche)
Each brief includes: style, hook line, structure, captions, b-roll cues, and CTA.

NICHE 1: REAL ESTATE (4 clips)
RE-1 Storytelling Hook (20–35s)
- Hook (on-screen): “This is why your listing isn’t getting showings…”
- Spoken structure: Problem → micro-story (a listing that stalled) → 1 fix → result.
- Caption beats:
  1) “Your listing isn’t getting showings because…”
  2) “Buyers decide in 3 seconds.”
  3) “Fix: lead with the *one* photo that sells the lifestyle.”
- B-roll cues: exterior hero shot, neighborhood vibe, kitchen wide shot, “before” listing photo grid.
- Edit notes: punch-in on “3 seconds”; add quick 3-photo montage.

RE-2 Listicle (25–40s)
- Hook: “3 upgrades that raise perceived value instantly (cheap).”
- Structure: Numbered list 1–3 with rapid examples.
- Captions:
  - “#1 Lighting > fixtures”
  - “#2 Paint: warm neutral only”
  - “#3 Curb appeal: mulch + clean lines”
- B-roll: closeups of lighting, paint swatches, curb/landscaping.
- Edit notes: use large numeric overlays; 0.3s swoosh transitions.

RE-3 Contrarian Take (20–30s)
- Hook: “Stop saying ‘open concept’ in your listing.”
- Structure: Claim → why it’s generic → what to say instead.
- Captions:
  1) “Hot take: ‘Open concept’ is meaningless.”
  2) “Say THIS instead:”
  3) “ ‘Entertain 12 comfortably + sightlines to the backyard.’ ”
- B-roll: dining table setup, living room flow, backyard door opening.
- Edit notes: add quotation pop-up for the replacement phrase.

RE-4 Testimonial Style (20–35s)
- Hook: “My client almost over-renovated… here’s what we did.”
- Structure: Client fear → agent guidance → outcome.
- Captions:
  - “They wanted to spend $25k.”
  - “We spent $2,500 instead.”
  - “Offer came in week 1.”
- B-roll: receipts/estimate graphic (mock), staged room shots, offer email blur (mock).
- Edit notes: add simple animated number counter (25,000 → 2,500).

NICHE 2: FITNESS (4 clips)
FT-1 Storytelling Hook (20–35s)
- Hook: “I trained wrong for 2 years… and here’s the fix.”
- Structure: Mistake → symptom → correction → how to apply today.
- Captions:
  - “I wasn’t ‘lazy’… I was under-recovering.”
  - “Fix: 2 hard days, 1 easy day.”
  - “Progress skyrocketed.”
- B-roll: gym warmup, timer, sleep/recovery visuals.
- Edit notes: heartbeat SFX (optional, royalty-free), punch-ins on “2 hard days”.

FT-2 Listicle (25–40s)

- Captions:

  - “#2 Ribs down”

- B-roll: squat form angles, foot tripod graphic, ribcage overlay.
- Edit notes: freeze-frame + arrows for 0.8s per cue.

FT-3 Contrarian Take (15–25s)
- Hook: “Cardio isn’t the best way to lose fat.”
- Structure: Contrarian claim → clarify (calorie deficit + steps + lifting) → simple plan.
- Captions:
  - “Fat loss: diet + steps + strength.”
  - “Cardio is optional.”
  - “Start with 8k steps/day.”
- B-roll: steps counter, meal prep, dumbbells.
- Edit notes: add small disclaimer text “Not medical advice.”

FT-4 Testimonial Style (20–35s)
- Hook: “Client dropped 10lbs without cutting carbs.”
- Structure: Starting point → one change → outcome.
- Captions:
  - “We changed: protein + consistency.”
  - “No extreme cuts.”
  - “10lbs down.”
- B-roll: grocery haul, protein plate, progress chart (mock).
- Edit notes: chart animation 0→10.
