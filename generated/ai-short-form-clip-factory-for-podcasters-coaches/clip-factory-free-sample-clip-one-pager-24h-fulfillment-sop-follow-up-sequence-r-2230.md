## 2) 24‑Hour Sample Clip Fulfillment SOP (internal)

**Goal:** Deliver one high-quality vertical clip fast, with minimal back-and-forth, to convert into DFY package.

### Intake checklist (from prospect)
1) Long-form source link (YouTube/Drive)
2) Platform target (Reels/TikTok/Shorts)
3) Niche + audience + offer/CTA (if any)
4) Tone preference (energetic, calm, contrarian, etc.)
5) Brand constraints (words to avoid, colors if known)

If missing: reply with a single email asking only for the missing items.

### Production steps (time-boxed)
**0. File capture (10 min)**
- Download source (or request permission if Drive restricted)
- Save as: `Client_EpisodeDate_SourceTitle.mp4`

**1. Find 1 highlight (20–30 min)**
Pick the moment with:
- a clear promise, takeaway, or contrarian point
- minimal crosstalk
- standalone context in first 2–3 seconds

### Highlight scoring (quick rubric)
- Clarity (0–3)
- Emotion/energy (0–3)
- Utility (0–3)
- Standalone context (0–3)
Choose the highest total.

**2. Build the 9:16 cut (20–40 min)**
- Target 15–45s
- Use punch-ins every 2–4 seconds if visually static
- Remove dead air >300ms where safe

**3. Captions (15–25 min)**
- Burned-in, mobile-safe margins
- 2 lines max on screen
- Emphasize 1–3 keywords (ALL CAPS or color)
- Ensure readability against busy backgrounds (shadow/box)

**4. Hooks + packaging (10–15 min)**
Deliver:
- Hook A/B/C (1 sentence each)
- Suggested on-screen title (5–8 words)
- Post caption (1–2 sentences) + 3–5 hashtags (optional)
- 3–5 b-roll cues as bullets

**5. Export + delivery (10 min)**
- Export 1080x1920, H.264
- Filename: `Client_FREE-SAMPLE_Clip01_9x16.mp4`
- Deliver via Drive link + brief email.
