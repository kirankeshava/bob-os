0) Outcome + Promise
==============================
Deliverable: 10 ready-to-post vertical clips per episode/video within 48 hours of receiving complete assets.
Each clip includes: hook-first cut, clean jump cuts, branded captions, basic b-roll cues (where relevant), platform-safe formatting, and export presets for TikTok / Reels / Shorts.
Priority: speed + consistency over perfection.

==============================
1) Tool Stack (speed-first + fallbacks)
==============================
Primary (fastest path):
- OpusClip (AI clip selection + auto captions + reframing)
- CapCut Desktop (final polish: caption template, b-roll inserts, end card, audio leveling)
- Whisper (optional transcription if source audio is difficult; can be via local Whisper / MacWhisper / any Whisper-based tool)
- Review + delivery: Frame.io (best) OR Google Drive folder + timestamped notes

Fallbacks:
- If OpusClip fails: Descript (find highlights + transcript-based editing) OR manual selects using transcript + timecodes
- If CapCut fails: Premiere / Resolve / Descript templates
- If Frame.io unavailable: Drive + “Feedback via timestamps” Google Doc

Day 0 (Hour 0–2): Intake validation + asset download + project setup

==============================
6) Highlight Selection Rules (to consistently produce 10 clips)
==============================
Goal: 10 clips, typically 20–45 seconds each (some niches can run 45–70s if value holds).

Selection criteria (must hit 3/5 minimum):
1) Clear hook potential within first 1.5 seconds (surprising claim, “most people get this wrong,” direct benefit, contrarian take).
2) Standalone context (viewer can understand without full episode).
3) One core idea per clip (avoid multi-topic rambles).
4) High energy / emotional contrast / crisp phrasing.
5) Practical takeaway (step, framework, number, mistake to avoid).

Hard rules:
- First 1.5s must include motion + meaning (no slow intros, no “today we’re going to…”).
- Pattern interrupt every 2–3 seconds (jump cut, zoom, b-roll, on-screen keyword, angle shift, sound cue).
- Remove filler: “um,” “like,” long breaths, repeated phrases.

Clip types to mix (for variety):
- Mistake/contrarian
- Step-by-step framework
- “If you’re X, do Y” advice
- Personal story with lesson
- Myth vs reality

==============================
7) Caption Rules + Templates (global)
==============================
Captions are mandatory.
- Style: bold, high-contrast, 2 lines max, ~5–9 words per line.
- Timing: snap to speech; keep readable; avoid over-highlighting every word if it reduces clarity.
- Emphasis: highlight 1–3 keywords per sentence (color or weight).
- Safe areas: keep captions above bottom UI zones for TikTok/IG; avoid extreme edges.
- Profanity: follow client preference; default: keep but censor if brand-sensitive.

Hook line treatment:
- First frame shows first keyword immediately (no fade-in delay).
- Optional top label: “DO THIS:” “STOP DOING THIS:” “3 STEPS” (niche dependent).

==============================
8) B-Roll Cues (speed-first)
==============================
Default approach: minimal b-roll, maximum clarity.
Add b-roll only when it creates a clear pattern interrupt or clarifies an example.

B-roll cue insertion rules:
- Insert 1 cue every ~6–10 seconds OR whenever energy drops.
- Use simple overlays: screenshots, charts, emojis/icons only if niche-appropriate, quick cutaways.
- If no b-roll is available: use motion effects (punch-in zoom 3–6%, quick jump cuts) + on-screen keywords.

Cue format in project notes (if needed):
[BROLL 00:12] overlay: “Zillow screenshot” / “before-after” / “calendar” / “email inbox”

==============================
9) Export Presets (platform-specific)
==============================
Core format:
- 9:16 vertical, 1080x1920
- 30 fps
- H.264, high bitrate (10–20 Mbps)
- Audio AAC 320 kbps (or 192 kbps), normalized.

TikTok:
- Keep captions mid-lower but above UI
- Slightly larger text

Instagram Reels:
- Similar to TikTok; ensure no critical text in bottom 15%

YouTube Shorts:
- Avoid too-low captions; keep more center-lower
- Prefer cleaner look, fewer stickers

File outputs:
- Deliver .mp4 files per platform OR one master if client requests.

==============================
10) QA Checklist (fast, consistent)
==============================
Content:
- Hook lands in first 1.5s
- One idea per clip; no confusing missing context
- No compliance red flags (medical/financial/housing claims)

Technical:
- 9:16 framing correct; face not cropped
- Captions readable, not blocked by UI
- Audio clear; no clipping; music (if used) low enough under voice
- No black frames; no abrupt cut-offs

Brand:
- Correct preset (font/color/position)
- CTA end card present (if included in scope)

12) Style Preset Packs (3 niches)
==============================
A) Real Estate Preset (clean, premium)
- Font: Montserrat Bold (captions), Montserrat SemiBold (labels)
- Colors: White text; accent #00B2FF (blue) for keywords; subtle shadow
- Caption position: lower-third but above UI; max 2 lines
- Motion: gentle punch-in on key statements; cutaway b-roll to listings/neighborhoods
- CTA end card (last 1.0–1.5s):
  “Want the exact checklist? Follow @___ / DM ‘HOME’” + small logo bottom-right

B) Fitness Preset (high energy)
- Font: Anton (big headers) + Inter Bold (captions)
- Colors: White text; accent #FF3B30 (red) or #00E676 (green) for keywords
- Caption position: mid-lower (slightly higher than others), big text
- Motion: faster jump cuts; speed ramps optional; on-screen numbers for steps
- CTA end card: “Save this • Try it today • Follow for workouts”

C) B2B Preset (minimal, trust-first)
- Font: Inter Bold (captions), Inter Medium (sub)
- Colors: White text; accent #7C3AED (purple) or #22C55E (green) for keywords
- Caption position: center-lower, more whitespace
- Motion: minimal; occasional clean punch-in; simple icon overlays
- CTA end card: “Steal this framework • Subscribe / Follow • Link in bio”

==============================
14) SLA + Revision Policy (speed-protecting)
==============================
Turnaround SLA:
- 10 clips delivered within 48 hours AFTER complete assets received.

Included revisions:
- 1 revision round included (batch feedback across all clips).
- Revisions limited to: caption fixes, small trims, subtitle positioning, light b-roll swaps, CTA text swap.

Not included (new scope):
- Re-cutting new clips beyond the 10 delivered
- Heavy content rewrites, new music direction per clip, advanced motion graphics

Feedback deadline (protects SLA):
- Client must submit revision notes within 24 hours of delivery to keep fast turnaround.

==============================
15) Failure-Mode Playbook (what to do when things break)
==============================
- OpusClip outage/poor selects → switch to Descript transcript search; manually pick 10 moments; assemble in CapCut.
- Bad audio → run basic cleanup (noise reduction, EQ). If unintelligible, ask client for better source; proceed with fewer clips if needed and document exception.
- Slow uploads/huge files → request YouTube link or smaller export; start transcript + selection while download runs.
- Missing brand kit → default to closest niche preset; confirm during revision round.
