H2: 1) What you’ll get (Delivery Pack)
Bullets:
- MP4 vertical clips (platform-ready)
- Captions burned-in (for immediate posting)
- Caption files (.SRT) for accessibility/editing
- B-roll cue sheet (timestamped suggestions)
- Posting suggestions (first-line caption ideas + hashtags + CTA options)

H2: 2) Our QA standards (how we prevent “editor problems”)
Checklist bullets:
- Safe zones respected (no UI-covered text; top/bottom margins for TikTok/Reels)
- Captions accuracy (names/terms checked; punctuation cleaned)
- Readability (line breaks, max 2 lines, consistent style)
- Audio leveling (voice clear; no clipping; background reduced)
- Jump cuts + pacing (remove dead air; keep momentum)
- Hook clarity in first 1–2 seconds
- Faces not cropped awkwardly; subject framed for vertical
- Export settings: 1080x1920, H.264 MP4, 30fps (or match source), high bitrate

H2: 3) Review process (timestamped comments)
Copy:
“We’ll send you a single review link (Frame.io / Vimeo / unlisted YouTube) where you can leave timestamped comments. Please keep feedback consolidated in that link so we can turn revisions fast.”

H3: Timing
- First delivery: per agreed schedule
- Revision window: 24 hours after delivery link is sent

H2: 4) Revisions (what’s free vs what changes scope)

H3: Free revisions (included within 24h window)
- Caption typo fixes
- Small timing tweaks (caption sync, minor cut trims)
- Audio tweaks (slight volume balance, noise reduction adjustments)
- Safe-zone/framing corrections (if anything risks being covered/cropped)

H3: Not included (scope changes)
- Changing the hook concept after delivery (new angle/topic)
- New clips not requested originally / extra deliverables
- Replacing core sections with entirely different source timestamps
- Brand/creative direction changes not provided in the intake
Note:
“If you request changes in the ‘Not included’ category, we’ll confirm the new scope before doing additional work.”

Goal: Standardized intake that prevents avoidable revisions.

1) Create
- Google account: use agent_bob_replit@agentmail.to if possible

Title: “Clip Factory — Client Intake Form”
Description:
“Fill this out before we start so we can match your platform, voice, and CTA. Questions? agent_bob_replit+clip-factory@agentmail.to”

2) Form settings
- Collect email addresses: ON
- Limit to 1 response: OFF
- Response receipts: ON
- Allow response editing: OFF
- Progress bar: ON

3) Sections + questions (exact)

SECTION 1 — Basics
Q1 (Short answer, Required): Project / Brand name
Q2 (Short answer, Required): Primary contact email
Q3 (Short answer, Optional): Instagram/TikTok/YouTube handle(s)

SECTION 2 — Source content
Q4 (Multiple choice, Required): Source type
Options: Podcast (audio), Podcast (video), Coaching call, Webinar, YouTube video, Other
Q5 (Short answer, Required): Link to source file (Drive/Dropbox/YouTube/etc.)
Q6 (Paragraph, Optional): Best timestamps / moments to prioritize (if you have them)

SECTION 3 — Platforms + deliverables
Q7 (Checkboxes, Required): Platforms
Options: TikTok, Instagram Reels, YouTube Shorts, LinkedIn (vertical)
Q8 (Multiple choice, Required): Caption style
Options: Clean (minimal), Bold (high emphasis), Word-by-word (faster pace)
Q9 (Multiple choice, Required): On-screen text style
Options: Minimal hooks only, Hook + key points, Heavy text (lots of punchlines)

SECTION 4 — Voice, audience, CTA
Q10 (Paragraph, Required): Describe your audience in 1–2 sentences
Q11 (Multiple choice, Required): Tone
Options: Educational, Motivational, Direct/No-fluff, Funny, Storytelling, Other
Q12 (Paragraph, Required): Primary CTA (what should viewers do?)
Example: “Follow for more,” “Book a call,” “Download lead magnet,” “Comment ‘X’.”

SECTION 5 — Constraints (prevents rework)
Q13 (Checkboxes, Optional): Hard no’s / banned topics
Options: Profanity, Politics, Medical claims, Income claims, Competitor mentions, Other
Q14 (Paragraph, Optional): Banned words/phrases (exact)
Q15 (Checkboxes, Optional): Compliance needs
Options: Add disclaimer, Avoid medical advice tone, Avoid financial guarantees, Other

SECTION 6 — Brand & assets
Q16 (File upload if available; if not, Short answer link, Optional): Logo/brand kit link
Q17 (Short answer, Optional): Website / landing page URL
