GOAL
Deliver 10 ready-to-post 9:16 clips from one long-form source (podcast/YouTube/interview), each with a strong hook, burned-in captions, basic b-roll cues, and platform-ready exports in 24 hours from asset receipt.

SCOPE (DEFAULT PACKAGE)
Input: 1 long video link (YouTube/Vimeo/Drive) up to 60 minutes.
Output: 10 vertical clips (15–45s each), 1080x1920, .mp4 H.264, hardcoded captions, optional title card, and a text doc containing: clip titles/hooks + suggested post copy + hashtags (optional) + b-roll cues.

TOOLS (NO-SPEND DEFAULT)
Editing: CapCut Desktop or DaVinci Resolve (free).
Captions: CapCut auto-captions (fast) + manual correction pass.
Asset transfer: Google Drive/Dropbox link from client; deliver via Drive folder.
Tracking: Google Sheet row per order (Client, Link, Deadline, Status, Notes, Revision count).

INTAKE CHECKLIST (REQUIRED TO START TIMER)
Client must provide:
1) Source link or file access (and permission).
2) Platform priority (TikTok/Reels/Shorts) and target audience.
3) Brand kit: colors/fonts (if any) OR choose “Clean Default.”
4) Caption style: (A) Minimal (B) Word-by-word highlight (C) Karaoke (avoid unless requested).
5) CTA + offer link (if any).
6) “Avoid list” (topics/phrases).
If missing: reply requesting missing items; 24-hour clock starts after receipt.

PRODUCTION SOP (TIMEBOXED)
T+0:00–0:20 — Source capture & setup
- Download source (if needed) or open link.
- Confirm: audio quality, speaker count, topic, and any “avoid list.”
- Create project folder + tracking entry.

T+0:20–1:20 — Find 20–30 candidate moments
- Skim at 1.5–2x speed.
- Mark timestamps for: contrarian takes, tactical steps, mistakes to avoid, story pivots, strong one-liners, “here’s the framework,” numbers/results.
- Write candidate list in /03_Notes/Candidates.txt with timestamps + 1-line hook.

T+1:20–2:00 — Select the 10 clips
Selection criteria (must hit at least 2):
- Clear standalone context within 3 seconds.
- Strong emotion (surprise, certainty, humor) or clear value (steps, list, framework).
- Specificity (numbers, examples, “do X, not Y”).
- “Loop close”: ending resolves the hook.

T+2:00–5:30 — Edit clips (assembly + vertical framing)
For each clip:
1) Set 9:16 sequence.
2) Auto reframe subject; keep eyes in top third.
3) Tighten dead air; keep pacing high.
4) Add opening hook text (5–9 words max) if the spoken hook is slow.
5) Optional: subtle zoom cuts every 3–6 seconds if visually static.

T+5:30–7:30 — Captions pass (accuracy + styling)
- Generate auto-captions.
- Correct names, jargon, numbers, and negations (critical).
- Style defaults:
  • Font: clean bold sans (CapCut default bold ok)
  • Size: readable on phone
  • High-contrast: white text + dark shadow/outline
  • Highlight: 1–3 key words per line (only if requested)

T+7:30–8:30 — Add b-roll cues (lightweight)
- Do NOT over-edit. Add cues in a notes doc unless client paid for full b-roll insertion.
- For each clip, write 1–3 b-roll suggestions (e.g., “Overlay screenshot of calendar,” “Cut to chart graphic,” “Show app UI”).
- Save to /03_Notes/BrollCues.txt.

T+8:30–9:30 — Export settings + batch export
- Export each clip:
  • 1080x1920
  • 30fps
  • H.264
  • 10–16 Mbps
  • AAC audio 320 kbps
- Quick playback check for caption timing + audio pops.

T+9:30–10:30 — Write hooks/titles + post copy
- For each clip: write 3 hook options + 1 recommended title.
- Write 1 short post caption (1–2 lines) + CTA.
- Save to /03_Notes/ClipIndex.txt in this format:
  Clip01 – Title – Hook – Timestamp – Suggested caption – Hashtags (optional)

T+10:30–11:00 — QC checklist + packaging
QC checklist (must pass):
- Audio is clear, no clipping; loudness roughly consistent across clips.
- First 1.5 seconds has a hook (spoken or on-screen text).
- Captions are accurate (esp. numbers, names, “not”).
- Subject framing: face not cut off; no constant hunting.
- No watermark; no black bars.
- Length: 15–45 seconds unless client specified.
- File names follow standard.

T+11:00–24:00 — Buffer + revisions window
- Deliver by T+12:00 when possible; reserve buffer for 1 revision round.

REVISION POLICY (SCOPE CONTROL)
Included: 1 revision round for up to 3 clips (caption fixes, trims, hook text swap, reframing).
Not included (new scope): changing clip selection after delivery, adding heavy b-roll/stock footage, full brand motion package, recreating from a new source.
Turnaround on included revisions: 24 hours from client notes.
