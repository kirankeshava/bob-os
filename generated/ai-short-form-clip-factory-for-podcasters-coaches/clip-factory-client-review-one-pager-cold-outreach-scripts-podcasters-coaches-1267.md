How to request revisions (required format)
To keep turnaround fast, please send feedback in this exact structure:
A) Clip file name (example: EP12_CL03_TikTok_v1.mp4)
B) Timestamp (start–end) (example: 00:07–00:10)
C) Change request (one sentence)
D) If wording changes: paste exact replacement text

Examples of “fast” revision requests
- EP12_CL03_TikTok_v1.mp4 | 00:07–00:10 | Remove the pause; jump cut tighter.
- EP12_CL06_Reels_v1.mp4 | 00:02–00:04 | Caption line 1: change to “Stop doing this in your warm-up”.
- EP12_CL01_Shorts_v1.mp4 | 00:18–00:22 | Swap b-roll cue: use “laptop/analytics” instead of “city skyline”.

What counts as a revision (included)
- Caption wording fixes and timing adjustments
- Minor pacing tweaks (tighten pauses, trim filler)
- Hook option swap (if we provided alternates)
- Reposition captions to avoid UI/face blocking

What counts as a new request (not a revision)
- Changing the chosen clip topic/angle entirely
- New b-roll concept across the full clip
- New music direction across all clips
- Requesting additional clips beyond the package
