## 3) 24-Hour Free Sample Clip Fulfillment SOP (internal)

**Goal:** Deliver 1 polished vertical clip in <24h to convert into a paid weekly package.

### Inputs required (minimum)
- Source link (YouTube/Spotify/Drive)
- Target platform(s): TikTok / Reels / Shorts
- Niche + audience (1 sentence)
- CTA (follow / book call / download lead magnet)
- Brand preferences (optional): colors/fonts, tone, examples

### Tools (free/low-cost)
- Download: yt-dlp or browser download if permitted
- Transcription: Whisper (local if available) or existing transcript
- Editing: CapCut / DaVinci Resolve (free)
- Captions: CapCut auto-captions + manual correction

### Process (time-boxed)
1) **Ingest (10 min):** Save source, confirm frame rate, note best 10–15 minute segments.
2) **Transcript scan (20 min):** Identify 3 candidate highlights using:
   - Contrarian statement
   - List/steps (“3 things…”) 
   - Strong emotion/story/conflict
3) **Select 1 highlight (5 min):** Choose the one with clearest hook in first 1–2 seconds.
4) **Edit vertical (45–75 min):**
   - Remove dead air, tighten pauses
   - Add jump cuts every 1–3 sentences (as needed)
   - Reframe to 9:16 (face-centered)
5) **Hook + on-screen text (10–15 min):** Add 3–7 word headline matching the first line spoken.
6) **Captions (20–30 min):**
   - Auto-captions → correct key terms/names
   - Highlight 1–2 keywords per sentence (style consistent)
7) **Audio polish (5–10 min):** Light normalize, remove harsh peaks if needed.
8) **Export (5 min):**
   - 1080×1920
   - H.264, 15–30 Mbps
   - Filename: {Client}_{Topic}_v1.mp4
9) **Delivery (10 min):** Email with:
   - MP4 attachment/link
   - Suggested post caption (2 variants)
   - 3 hook alternatives (text-only) for testing
