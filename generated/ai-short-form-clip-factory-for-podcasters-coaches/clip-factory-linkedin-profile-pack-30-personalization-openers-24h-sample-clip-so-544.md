### B. Clip selection (30 minutes)
Pick **1 clip** that has:
- Clear hook in first 1–2 seconds
- Single idea, no context needed
- Payoff within 20–35 seconds
Write a 1-sentence hook overlay + optional mid-clip emphasis words.

### C. Edit spec (keep consistent)
- 9:16, center-crop speaker; add safe margins
- Captions: large, 2 lines max, highlight keywords
- Light cleanup: remove dead air, add subtle zooms every 3–5s
- Optional b-roll cue only if it clarifies meaning

### D. Delivery (within 24 hours)
Send:
1) MP4 final
2) Caption SRT (optional)
3) “Posting notes” (best caption line + 3 hashtags + CTA suggestion)

### E. Upsell (immediately after delivery)
Send a short conversion message:
“Want me to turn this same episode into a full batch? Typical output is 10–20 clips per episode, delivered in 48h. Reply ‘batch’ and I’ll send package options.”

### F. Quality bar (acceptance)
- Clip can be posted as-is (no additional editing)
- Captions are readable on mobile and synced
- Hook is visible within first second
- Export is platform-safe and not blurry
