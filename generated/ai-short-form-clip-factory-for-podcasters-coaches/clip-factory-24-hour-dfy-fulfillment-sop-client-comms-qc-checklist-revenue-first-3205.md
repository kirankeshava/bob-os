============================
1) DFY 24-Hour Fulfillment SOP (end-to-end)
============================
Goal: Deliver a first batch of ready-to-post vertical clips within 24 hours of receiving assets, with consistent formatting and minimal revisions.

Inputs we need from client (minimum):
- Long-form source link (YouTube, Spotify video, Drive link) or uploaded file
- Target platforms: TikTok / Instagram Reels / YouTube Shorts
- Brand preferences: fonts/colors (or “use clean defaults”), caption style (word-by-word vs sentence), logo/watermark (optional)
- CTA + offer: “follow for X”, “book a call”, “download lead magnet”, etc.
- Any “do not include” topics or phrases

Tooling (use what’s available; keep free/low friction):
- Editing: CapCut Desktop OR DaVinci Resolve (free)
- Captions: CapCut auto-captions OR Subtitle Edit + manual cleanup
- Optional transcription assist: Whisper (only if already available; do not spend money week 1)
- Asset storage: Google Drive folder per client

Step-by-step process:
A) Intake + project setup (10–15 min)
1. Create client folder structure:
   /ClientName/
     /01_Source/
     /02_Selects/
     /03_Exports/
     /04_Thumbnails (optional)
     /05_Notes/
2. Create a “Project Notes” doc with:
   - Source link(s)
   - Date received
   - Clip quota (e.g., 10 clips)
   - Platform targets
   - CTA + link
   - Caption style choice

B) Content scan + highlight selection (30–60 min)
1. Watch at 1.5x–2x speed; mark timestamps for:
   - Strong contrarian take / surprising stat
   - Clear “how-to” steps
   - Personal story with payoff
   - Objection handling (“people think X, but…”) 
2. Choose 8–12 candidate moments for a 10-clip delivery (buffer helps quality).
3. Prioritize moments that can stand alone without heavy context.

C) Clip assembly (2–4 hours depending on quantity)
For each clip:
1. Create 9:16 sequence (1080x1920).
2. Place speaker center-frame; use auto-reframe if needed.
3. Add hook text in first 1–2 seconds (either pulled from line or created):
   - Pattern: “Stop doing X”, “Nobody tells you this about Y”, “3 mistakes with Z”, “The real reason…”
4. Add captions:
   - Accuracy pass: names, numbers, key terms
   - Formatting: max 1–2 lines, high contrast, safe-zone compliant
5. Add b-roll cues (lightweight):
   - If relevant, insert 1–3 quick cutaways or zooms per clip
   - Use simple jump cuts; avoid over-editing
6. Add CTA end card (optional): last 0.5–1.5 sec.

D) Quality control + export (30–60 min)
1. Run QC checklist (section 3).
2. Export settings:
   - Format: MP4 (H.264)
   - Resolution: 1080x1920
   - FPS: match source (usually 30)
   - Bitrate: 10–20 Mbps (balance quality + upload speed)
   - Audio: AAC, 320 kbps
3. Name files consistently (section 4).

E) Delivery (10–15 min)
1. Upload to client Drive folder / Dropbox link.
2. Send delivery email (templates below).
3. Ask for: “Pick top 3 favorites + any caption style tweaks” to lock style for next batch.

C) Revision policy (include in onboarding or first delivery)
Revisions policy (keeps things fast):
- Included: 1 revision pass per clip for captions, hook text, and trims (within 7 days of delivery)
- Not included (unless agreed): content rewrites that change the meaning, new b-roll sourcing requests per clip, or re-cutting from a different section of the episode
- If you want a different style direction, send 1–2 reference clips and we’ll align the next batch

============================
3) QC Checklist (every clip)
============================
Hook + structure
- First 1–2 seconds have a strong opening line or on-screen hook
- Clip has a clear payoff (insight, step, contrarian point)
- No dead air at start/end; tight trims

Captions
- Spelling correct for names/brands/numbers
- Captions do not cover face/mouth; respects safe zones
- Contrast is strong; readable on mobile

Framing + pacing
- Subject centered; no awkward cropping
- Jump cuts are clean; no distracting zoom spam

Audio
- Voice is clear; no clipping
- Background music (if used) is low and not competing

Platform specs
- 1080x1920, MP4 H.264
- No watermarks (unless client logo)
