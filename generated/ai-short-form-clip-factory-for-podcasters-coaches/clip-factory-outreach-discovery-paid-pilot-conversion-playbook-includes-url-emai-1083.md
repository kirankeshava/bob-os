## E) Feature scoring rubric (for $99–$299/mo validation)
After each call, score each requested feature 1–5.
- **Revenue impact (1–5):** directly increases WTP or closes sale
- **Frequency (1–5):** how many clients asked for it
- **Time saved (1–5):** removes major manual work
- **Complexity (1–5, reverse):** 1 = easy, 5 = hard

**Priority score = (Revenue impact + Frequency + Time saved) − Complexity**

Candidate features to score:
1) Auto highlight detection quality controls (min/max length, “viral” scoring)
2) Diarization / speaker labels
3) Caption styling templates + brand presets
4) Hook generation (brand voice) + multiple options
5) B-roll cues / on-screen text callouts
6) Auto reframing / face tracking for 9:16
7) Team workflow (approve/reject, comments)
8) Scheduled posting / exports per platform
