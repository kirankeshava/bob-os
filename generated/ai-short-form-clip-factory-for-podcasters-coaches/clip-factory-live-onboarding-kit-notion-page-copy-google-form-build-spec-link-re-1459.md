1) What you’ll receive (Delivery Pack)
For each batch we deliver a folder containing:
A) Final clips (MP4) — ready to post
- Vertical 9:16 exports for TikTok/IG Reels/YouTube Shorts (unless you request otherwise).
B) Captions file (SRT)
- A matching .srt per clip so you can upload native captions if you prefer.
C) B-roll cue sheet (Google Doc or .md/.txt)
- Timestamped “insert b-roll here” suggestions (stock concepts or your own assets).
D) Posting suggestions sheet
- Suggested first-line caption (hook), CTA, and a starter hashtag set per clip.

2) QA standards (what we check before you ever see it)
We run QC so you don’t have to. Every clip is checked for:

A) Safe zones + framing
- No cropped faces (forehead/chin) and no cut-off on-screen text.
- Captions placed within platform safe zones so UI doesn’t cover key text.
- Speaker is centered or intentionally composed (rule-of-thirds), with clean headroom.

B) Captions accuracy + readability
- Captions match the spoken words (no meaning changes).
- Punctuation and line breaks improve readability.
- Highlighting (if used) matches the spoken emphasis.
- Profanity/banned words handled per your intake instructions.

C) Audio leveling
- Consistent loudness across clips.
- No harsh peaks/clipping; no distracting background hum if avoidable.
- Dialogue prioritized; music (if any) stays under voice.

D) Hook clarity + pacing
- Hook is clear in first 1–2 seconds.
- Removes dead air, filler starts, long breaths, and rambling intros.
- Jump cuts are clean and intentional.

E) Visual polish
- No glitch frames; no random zoom jitter.
- B-roll cues do not distract from the main point.
- If you request brand colors/fonts, we apply them consistently.

F) Export settings
- Resolution: 1080x1920 (unless requested otherwise)
- Codec: H.264
- FPS: match source where possible (typically 30fps)
- Audio: AAC, 48kHz
- File naming: consistent, sortable, and unique.

3) Review flow (timestamp comments)
We send a single review link for all clips (Frame.io / Vimeo / unlisted YouTube) OR a Drive folder if you prefer.

How to leave feedback
- Use timestamped comments (e.g., “00:12 change caption wording”, “00:19 remove ‘um’”).
- Consolidate feedback in one pass if possible.
- If you want a “style change,” please describe it once and we’ll apply it across the set (within scope).

Revision window
- You have 24 hours from delivery of the review link to request revisions included in the package.
- After 24 hours, requests are treated as a new round and may be paid depending on scope.

4) Revision policy (what’s free vs paid)
Goal: fast, fair revisions that protect your timeline and our editor time.

Free revisions (included, within the 24h window)
- Caption typo fixes, minor wording tweaks that don’t change the meaning.
- Slight timing adjustments to captions.
- Minor audio level tweaks (e.g., “voice a little louder”).
- Minor cut trims (e.g., remove 1–2 seconds, tighten pause).
- Reposition captions within safe zones.
- Replace a b-roll suggestion if it’s clearly off-topic.

Paid changes (quoted before work starts)
- Changing which moments were selected (new clip selection) after delivery.
- Re-cutting the narrative (major restructuring) or changing the hook concept.
- Adding new branded motion graphics packages beyond the agreed style.
- Heavy content compliance requests not mentioned in intake (e.g., new banned topics/words after the fact).
- Revisions requested after the 24-hour window.

Not included / can’t do
- Guaranteed performance metrics (views, followers). We optimize for best practices, but algorithms vary.

5) How to get the best results (client tips)
- Share your target platform(s): TikTok, IG Reels, YouTube Shorts.
- Tell us your tone: “punchy,” “calm authority,” “high energy,” etc.
- Provide CTA: “book a call,” “download lead magnet,” “comment keyword,” etc.
- Send brand assets: logo, colors, font preferences (optional).

ISSUE TRIAGE RULES (Internal quick reference)

When feedback arrives, categorize within 5 minutes:
A) Fix-now (Free, within scope)
- Typos, caption timing, safe zone placement, minor audio gain, micro-trims.
Action: acknowledge receipt + execute; no debate.

B) Clarify (Free if minor; needs question)
- Vague notes like “make it punchier,” “needs better hook,” without specifics.
Action: ask 1–2 clarifying questions; propose a single change.

C) Scope-change (Paid)
- New moments, new clip selection, major restructuring, new style direction across batch, brand kit introduction mid-stream.
Action: quote before work; offer cheaper alternative: 1 clip re-cut only.

D) Out-of-window (Paid unless goodwill exception)
- Requests after 24h.
Action: offer paid round; optionally fix a tiny typo as goodwill once.

Refund-risk triggers (escalate immediately)
- Missed deadline, wrong aspect ratio, inaccessible links, broken audio, severe caption inaccuracies, cropped face.
Action: prioritize fix within hours; send proactive update.
