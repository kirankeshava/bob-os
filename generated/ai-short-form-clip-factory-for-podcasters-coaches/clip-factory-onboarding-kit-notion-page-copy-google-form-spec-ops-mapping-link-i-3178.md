## 1) What you’ll receive (Delivery Pack)
For each batch of clips, you’ll receive a **Delivery Pack** containing:
1. **Final vertical MP4s** (ready to post)
2. **Caption files (SRT)** for each clip
3. **B-roll / visual cue sheet** (timecodes + suggestions)
4. **Posting suggestions** per clip (first-line caption + hashtags + platform notes)

**Folder structure (standard):**
- `01_FINAL_MP4/`
- `02_CAPTIONS_SRT/`
- `03_BROLL_CUE_SHEETS/`
- `04_POSTING_SUGGESTIONS/`
- `05_REVIEW_WORKFILES/` (optional)

## 2) Quality standards (our QC checklist)
We run a final QA pass on every clip before delivery:

### Framing & safe zones
- Subject face is not cropped (forehead/chin safe, no cut-off at edges)
- On-screen text stays within safe zones (avoid UI overlays)
- No important elements under platform UI (bottom caption bar, right-side icons)

### Captions accuracy
- Captions match spoken words (names/brands checked)
- Punctuation supports clarity and pacing
- Line breaks are readable (no walls of text)
- No banned words or restricted terms per your notes

### Hook & clarity
- First 1–2 seconds contain a clear hook (text +/or audio)
- Clip has a single clear takeaway; no confusing context gaps
- CTA (if requested) is included and not obstructed by UI

### Audio leveling
- Consistent loudness across clips (no big volume jumps)
- Dialogue is clear; background noise reduced where possible
- No clipping/distortion

### Edits & pacing
- Clean jump cuts (no awkward dead air)
- Intentional pacing (trim rambling / repeated phrases)
- Visual changes match beat changes where helpful

### Export & formatting
- Vertical export (9:16), platform-safe bitrate, clean rendering
- No watermark unless requested
- Filename format consistent and sortable

## 3) Review process (timestamp comments)
We deliver a **review link** (Frame.io, Vimeo, or unlisted YouTube) so you can comment with timecodes.

**How to request changes (fastest):**
- Leave **timestamp comments** directly on the review link.
- If email is easier, list notes as: `Clip 03 — 00:07–00:10: replace caption wording`.

## 4) Revision window (24 hours)
To keep turnaround fast and schedules predictable:
- You have **24 hours from delivery** to submit revision notes.
- We will confirm receipt and deliver revisions within the agreed turnaround.

If we don’t receive notes within 24 hours, the delivery is treated as accepted (you can still request changes later; see policy below).

## 5) Revision policy (what’s included vs scope change)
### Free revisions (included)
- Caption typos / mis-hears / punctuation fixes
- Minor timing adjustments to captions
- Minor crop/reframe fixes (e.g., face safe zone)
- Audio level tweaks (small adjustments)
- Swap in an alternate hook text from the same clip concept

### Not free (scope changes)
- New clip selection from different timestamps not previously included
- New creative direction after delivery (new style, new templates, new b-roll strategy)
- Adding new branded elements not provided in intake
- Major restructuring (new story arc) after delivery

If a request is a scope change, we’ll label it clearly and propose the lightest option.

## 6) What we need from you (Intake)
Please complete the **Clip Factory Intake Form** before editing starts. It ensures:
- correct handles/links/CTA
- preferred platforms and formatting
- banned words / compliance constraints
- brand tone and caption style
