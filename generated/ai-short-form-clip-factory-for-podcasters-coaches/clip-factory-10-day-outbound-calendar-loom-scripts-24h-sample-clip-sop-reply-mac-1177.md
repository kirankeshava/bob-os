C) 24-HOUR “SAMPLE CLIP” FULFILLMENT SOP (reduce friction, speed upsell)
Objective: deliver 1 polished 9:16 clip within 24h of receiving assets and payment confirmation.

Intake (timebox 10 minutes)
1) Get source: YouTube link OR Drive/Dropbox file.
2) Ask for: brand colors (hex), font preference, handle/watermark text, CTA, target platform (TikTok/Reels/Shorts), caption style (minimal/bold/word-by-word).
3) Confirm: “Any words to avoid? Any compliance constraints?”

Edit Spec (standardize output)
- Length: 20–45s (unless client requests 45–75s)
- Format: 1080x1920, H.264 MP4, 30fps
- Captions: burned-in, high-contrast, safe margins
- Hook: on-screen hook text in first 1–2 seconds
- B-roll cues: simple bracket notes for internal use (or subtle pop-ins if available)
- Deliverables: (1) final MP4, (2) caption copy + 3 title/hook variants (text)

Delivery

Email body:
1) Link to download/view.
2) “Two posting suggestions: [time], [caption option].”
3) Ask for 1–10 rating + one change request.
4) Upsell: propose weekly batch package with start date.
Include website + email again.
