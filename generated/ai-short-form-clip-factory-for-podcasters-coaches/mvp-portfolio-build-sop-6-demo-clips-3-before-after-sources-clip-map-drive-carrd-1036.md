B) Clip Style Spec (consistent across all demos)
Format: 9:16 vertical, 1080x1920, 30fps, 8–60 seconds.
Safe zones: Keep captions and key text within center 80% (avoid UI overlays).
Caption style: burned-in, bold, high-contrast, 2–3 lines max. Highlight keywords (color accent) every 6–10 words.
Motion: 6–10% punch-in zoom on emphasis; quick jump cuts (remove silences).
B-roll cues: Use simple stock/B-roll prompts as on-screen callouts (e.g., “[B-ROLL: map zoom-in]”) or insert free Pexels clips.
CTA end card (last 1–2 seconds): “Want 10 clips/week? Email agent_bob_replit+clip-factory@agentmail.to” + website short reference.

2) Remove filler + pauses; keep 120–170 wpm pacing.
3) Add hook text in first 1.5 seconds.
4) Add captions with keyword highlighting.
5) Add 2–4 punch-in zooms.
6) Add 1–2 b-roll moments OR add b-roll cue callouts.
7) Add end CTA (email + website) last 1–2 seconds.
8) Export 1080x1920 H.264, 15–25 Mbps.
