---
A) What you’ll receive (Delivery Pack)
For each order, you’ll receive a Delivery Pack containing:
1) Final vertical MP4 clip(s) (platform-ready)
2) Captions file(s): .SRT for each clip
3) B-Roll / On-screen cue sheet (timestamps + suggestions)
4) Posting suggestions:
   - First-line caption options (hooky text you can paste)
   - Hashtag suggestions (platform-specific)
   - Recommended posting notes (e.g., pinned comment prompt, CTA)

---
B) Review flow (fast and timestamped)
1) We send a review link (Frame.io / Vimeo / unlisted YouTube).
2) Please leave timestamped comments for changes (example: “00:07 – change ‘their’ to ‘there’”, “00:12 – zoom out slightly”).
3) Consolidate feedback into ONE pass whenever possible. Multiple scattered messages slow turnaround.

If you prefer email-only feedback, use this format:
- Clip Name:
- Timestamp:
- Change requested:

---
C) Revision window + turnaround
• Standard revision window: 24 hours after we send the review link.
• We apply in-scope revisions and return the updated file(s) as quickly as possible (typically next business day).

If we don’t hear back within 24 hours, we assume approval and move to final delivery (unless you request an extension in writing).

---
D) Quality standards we QA before you see anything
We check:
• Safe zones / no cropped faces / correct vertical framing
• Caption accuracy, punctuation, and readability
• Audio leveling (no clipping; voice intelligible)
• Hook clarity in first 1–2 seconds
• Clean jump cuts (no distracting stutters)
• Export settings + platform compatibility

---
E) What counts as a free revision vs a paid change
Free revisions (included):
• Caption typos, punctuation fixes, minor timing adjustments
• Small framing tweaks (e.g., nudge, slight zoom)
• Audio leveling fixes
• Replace a single word/phrase for compliance or clarity
• Fixes where we missed your provided brand terms/preferences

Paid changes (out of scope):
• Changing the clip concept/hook direction after delivery
• Swapping to different timestamps/segments (new clip selection)
• New caption style rules introduced after we started (e.g., “change to karaoke style for all clips”)
• Significant re-edits: new b-roll plan, new structure, additional versions

If a request is out-of-scope, we’ll quote the change clearly before doing work.

2) CLIENT INTAKE FORM SPEC (Google Form)
==============================
Form Title: Clip Factory — Client Intake (Brand + Clip Preferences)
Form Description: This takes ~3 minutes and helps us deliver clips that match your brand (and reduces revisions).

Required Fields (recommended):
1) Name
2) Email (must match delivery email)
3) Brand / Channel name
4) Primary platform target (checkboxes): TikTok / Instagram Reels / YouTube Shorts / LinkedIn
5) Content type (dropdown): Podcast (video) / Podcast (audio-only w/ static) / Coaching call / Webinar / YouTube long-form
6) Upload link(s) to source content (Google Drive/Dropbox/YouTube/private link) + any passwords
7) Desired clip length (dropdown): 15–25s / 25–40s / 40–60s / 60–90s
8) Caption style (checkboxes): Clean minimal / Bold highlight keywords / Word-by-word (karaoke) / Mixed
9) Brand words/phrases to ALWAYS spell correctly (short answer + example)
10) Words/phrases to NEVER use (banned words) (short answer)
11) Call-to-action preference (checkboxes): Follow / Comment / DM / Download / Book call / Other
12) Hook preferences (checkboxes): Contrarian / Story / Direct promise / Question / Stats/data / Other
13) Competitors or inspiration accounts (optional links)
14) Approval rules (checkboxes):
   - “I will respond to review within 24 hours” (required)
   - “If I don’t respond within 24 hours, you may proceed to final delivery” (required)

Optional but helpful:
15) Brand kit link (fonts/colors) and any existing caption examples
16) Compliance notes (medical/financial disclaimers, sensitive topics)
