## 3) 24-hour “Sample Clip” Fulfillment SOP (DFY)
**Goal:** Deliver 1 high-quality sample clip fast to convert to a paid package.

### Inputs required (ask in a single message)
Send to: agent_bob_replit+clip-factory@agentmail.to
Client provides:
1) YouTube link (or file upload link)
2) Target platform (Reels/TikTok/Shorts)
3) Niche + target audience (1 sentence)
4) Style reference (1 creator/account link)
5) Any banned words/topics (optional)

### Internal process (timed)
**0–15 min: Intake + confirm**
- Reply confirming receipt, ETA (24 hours), and what you’ll deliver (1 vertical clip with captions + 2 hook options).

**15–60 min: Select segment**
- Scan transcript/chapters or scrub video.
- Pick a segment with:
  - a clear ‘problem → insight → takeaway’ arc
  - minimal context needed
  - strong first sentence potential

**60–120 min: Draft hooks + caption plan**
- Write 2 hook options (5–10 words each).
- Decide caption style: short lines, max 1–2 lines on screen, high contrast.

**2–6 hours: Edit + captions**
- Export 9:16.
- Center speaker; add gentle zoom/punch-ins if needed.
- Burn captions; correct proper nouns.
- Add simple b-roll cue notes if relevant (as a text doc or in email).

**6–24 hours: QA + deliver**
- Watch fully with sound off (caption readability) and with sound on (timing).
- Deliver:
  - MP4 vertical
  - Optional SRT
  - Hooks (text)
  - 1-paragraph recommendation on posting (caption copy + CTA suggestion)
