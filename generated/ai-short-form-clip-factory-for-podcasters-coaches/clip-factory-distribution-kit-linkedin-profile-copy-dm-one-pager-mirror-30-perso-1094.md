## 4) 24-Hour “Sample Clip Offer” Fulfillment SOP (2 free clips)

**Goal:** Deliver 2 posting-ready vertical clips in 24 hours to convert prospects into DFY packages.

### Intake (0–10 min)
Request (email reply is fine):
- Link to episode (YouTube, Spotify video, Riverside, Drive) + time zone
- Target platform (Reels/TikTok/Shorts)
- Niche + target audience
- Any words to avoid / brand sensitivities
- 1–2 style references (creators they like)

Log in CRM: Name, link, deadline, niche, platform, status.

### Transcribe (10–30 min)
- Pull audio and run transcription (tool of choice).
- Quick scan for obvious high-energy segments, lists, contrarian takes, stories, and “how-to” moments.

### Select highlights (30–60 min)
Pick **2 segments** using these rules:
- 15–35 seconds each (unless the content demands longer)
- Clear standalone value without heavy context
- Strong first sentence (or rewrite hook as on-screen text)

### Hook + captions (60–120 min)
- Create a hook line for each clip (on-screen text in first 1–2 seconds).
- Clean captions: remove filler words when it improves readability, keep meaning intact.
- Add 1 simple CTA at end (optional): “Follow for [benefit]” / “Comment ‘CLIPS’ for part 2”.

### Format + export (120–180 min)
- 9:16, safe margins, speaker centered.
- Burned-in captions, high contrast.
- Filename: `Client_EpTitle_Clip1_YYYYMMDD.mp4`

### Delivery (by hour 24)
Send email to: agent_bob_replit+clip-factory@agentmail.to (CC client) with:
- 2 video files (or Drive link)
- Suggested post copy (2 options)
- Hashtag starters (5–10)
- Next-step upsell: “Want 10/week or 20/week? Reply with your preferred package and we’ll start Monday.”

### Acceptance criteria for the sample
- Delivered within 24h
- Clean readable captions, no obvious errors
- Correct 9:16 export and framing
- Hooks are explicit (text + first line)
- Client can post immediately with zero editing
