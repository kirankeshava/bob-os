### Workflow (time-boxed)
**0:00–0:10 — Confirm objective**
- Pick ONE goal: follower growth, lead magnet, or credibility.
- Define CTA: “Comment ‘X’”, “DM me ‘Y’”, or “Link in bio for Z”.

**0:10–0:35 — Find the best 45–75 seconds**
- Skim transcript for:
  - Contrarian/opinion start
  - Specific number/metric
  - Clear pain point + quick solution
  - A “mistake” framing
- Choose a segment with a natural beginning (no heavy context required).

**0:35–1:10 — Edit vertical clip (9:16)**
- Format: 1080x1920.
- Layout rule: speaker centered, add safe margins for UI.
- Add jump cuts every 1–2 sentences to keep pace.

**1:10–1:50 — Captions + hook pass**
- Burned-in captions, high contrast.
- Emphasize 3–7 keywords by color/bold (manually if needed).
- Add a 1-line on-screen hook at start (first 1.5s), e.g.:
  - “Most coaches waste 3 hours on this…”
  - “If you’re posting daily and still not growing…”

**1:50–2:10 — B-roll cues (lightweight)**
- If no b-roll, use zooms + emojis/icons sparingly.
- Add 1–2 quick overlays (e.g., headline text, simple chart screenshot) if it supports clarity.

**2:10–2:30 — Export + deliverables**
Deliver a single folder:
- `FINAL_clip_01_9x16.mp4`
- `Posting_Copy.txt` (caption + 3 hashtags)
- `Notes.txt` (what we’d improve for batch production)
