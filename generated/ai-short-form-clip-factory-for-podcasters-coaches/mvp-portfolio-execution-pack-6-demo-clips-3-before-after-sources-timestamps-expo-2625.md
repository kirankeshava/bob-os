## 2) Clip Styles + Edit Direction (6 total)
We’re doing 2 styles per niche for speed and clarity:

### Style 1 — Storytelling Hook (3 clips)
**Structure:** Pattern interrupt → tension/problem → punchy insight.
- First 1.0s: big hook text (5–7 words max)
- Every 1.5–2.5s: subtle jump/zoom cut (3–6%)
- Captions: 2 lines max, high contrast, keyword highlights
- B-roll cues: 2–4 inserts total (stock, screenshots, charts), max 0.8s each
- CTA (last 2s): “Want clips like this? agent_bob_replit+clip-factory@agentmail.to”

Hook examples to use by niche:
- Real Estate: “Everyone’s watching the wrong rate…”
- Fitness: “This one cue fixes most squats…”
- B2B: “Stop selling features. Do this instead…”

### Style 2 — Listicle (3 clips)
**Structure:** “3 things” / “2 mistakes” / “4 steps” with big on-screen numbering.
- Open: “3 [niche] mistakes I see weekly”
- Each point: 6–10 seconds, large number badge (1/2/3)
- B-roll: show example visuals per point (charts, app UI, workout demo, neighborhood imagery)
- CTA: “Daily clips for your content: agent_bob_replit+clip-factory@agentmail.to”
