## 1) What you’ll receive (Delivery Pack)
For every order, you’ll receive a **Delivery Pack** containing:
1. **Final vertical clips (MP4)** — ready to post (TikTok/IG Reels/YouTube Shorts)
2. **Captions file (SRT)** — editable captions file for repurposing
3. **B-roll cue sheet (DOC or PDF)** — timestamps + suggested cutaways/overlays
4. **Posting suggestions (DOC)** — hook text, first-line caption, CTA, and hashtag set per platform

## 2) Quality Standards (QA we run before you see anything)
We QC every clip against these standards:
- **Safe zones:** no text/face/critical elements in platform UI areas
- **No cropped faces:** especially top-of-head and chin; reframing applied as needed
- **Captions accuracy:** names/terms verified; punctuation for readability; line breaks timed to speech
- **Audio leveling:** consistent loudness clip-to-clip; de-essing/light noise reduction if needed; no peaking
- **Hook clarity:** first 1–2 seconds clearly communicate the payoff; no dead air
- **Jump cuts:** paced but not jarring; remove filler words when it improves clarity
- **Export settings:** vertical 1080x1920, H.264 MP4, high quality, compatible with mobile upload

## 3) Review process (timestamp comments)
You’ll receive a **single review link** (Frame.io / Vimeo / unlisted YouTube) where you can leave **timestamped comments**.

**How to request revisions (best practice):**
- Leave notes with timestamps (e.g., “00:08–00:12 remove pause”)
- Group feedback per clip (Clip 01, Clip 02, etc.)
- If you want a different “angle,” tell us the goal (e.g., “more controversial,” “more educational,” “more punchy”)

## 4) Revision window (protects speed + quality)
- You have **24 hours from delivery** to submit revision notes.
- We start revisions once we have a single, consolidated note set.
- Typical revision turnaround: **within 24 hours** (unless volume is unusually high).

## 5) What counts as a free revision vs. a paid change
### Free revisions (included)
- Caption fixes (typos, missed words, punctuation, timing)
- Minor reframes to avoid cropped faces/safe-zone issues
- Audio tweaks (leveling, minor noise)
- Small pacing edits (remove a pause, tighten a sentence)
- Replace a b-roll suggestion/overlay cue with a similar one

### Paid changes (scope change)
- New clip direction/angle after delivery (e.g., “make all clips inspirational instead”)
- New clips beyond the agreed quantity
- Major script rework / re-structuring that changes the narrative
- Adding new assets not supplied originally (logos, brand kits) and requiring rework
- Revisions requested after the 24-hour window (unless we missed a QA standard)

If a request is paid, we’ll quote it before starting. No surprises.
