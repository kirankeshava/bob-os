A) 24-HOUR “SAMPLE CLIP” DFY FULFILLMENT SOP (NO-CODE / MANUAL WORKFLOW)
Goal: Deliver 1 vertical (9:16) ready-to-post clip within 24 hours to prove quality and convert to a paid DFY package (which funds the $5k SaaS gate).
Definition of Done (Sample Clip):
- 1 clip (20–45s recommended; up to 60s if needed)
- Burned-in captions (word-highlight optional)
- Strong hook text in first 1–2 seconds
- Output formats: MP4 1080x1920, 30fps (or match source), + a .txt with caption copy + suggested post text
- Delivered via Google Drive/Dropbox link with filename conventions

Tooling (all free/low-cost options; no spend required this week):
- Transcript: YouTube transcript (if available) OR manual transcription using free tier tools
- Editing: CapCut Desktop (free) or Descript free tier if available
- Captions: CapCut auto-captions (free)
- B-roll: Optional from royalty-free (Pexels) if needed; otherwise use jump cuts + zooms
- Delivery: Google Drive share link

Step-by-step:
1) Intake (5 minutes)
   - Receive: YouTube link or uploaded file + goal (followers vs leads) + target platform (TikTok/Reels/Shorts).
   - Confirm turnaround + expectations: “1 sample clip in 24 hours; 1 revision included.”
   - Collect minimum brand inputs (if none, use a clean default style):
     a) Brand colors (or “use black/white + one accent”)
     b) Font preference (or “use a clean sans font”)
     c) Caption style examples they like (2 links if possible)

2) Find 2–3 candidate moments (15–25 minutes)
   - Pull transcript (YouTube transcript or quick transcribe tool).
   - Identify 2–3 highlight segments using this heuristic:
     - Contains a contrarian claim, a step-by-step, a mistake to avoid, or a surprising result
     - Can stand alone without intro context
     - Has a clean 1-sentence hook you can place as on-screen text
   - Choose 1 best segment for the sample based on clarity + emotional punch.

3) Build Hook + Caption Script (10–15 minutes)
   - Write:
     - Hook text (on-screen in first 1–2 sec): 6–10 words max.
     - Optional micro-subtitle: 1 short line clarifying stakes.
   - Create a short “post caption” for the client (for TikTok/Reels description): 1–2 sentences + CTA.

4) Edit for 9:16 (30–60 minutes)
   - Create a 1080x1920 project.
   - Place talking head center with mild zoom (5–10%) and cut dead air.
   - Add pattern interrupts every 2–4 seconds: punch-ins, quick cuts, or simple motion.
   - If needed, insert B-roll cues (optional): quick overlay b-roll for 1–2 seconds to cover edits.

5) Captions + Styling (15–30 minutes)
   - Auto-caption and correct obvious errors (names, brand terms).
   - Caption settings:
     - Large, high-contrast
     - Safe margins (avoid UI overlays)
     - Highlight key words sparingly
   - Add top hook text; keep it within safe area.

6) Export + QC (10 minutes)
   - Export MP4: 1080x1920.
   - Watch full clip:
     - First 2 seconds grab attention?
     - Captions readable on phone?
     - No awkward mid-thought start/end?
   - Rename file: [ClientName]_[TopicKeyword]_SampleClip_v1.mp4

7) Delivery (5 minutes)
   - Upload to Drive folder: /ClientName/SampleClip/
   - Send delivery email (template below) including:
     - Drive link
     - Suggested post caption
     - 2 alternate hook options
     - Next-step upsell (bundle of clips/week)

8) Revision loop (up to 20 minutes)
   - Allow 1 revision: hook text change, caption style tweak, trim start/end.
   - Set boundary: “Structural changes (different segment) would be a new sample or part of a package.”

9) Upsell to DFY package (same day)
   - Offer a clear package with turnaround + cadence.
   - Ask: “Want 5 like this per week from your next episode?”
