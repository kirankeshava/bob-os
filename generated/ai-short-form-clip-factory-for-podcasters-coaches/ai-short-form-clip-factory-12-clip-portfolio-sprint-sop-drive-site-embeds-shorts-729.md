B) CapCut production SOP (fast, consistent quality)
Project settings:
- Canvas: 9:16 (1080x1920)
- FPS: match source (usually 30). Export 30 FPS if unsure.
- Audio: normalize voice to ~-14 LUFS (or use CapCut “Loudness normalization” if available).

Clip length targets:
- 25–45s typical; up to 60s if the payoff lands.

Edit stack (repeatable template):
1) Hook (0–2s): big on-screen headline; hard jump cut into the most emotional/clear sentence.
2) Body (2–35s): tight jump cuts every 1.0–2.5s; remove filler.
3) Pattern interrupt (every 6–10s): zoom punch-in/out, quick b-roll overlay, or on-screen icon.
4) Payoff + CTA (last 3–6s): “Want 10 clips like this/week? See samples: [site] Email: [email]”

Caption style rules (burned-in):
- Position: lower third, but avoid covering mouths.
- Font: bold, high contrast; 2 lines max.
- Highlight 1–3 keywords per sentence (brand color optional; keep consistent).
- Add “speaker labels” only when it increases clarity (e.g., “Coach:” / “Client:” in testimonial style).

B-roll cue implementation (for demos):
- If you can’t add real b-roll quickly, use on-screen “B-ROLL:” callouts as overlays for 0.5–1.0s.
- Examples: “B-ROLL: house exterior drone shot” / “B-ROLL: spreadsheet + growth chart” / “B-ROLL: gym training montage”.

Export preset:
- 1080x1920, 30 FPS, H.264
- Bitrate: 10–16 Mbps
- Filename exactly matches the naming convention.

2) Listicle: “3 mistakes / 5 moves / 7 tips” with fast pacing + numbered on-screen beats.
3) Contrarian Take: “Stop doing X. Do Y instead.” Strong first line + proof.
4) Testimonial: transform a quote/case study into a punchy before→after narrative.
