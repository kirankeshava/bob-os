## 2) Hosted ‘What to Expect + Review Instructions’ Page — Paste-Ready Copy

**Page title:** Clip Factory — What to Expect (QA, Review, Delivery)

**Intro:**
Clip Factory turns your long-form video/podcast into ready-to-post vertical clips with hooks, captions, and platform formatting. This page explains how review works, what you’ll receive, and how revisions are handled.

**Contact + legitimacy (top of page):**
Support: agent_bob_replit+clip-factory@agentmail.to  
Legitimacy URL: https://6ff73b98-dcff-4f53-a10d-9ad392f63342-00-1n4h4nuiq6w8j-dbyyfbxp.picard.replit.dev/sites/3

### A) What you’ll receive (Delivery Pack)
For every batch/order, you’ll receive a Delivery Pack containing:
- **Final vertical MP4 clips** (ready to post)
- **Caption files (SRT)** for each clip
- **B-roll cue sheet** (optional visual suggestions by timestamp/moment)
- **Posting suggestions** per clip (first-line caption + hashtag set + CTA suggestions)

### B) Our quality checks (high-level)
Before anything is sent, we verify:
- Captions match spoken audio (names/brand terms checked)
- No clipped/cropped faces; safe margins for TikTok/Reels UI
- Audio level is consistent; no peaking/distortion
- Hook is clear in the first 1–2 seconds
- Jump cuts feel natural (no confusing mid-word cuts)

### C) Review flow (timestamp comments)
1) You receive a **review link** (unlisted) where you can watch all clips.
2) Leave feedback using **timestamped comments** (preferred). Example: “00:07 change caption to ‘…’”.
3) We apply revisions and deliver the final Delivery Pack.

### D) Revision window
You have a **24-hour revision window** starting from the moment the review link is delivered.
- If you respond within the window: we’ll prioritize fixes.
- If feedback arrives after the window: we can still help, but it may be treated as a new request depending on scope.

### E) What counts as a free revision vs paid change
**Free revisions (included):**
- Caption corrections (typos, misheard words)
- Minor timing tweaks (caption sync)
- Small hook/cut tweaks using the same source content
- Safe-zone/framing fixes (if faces/UI coverage slipped)

**Paid changes (scope change):**
- New clip selection direction (“replace all clips with different moments”)
- New branding package after delivery (fonts/colors/logo changes)
- Rewriting hooks/captions to a new voice/style not specified in intake
- Adding new source files or changing platform targets after the fact

## 3) Internal Gating Rule (Refund Prevention)
**Rule:** No editing starts until the client submits the Intake Form (or explicitly waives it in writing via email).

**Why:** Most refund-triggering issues come from missing brand terms, unclear hook preference, and platform ambiguity. The intake form makes expectations explicit and reduces rework.

**Enforcement:**
- If intake is missing, send the Review Request Email with the intake link.
- Start the turnaround clock only after intake submission (or written waiver).
