Clip Factory — How to Give Feedback Fast (and still hit the 48-hour turnaround)

This page is designed to help you get the best results with the fewest review cycles, so we can deliver 10 short-form clips within 48 hours.

Where to leave feedback
1) Preferred: Frame.io review comments on each clip (timestamped).
2) Backup: Google Doc / email with timestamps.

The 3 rules that make feedback actionable
Rule 1 — Timestamp everything
• Good: “00:03–00:05: Replace ‘3 mistakes’ with ‘3 costly mistakes’.”
• Not ideal: “The opening feels off.”

Rule 2 — Choose a priority level per note
• P0 (Must-fix): brand/legal/compliance, factual error, wrong name/claim, misquote, offensive/unsafe.
• P1 (Strongly prefer): hook wording, caption phrasing, pacing, b-roll choice, on-screen emphasis.
• P2 (Nice-to-have): stylistic preferences, alternate emojis, optional music change.

Rule 3 — One clear instruction per comment
Keep each note single-purpose so we can apply it quickly. If you want multiple changes, leave multiple comments.

What we can revise within the included revision policy
Included (1 revision round, up to 10 clips):
• Hook line tweaks (text + timing)
• Caption corrections and emphasis changes
• Small trims (remove filler, tighten pauses)
• Replace b-roll cue with an alternative (within the same moment)
• CTA/end-card text tweaks
Not included (requires new scope / timeline):
• Re-cutting an entirely different segment as a replacement clip
• Rebuilding the style system (new fonts/colors/layout) after approval
• Major audio repair that wasn’t present/possible in source

Fast approval workflow (recommended)
1) Approve a “Style Lock” on Clip #1 or #2 first.
   • Once you say “Style approved,” we replicate it across the rest.
2) Send feedback in one batch within 12 hours of delivery.
   • A single consolidated round is the fastest path to final.

If you need to change direction
If you decide to change the topic selection, tone, or niche after seeing first drafts, tell us explicitly:
• “New direction:” (what’s changing)
• “Keep:” (what stays)
• “Replace:” (which clips should be swapped)
We’ll confirm whether it fits inside the 48-hour SLA or needs an extended timeline.
