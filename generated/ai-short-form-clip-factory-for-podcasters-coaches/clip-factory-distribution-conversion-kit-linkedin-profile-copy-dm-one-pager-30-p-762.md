### B) Clip selection (20–30 minutes)
Pick 1–2 moments that match at least 2 criteria:
- Strong hook potential in first 1–2 seconds
- Clear payoff/insight/story within 20–45 seconds
- Emotion/contrast (mistake vs fix, myth vs truth, surprising result)
Write 3 hook options per clip.

### C) Edit spec (what “done” means)
For each sample clip:
- 9:16 vertical crop (speaker framed)
- Burned-in captions (high contrast)
- Hook text on screen for first beat
- Light jump cuts for pace
- Add 2–4 b-roll cues (text note is acceptable if not inserting b-roll yet)
- Export .mp4 + (optional) .srt

### D) Delivery package (send in one message)
Send:
1) Download link to clip(s)
2) The 3 hook options you considered + which one you used
3) Posting suggestion: platform + caption copy + 3 hashtags
4) Upsell: “Want 10 like this every week?” with package options
