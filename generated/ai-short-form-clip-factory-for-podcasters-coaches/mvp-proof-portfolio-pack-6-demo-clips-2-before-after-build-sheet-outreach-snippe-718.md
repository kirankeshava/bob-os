Goal: ship the smallest set of proof that closes prospects while outbound runs. MVP deliverables = 6 finished vertical demo clips (2 per niche) + 2 before/after comparisons (raw excerpt → final). Keep the 12+3 “full portfolio” as backlog; do not block outreach on it.

A) MVP CLIP MATRIX (6)
Each clip must show: (1) strong first 1.5s hook, (2) burned-in captions with emphasis, (3) pacing/zoom cuts, (4) clear formatting for Reels/TikTok/Shorts.

Niche 1: Real Estate
1) Storytelling hook (20–35s): “I almost lost $20k on this inspection detail…”
   - Edit beats: fast cold open, 0.3–0.6s jump cuts, punch-in on key words.
   - Captions: ALL CAPS for 3–5 hook words; highlight $ amounts.
   - B-roll cues (on-screen): [B-ROLL: home inspection photos], [B-ROLL: contract close-up].
2) Listicle (25–40s): “3 things buyers should do BEFORE touring.”
   - Edit beats: number pops, subtle whoosh SFX optional (free).
   - Captions: show 1/2/3 markers; keep safe margins.

Niche 2: Fitness
3) Contrarian take (20–35s): “Stop doing HIIT for fat loss (here’s why).”
   - Edit beats: contrarian opener, quick cut to explanation, pattern interrupt at ~8–10s.
   - Captions: emphasize “STOP” / “HERE’S WHY”.
   - B-roll cues: [B-ROLL: treadmill vs weights], [B-ROLL: macro chart].
4) Story + lesson (25–45s): “I trained 5x/week and gained weight… until I fixed this.”
   - Edit beats: micro-pauses removed; keep authenticity.
   - Captions: highlight “GAINED WEIGHT” / “FIXED THIS”.

Niche 3: B2B (consulting/agency/coaching)
5) Listicle (20–35s): “3 reasons your discovery calls are stalling.”
   - Edit beats: hard cuts; add on-screen keywords (PAIN / IMPACT / NEXT STEP).
   - Captions: highlight “DISCOVERY CALLS”.
   - B-roll cues: [B-ROLL: CRM pipeline], [B-ROLL: calendar blocked].
6) Testimonial-style proof (15–30s): “We added $X pipeline in 30 days by fixing ____.”
   - Edit beats: start with result; then one sentence of mechanism.
   - Captions: highlight result number and timeframe.
