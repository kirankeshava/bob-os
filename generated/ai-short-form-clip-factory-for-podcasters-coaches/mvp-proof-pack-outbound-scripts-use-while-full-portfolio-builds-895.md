## Goal (CEO directive)
Build the minimum proof needed to close prospects while outbound runs. Do not wait for a perfect 12-clip library.

## MVP Deliverables (what we ship first)
**MVP Proof Set = 6 demo clips + 2 before/after comparisons**
- **Niches:** Real Estate, Fitness, B2B (2 clips per niche)
- **Clip styles (per niche):**
  1) **Storytelling hook** ("I wish I knew this earlier…")
  2) **Listicle / framework** ("3 mistakes / 3 steps")
- **Before/After (2 total):**
  - Before = raw excerpt (no captions, no zooms)
  - After = final clip (captions + zoom cuts + b-roll cues)
  - Format: sequential (Before 3–5s → After 15–25s) OR split-screen.

## Production Spec (consistent look)
- **Format:** 9:16, 1080x1920, 30fps
- **Length:** 18–35 seconds each
- **Captions:** burned-in, high-contrast, 2-line max, keywords highlighted.
- **Cut cadence:** remove dead air; add subtle punch-ins every 1.5–3s.
- **Safe margins:** keep captions above UI safe area (bottom ~15%).
- **Audio:** normalize to ~-14 LUFS integrated (or platform default if not available).
- **B-roll cues:** on-screen bracket notes like: **[B-ROLL: charts] [B-ROLL: listing photo]** (for proof of “ready-to-edit” direction even if b-roll is not inserted).
