A) 20-MINUTE SALES CALL SCRIPT (DFY-FIRST + SaaS SIGNALS)
====================
Goal: close a DFY package today; second goal: capture what they’d pay $99–$299/mo for later.

1) Open (1 min)
“Thanks for hopping on. I’m Bob from Clip Factory (site: https://6ff73b98-dcff-4f53-a10d-9ad392f63342-00-1n4h4nuiq6w8j.picard.replit.dev/sites/3). Quick plan: I’ll ask a few questions about your content + goals, I’ll tell you exactly what we’d ship each week, and if it makes sense we’ll pick a package and I’ll send next steps. Sound good?”

2) Context + pain (4 min)
- “What are you publishing today (podcast, YouTube, livestreams)? How often?”
- “Where do you want to grow most in the next 30 days—TikTok, Reels, Shorts, LinkedIn?”
- “What’s the bottleneck right now—finding moments, editing time, posting consistency, or not knowing what will perform?”
- “If clips worked, what would success look like: leads, booked calls, course sales, brand awareness?”

3) Diagnose and mirror (2 min)
“So the real issue is: you already have long-form, but turning it into consistent short-form that drives [leads/bookings] is eating time and not predictable. Did I get that right?”

4) Offer (5 min) — DFY packages
“We’re a short-form clip factory. You send a link (YouTube/Riverside/Drive). We deliver ready-to-post vertical clips with: hook options, burned-in captions, suggested title/description, and posting notes.

Packages (choose one):
- Starter: 12 clips/month (3 per week)
- Growth: 20 clips/month (5 per week)
- Domination: 30+ clips/month (daily cadence)

We can also do add-ons if needed: thumbnail frames, platform-specific variants, or pulling clips from multiple episodes per week.”

5) Proof without case studies (2 min)
“To make this low-risk, I can produce 1 sample clip in 24 hours from your latest episode. If you like the style and pacing, we roll into a package and keep a weekly cadence.”

6) Close questions (3 min)
- “If we could deliver [X] clips/week consistently starting this week, would you want to start with Starter/Growth/Domination?”
- “What would stop you from starting today—budget, timing, or uncertainty about performance?”

7) Objection handling (2–4 min)
Budget: “Totally fair. If we made this pay for itself, what’s the minimum you’d need—one client per month, 10 leads, or just consistent growth? We can start with Starter for proof, then scale.”
Timing: “Let’s lock the cadence now and start with the sample clip. If it hits, we’re already rolling.”
Already have editor: “Makes sense. Where editors usually struggle is picking moments + packaging for platforms. We can complement them by delivering highlights, hooks, and captioned vertical exports that are ready-to-post.”

8) SaaS validation questions (2 min, only after interest)
“Quick product question—if there were a self-serve tool that did this from a link (detects highlights, auto captions, exports vertical templates), what would you pay monthly if it saved you editing time?”
- “$99/mo if it reliably found highlights + captions?”
- “$199/mo if it also did brand templates + your preferred hook styles?”
- “$299/mo if it added speaker detection/diarization + niche templates + scheduled exports?”
Then ask: “Which 2 features would be non-negotiable?” (diarization, brand voice hooks, niche templates, b-roll prompts, team collaboration)

9) Next steps (1 min)
“Cool. Email me your latest episode link + brand preferences to agent_bob_replit+clip-factory@agentmail.to. I’ll deliver a sample clip within 24 hours. If you approve it, we start your first batch immediately.”

====================
C) 24-HOUR SAMPLE CLIP FULFILLMENT SOP (FREE TOOLCHAIN)
====================
Objective: deliver 1 impressive vertical clip fast, to convert into a DFY package.

Input from prospect (required):
- One long-form link (YouTube/Riverside/Drive)
- Target platform (TikTok/Reels/Shorts)
- CTA + offer (what should viewer do?)
- Any brand constraints (fonts/colors/logo optional)

Step-by-step:
1) Intake + confirm (5 min)
Reply confirming receipt + delivery time. Re-share legitimacy links:
“Got it. I’ll deliver within 24 hours. Process: https://6ff73b98-dcff-4f53-a10d-9ad392f63342-00-1n4h4nuiq6w8j.picard.replit.dev/sites/3. Questions anytime: agent_bob_replit+clip-factory@agentmail.to”

2) Find 2 candidate moments (20–30 min)
- Skim for: strong opinion, contrarian take, clear tactical list, emotional moment, founder story.
- Choose 30–45 seconds (or up to 60s) with a natural hook.

3) Draft 3 hook options (10 min)
Write 3 on-screen hook variants (first 1–2 seconds).
Examples:
- “Most podcasters waste their best moments…”
- “If you’re not getting leads from content, do this…”
- “Stop posting ‘highlights’—post this instead…”

4) Edit + captions (60–90 min)
Free approach options:
- CapCut Desktop (free): auto captions + style, export 1080x1920.
- DaVinci Resolve (free): manual captions or import SRT.
- Descript (free tier as available): transcript-based cut + captions.
Output requirements:
- 9:16 1080x1920
- burned-in captions
- safe margins for TikTok UI
- audio normalized; remove long pauses

5) Packaging (10 min)
Deliverables to send:
- 1 MP4 (vertical)
- 3 hook text options
- Suggested caption (1–2 sentences) + 3 hashtags
- 1 posting note: “Post as Reel, add CTA: ‘Comment CLIP’”

6) Delivery email (copy/paste)
Subject: Your sample clip is ready (24h)
Body:

Quality checklist (before sending):
- Hook appears within first 1 second
- Captions are readable on mobile (2–4 lines max)
- No dead air > 0.3s
