Below is an operator-ready system to fulfill a “Free Sample Clip in 24 Hours” and convert into DFY clip packages. Keep everything lightweight and fast; the goal is to ship publish-ready clips reliably, learn client preferences, and close DFY revenue (the SaaS MVP remains gated until $5k DFY is collected).

========================
1) 24-HOUR DFY FULFILLMENT SOP (3–10 clips)
========================
Objective: Turn one long-form asset (YouTube link, podcast video, Zoom recording) into ready-to-post 9:16 clips with burned-in captions, a strong hook, and platform-safe formatting.

Inputs (required):
- Source link or file (YouTube/Vimeo/Drive upload)
- Target platform(s): TikTok / Instagram Reels / YouTube Shorts
- Topic + target audience
- CTA (follow, comment keyword, book call, download lead magnet)
- Brand kit (optional): colors, fonts, logo, preferred caption style

Tooling (free/low-cost defaults; no spending required):
- Transcript: YouTube transcript (if available) or local transcription tool already accessible
- Editing: CapCut desktop/web (free) or existing NLE
- Captions: CapCut auto-captions + manual corrections
- Storage/Delivery: Google Drive folder + share links

Workflow (timeboxed):
A. Intake + setup (10 minutes)
1) Create a client folder: /ClientName/Date_SourceTitle/
2) Add subfolders: /01_Source, /02_Project, /03_Exports, /04_Thumbs, /05_Notes
3) Log job in tracker (see Section 4).

B. Identify clip moments (30–45 minutes)
Goal: Select 3–10 moments that can stand alone without context.
Selection rules:
- Must open with tension/curiosity within first 1.0 seconds.
- One idea per clip.
- Aim length: 20–45s for Reels/TikTok; 15–35s for Shorts (flex if story demands).
- Prefer “contrarian” statements, lists (3 steps), mistakes, frameworks, quick wins, strong opinions.

C. Build clip templates (20 minutes once; then reuse)
Create 2 simple styles:
Template A (Clean):
- 9:16 canvas 1080x1920
- Speaker centered (upper 60% of frame)
- Captions bottom third, 2 lines max
- Progress bar (optional)
Template B (Bold Hook):
- Large 5–8 word hook at top for first 2–3 seconds
- Captions mid-lower third
- Quick punch-in zooms on emphasis

D. Edit each clip (10–20 minutes per clip)
Per clip checklist:
1) Start with the strongest sentence (remove greetings).
2) Add jump cuts to remove pauses.
3) Add light punch-in / reframe to keep motion.
4) Captions: auto-generate, then manually correct names/terms.
5) Add subtle background music at low volume ONLY if it doesn’t harm clarity.
6) Add CTA end-card (0.5–1.5s) if requested.

E. QC pass (2 passes; 3–5 minutes per clip)
QC Pass 1 (content):
- Does it make sense standalone?
- Is the hook clear in first 1–2 seconds?
- Any risky claims or sensitive content?
QC Pass 2 (technical):
- Captions not cut off by UI safe zones.
- Audio levels consistent; no clipping.
- No awkward mid-word cuts.

F. Export settings (platform-safe)
- Format: MP4 (H.264)
- Resolution: 1080x1920
- FPS: match source (usually 30)
- Bitrate: 8–16 Mbps
File naming convention:
Client_Platform_Topic_Clip##_##seconds_v1.mp4
Example: AcmeCoach_Reels_ColdEmail_Clip03_32s_v1.mp4

G. Delivery package (within 24 hours)
Deliver:
- 3–10 exported MP4 clips
- Optional: a simple “Posting Notes” doc per clip: suggested caption + 3 hashtags + CTA
- One Drive folder link
