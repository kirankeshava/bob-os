Definition of Done (DoD)
A delivery is “done” when:
1) Each clip is 9:16 (1080×1920), with burned-in captions, readable safe margins, and clean audio.
2) Each clip has a clear hook in the first 1–2 seconds (via content selection + optional on-screen header).
3) Captions are accurate (names/terms corrected) and styled consistently.
4) File naming + delivery message includes recommended post copy ideas and a suggested title/hook.

Inputs required (Intake checklist)
Collect via email/DM (copy/paste questions):
1) Source link(s): YouTube/Drive/Dropbox link(s) + timestamps if they have them.
2) Platform priority: TikTok / Reels / Shorts (choose 1–2).
3) Target audience + niche (1 sentence).
4) Brand preferences: colors, font preference (or “default clean style”), logo file (optional).
5) Do-not-use list (topics/words) + any compliance notes.
6) CTA preference: “Follow for X”, “Comment ‘Y’”, “Book a call”, etc.

Tooling (free/low-cost defaults)
- Transcript: YouTube transcript (if available) or Whisper (only if already accessible in your workflow).
- Editing: CapCut Desktop (free) or DaVinci Resolve (free).
- Captions: CapCut auto-captions + manual corrections.

24-hour workflow (time-boxed)
T=0–30 min: Intake + project setup
1) Confirm receipt of source links and platform priority.
2) Create a project folder: ClientName_Date_SamplePack.
3) Download source video/audio.
4) Create a simple tracking note with 5 candidate moments.

T=30–90 min: Transcript + highlight mining
1) Pull transcript (YouTube auto transcript or transcribe).
2) Identify 8–12 “clip candidates” using these heuristics:
   - Strong claim or contrarian point
   - Step-by-step list (“3 ways to…”) or framework
   - Emotional/confessional moment
   - Quick story with a punchline/lesson
   - Clear before/after or mistake → fix
3) For each candidate, note:
   - Start/end time
   - 6–10 word hook summary
   - Any b-roll cue (optional)

T=90–150 min: Select final 3–5 clips
Selection criteria:
- Must stand alone without full episode context.
- First sentence must be punchy or can be made punchy with an on-screen header.
- Avoid long preambles; trim aggressively.
Target lengths:
- TikTok/Reels: 20–45s (ok up to 60s if tight)
- Shorts: 15–45s

T=150–360 min: Edit + captions (per clip 30–45 min)
For each clip:
1) Cut to the strongest start (remove “um/so” and slow lead-ins).
2) Normalize audio (ensure voice is clear; remove dead air).
3) Convert to 9:16:
   - Center subject; use auto-reframe if available.
   - If wide shot: use dynamic zoom/punch-ins every 4–8 seconds.
4) Add captions:
   - Style: high-contrast, large, 2 lines max.
   - Highlight 1–3 keywords per sentence (consistent color).
   - Correct proper nouns and domain terms.
5) Optional: Add a simple on-screen header for the first 2 seconds if the spoken hook is weak.
6) End cleanly (no awkward cutoffs). If needed, add a quick text CTA card (0.5–1.0s).

T=360–420 min: QC pass + export
QC checklist (must pass all):
- Captions: 98%+ accurate; no distracting typos; punctuation doesn’t break readability.
- Safe zones: captions not blocked by UI (keep lower captions above bottom UI area).
- Hook speed: first 1–2s contains a claim/question/problem.
- Visuals: face not cropped; no jitter; no black bars unless intentional.
- Audio: no peaking/distortion; consistent loudness across clips.
Export settings:
- 1080×1920, H.264, 30fps, 10–20 Mbps.
File naming:
- Client_Handle_Topic_01.mp4, etc.

T=420–480 min: Delivery
1) Upload to Drive/Dropbox.
2) Send delivery message (templates below).
3) Ask 2 questions to move toward DFY retainer (posting cadence + next episode link).

Rework policy (for sample pack)
- Included: 1 round of minor edits per clip (caption fixes, tighter trims, hook header tweak).
- Not included: full redesign of style, new brand kit integration, heavy b-roll sourcing, or changing the chosen moments entirely (unless we made a clear mistake).
- Turnaround for rework: 24 hours.
