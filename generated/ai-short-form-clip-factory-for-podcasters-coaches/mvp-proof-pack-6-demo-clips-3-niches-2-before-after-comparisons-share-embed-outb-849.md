Why this is enough
Prospects need to see: (1) hooks, (2) readable captions, (3) pacing/zoom cuts, (4) platform-safe 9:16 formatting, (5) optional b-roll cues. Two styles per niche demonstrates range without bloating workload.

Clip lineup (MVP matrix)
For each niche, produce these two styles:
1) Storytelling hook (30–45s)
- Pattern: “Most people think X… here’s what actually works.”
- Editing: 1.05–1.12x, remove dead air, punch-in every 2–3 seconds, 3–5 emphasized caption words/line.
- Overlay: subtle progress bar or section headers.

2) Listicle/value bullets (20–35s)
- Pattern: “3 mistakes / 3 tips / 3 red flags”
- Editing: hard cuts, quick b-roll cue callouts, big keyword emphasis.
- Captions: one idea per line, max 6–9 words/line.

Before/after comparisons (2)
Format A (sequential):
- 0:00–0:02 title card: “Before → After (Clip Factory Sample)”
- 0:02–0:10 BEFORE: raw excerpt (no captions), label top-left “RAW”
- 0:10–0:28 AFTER: final clip (captions + zoom cuts + b-roll cues), label top-left “FINAL”
- 0:28–0:30 CTA slate: “Want this for your content? agent_bob_replit+clip-factory@agentmail.to”

Format B (split-screen):
- Left = raw, Right = final, same timecode; label each side. Keep split-screen only if readability remains strong on mobile.

Production checklist (batch, minimal ambiguity)
1) Source selection (license-safe)
- Pick ONE longform video per niche (Creative Commons / CC0 / creator-permitted reuse).
- Capture proof: URL + license type (screenshot or note) in an “00_LICENSE_PROOF” folder.

2) Edit settings (consistent look)
- Canvas: 1080×1920 (9:16)
- Safe margins: keep text within central 80% width; avoid bottom UI region.
- Captions: bold, high-contrast, 2–3 lines max; highlight keywords in yellow.
- Loudness target: ~-14 LUFS integrated; limit peaks -1 dB.

3) Clip assembly (per clip)
- Create hook first 1.5 seconds (text-on-screen hook)
- Remove filler words + silences
- Punch-in/zoom rhythm every 2–3 seconds
- Add 2–4 b-roll cues (on-screen labels like “B-ROLL: house exterior”, “B-ROLL: workout form”, “B-ROLL: dashboard/CRM”)
- End slate (0.8–1.2s): “Sample by Clip Factory — Get clips: agent_bob_replit+clip-factory@agentmail.to”
