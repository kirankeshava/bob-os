3) INTERNAL QA SCORECARD (CHECKLIST)
============================================================
Use this checklist on every clip before export/delivery. Goal: fast, consistent, platform-safe.

Clip Metadata
[ ] Clip ID matches naming convention
[ ] Correct client/episode folder
[ ] Duration within target range (15–60s)

Hook + Structure
[ ] Hook occurs within first 0.0–1.5s
[ ] First frame visually readable (no dead air, no blank frames)
[ ] Pattern interrupt at least every 2–3s (cut, zoom, b-roll, text beat, angle switch)
[ ] Ending has a clean “button” (no awkward mid-word cut)

Captions
[ ] Captions have no obvious ASR errors (names/brands checked)
[ ] Caption safe zone respected (not blocked by UI)
[ ] Highlighting consistent with chosen preset
[ ] Profanity policy followed per client (bleep/omit if needed)

Framing + Visuals
[ ] Face/subject centered for 9:16; no forehead/chin crops
[ ] Reframe tracks speaker during movement
[ ] B-roll/overlay does not obscure key facial expressions
[ ] No copyrighted logos/music accidentally introduced

Audio
[ ] Loudness consistent across clips (no sudden jumps)
[ ] No distracting background noise (light cleanup OK)
[ ] No clipped/distorted peaks

Brand + CTA
[ ] Correct style preset applied (font/color/position)
[ ] CTA end card present (if required)
[ ] Any disclaimers added for regulated niches (if required)

Export
[ ] Export preset correct (9:16, H.264, 1080x1920)
[ ] File plays on mobile without glitches
[ ] Filename correct

Pass/Fail:
- PASS if all critical items met: hook, captions accuracy, framing, export.
- FAIL if any of the above are missing; fix before delivery.

Status definitions (use data validation dropdown):
- Intake: waiting on client info/assets
- Sourcing: downloading/transcoding source
- Transcript: Whisper/ASR processing
- Selects: highlight selection underway
- Editing: cut + reframes + pattern interrupts
- Captions: caption styling + corrections
- QA: scorecard pass
- Exported: files rendered
- Delivered: link sent
- Revision: 1st revision round in progress
- Complete: approved and archived
