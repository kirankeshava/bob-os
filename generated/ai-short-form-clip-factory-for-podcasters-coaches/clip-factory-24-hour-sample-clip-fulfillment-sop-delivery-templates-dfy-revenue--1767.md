## Standard Delivery Specs
- Format: 9:16 vertical, 1080×1920, .mp4
- Length: 20–45s (unless topic needs 45–75s)
- Captions: burned-in, 2-line max, high contrast, keyword emphasis
- Hook: first 1–2 seconds must contain strongest claim/question
- Safe margins: keep key text within center-safe area

## Turnaround Timeline (24 hours)
T+0–1h: Intake confirmed + source downloaded/queued
T+1–4h: Transcript + highlight selection + hook options
T+4–10h: Edit + captions + layout
T+10–18h: QC pass + revisions
T+18–24h: Deliver + ask for approval/testimonial + close to package

## Step-by-Step Process
### 1) Intake + Logging (10 minutes)
- Create client folder in Drive:
  - /ClientName/01_Source
  - /ClientName/02_Exports
  - /ClientName/03_Transcripts
  - /ClientName/04_Notes
- File naming convention:
  - Client_EpDate_Topic_Clip01_v1.mp4
  - Client_EpDate_Topic_Clip01_v2.mp4
- Log in CRM (even if just a Sheet):
  - Client, email, niche, source link, date received, promised delivery time, status

### 2) Source Acquisition (15–30 minutes)
- If YouTube: download highest quality available.
- If Drive: request “anyone with link can view.”
- Confirm audio is clear; if not, note it early.

### 3) Transcription (30–90 minutes depending on length)
- Generate transcript (any tool available to you; accuracy is the goal).
- Add timecodes if possible.
- Quickly scan for:
  - “Hot takes,” contrarian statements
  - Step-by-step frameworks
  - Story moments (conflict → lesson)
  - Lists (3 mistakes / 5 tips)

### 4) Highlight Selection (30–60 minutes)
Select 3 candidate segments, then pick 1 for the sample.
Criteria:
- Clarity: can it stand alone without context?
- Punch: strong claim, insight, or story
- Brevity: minimal setup required
- Specificity: numbers, examples, “I did X and got Y”

Write 2 hook options for the chosen segment:
- Hook A: bold claim
- Hook B: question/curiosity
Example structure:
- “Most coaches waste money on X. Do this instead.”
- “If your podcast isn’t growing, it’s probably because of this.”

### 5) Edit Assembly (1–3 hours)
- Create 9:16 sequence.
- Primary layout options (choose one):
  1) Center speaker crop + captions bottom
  2) Speaker top + captions middle + b-roll cues as quick cutaways
- Add quick pattern interrupts every 2–4 seconds (zoom/cut/b-roll) if needed.
- Remove filler words and dead space.

### 6) Captions + Styling (45–120 minutes)
- Burn-in captions.
- Emphasize keywords (caps or color) sparingly.
- Keep to 2 lines max; avoid covering mouth.
- Correct proper nouns.

Optional “B-roll cues” (simple):
- Add 2–3 quick overlays (e.g., “Client Results”, “3 Steps”) or cutaway b-roll moments.

### 7) Quality Control Checklist (10 minutes)
- Audio: no clipping, consistent volume
- Captions: readable on phone, no typos
- Hook: first 2 seconds strong + aligned with topic
- Framing: face not cut off; safe margins
- Export: plays on mobile, correct resolution
