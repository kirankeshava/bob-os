## 1) What you’ll receive (Delivery Pack)
For every order we deliver a **Delivery Pack** containing:

3. **B-roll / cut cue sheet** (timecoded suggestions; optional but included by default)
4. **Posting suggestions** per clip: first-line caption + CTA + hashtag set + platform notes (TikTok/Reels/Shorts)

## 2) QA Standards (what we check before you see anything)
We run a checklist before delivery. Key standards:

### A. Safe zones / framing
- Subject’s face and key on-screen text stays within safe zones (top/bottom UI areas avoided for TikTok/Reels).
- No cropped chins/foreheads unless stylistic and intentional.
- If screen-recording: critical UI is readable; zoom/crop is stable.

### B. Captions accuracy
- Captions reflect spoken words accurately; no meaning changes.
- Punctuation for readability; line breaks not awkward.
- Speaker labels only when needed.
- Profanity policy matches your intake (bleep/replace/leave).

### C. Audio leveling
- Speech is clearly intelligible on phone speakers.
- No clipping/distortion.
- Background noise reduced without “watery” artifacts.
- Loudness consistent across clips (avoid huge volume swings).

### D. Jump cuts & pacing
- Cuts remove dead air while keeping meaning.
- No jarring mid-word cuts.
- Visual continuity is acceptable (or intentionally stylized).

### E. Hook clarity
- First 1–2 seconds has a clear hook (question, bold claim, contrast, pain/solution).
- On-screen headline (if used) matches the spoken hook.

### F. On-screen text / branding
- Font size readable on mobile.
- Colors contrast the background.
- If you provide brand kit: correct font/color/logo placement.

### G. Export settings (default)
- Format: MP4 (H.264)
- Aspect: 9:16 vertical
- Resolution: 1080×1920 (unless source limits)
- Frame rate: match source (commonly 30fps)
- Audio: AAC

## 3) Review Flow (fast, timestamped, no confusion)
We provide a **review link** (preferred options):
- Frame.io review link (free tier when possible)
- Vimeo unlisted review link
- YouTube unlisted link (fallback)

### How to request revisions (required)
To keep turnaround fast, please leave notes as **timestamped comments**:
- Example: “00:07 — change headline to ‘3 mistakes…’”
- Example: “00:14 — remove ‘um’ and tighten pause”

### Revision window
You have **24 hours from delivery** to submit revision notes for included revisions. After 24 hours, we can still help, but changes may be treated as a new request depending on scope.

## 4) Revision & Issue Triage (what’s free vs paid)
We optimize for happy clients while preventing scope creep.

### Free revisions (included)
If requested within the 24-hour window:
- Caption typos / timing fixes
- Minor crop/safe-zone adjustments
- Audio level tweaks
- Small on-screen text edits (headline wording, minor reposition)
- Removing a filler word or tightening a small pause
- Fixing missed requirement that was clearly stated in your intake

### Paid changes (out of scope / new work)
We’ll quote before doing any paid work:
- Changing the clip selection/theme after delivery (“make it a different moment”)
- Adding new b-roll style beyond the agreed approach across many clips
- New branding direction after we followed the provided brand kit
- Major restructuring, adding new sections, or creating additional versions per clip
- Requests submitted after the 24-hour window (when it requires reopening the project)

### True errors (our responsibility)
If we made a mistake (wrong format, broken file, missing deliverable, obvious caption mismatch, unsafe crop), we fix it promptly regardless of window.
