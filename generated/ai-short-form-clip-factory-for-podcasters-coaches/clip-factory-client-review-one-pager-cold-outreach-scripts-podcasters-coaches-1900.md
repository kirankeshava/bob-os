## 1) Client One-Pager: How to Review Clips + Request Revisions (Fast)

**Thanks for trusting Clip Factory. Here’s the fastest way to review so we can keep the 48-hour pace.**

### What you’ll receive
- **10 vertical clips** (TikTok/IG Reels/YT Shorts ready)
- **Captions burned in** (plus optional .SRT if requested)
- **Clip list** (titles + timestamps + 1-line hook summary)
- **Posting notes** (recommended caption idea + CTA suggestion)

### How to review (best practice)
1) **Watch the first 2 seconds** of each clip first. If the hook doesn’t land, mark it.
2) If hook is good, **scan captions** quickly for readability and brand tone.
3) Check **framing** (face centered, safe margins, no blocked text).
4) If you want changes, request them in the format below.

### How to request revisions (copy/paste format)
Please send one message/email with all edits using this structure:
- **Clip File Name:** (example: CF_JoeRogan_2026-04-09_C01_v1_TT.mp4)
- **Type:** Hook / Captions / Cut / Framing / CTA / Other
- **Timestamp:** 00:00–00:00
- **Change Requested:** (1 sentence)
- **Optional replacement line:** (the exact text you want on screen)

**Examples**
- Clip File Name: CF_AlexHormozi_2026-04-09_C03_v1_IG.mp4
  - Type: Hook
  - Timestamp: 00:00–00:02
  - Change Requested: Make the first line more direct and remove the pause.
  - Replacement line: “If you’re not getting clients, it’s because your offer is unclear.”

- Clip File Name: CF_FitnessCoach_2026-04-09_C07_v1_YS.mp4
  - Type: Captions
  - Timestamp: 00:11–00:14
  - Change Requested: Fix wording: change “loose” → “lose”.

### Revision policy (fast + fair)
- Includes **1 revision round** for the batch (up to **10 small changes total** across clips).
- “Small changes” = text tweaks, trimming 1–2 seconds, caption timing fixes, crop/framing adjustments, swapping CTA text.
- “New creative direction” (new clips, different topics, new style template) is treated as a **new batch**.
