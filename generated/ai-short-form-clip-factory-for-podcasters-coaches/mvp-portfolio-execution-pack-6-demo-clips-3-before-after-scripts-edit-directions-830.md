NICHE 1: REAL ESTATE

RE-01 (Story Hook) — “The $18,000 mistake first-time buyers make”
- Hook (0–2s) on-screen text: “First-time buyers: this mistake costs ~$18K”
- Voice pull: choose a segment where speaker mentions a costly oversight (closing costs, inspection, rate lock, etc.).
- Edit beats:
  1) Start mid-sentence for pattern interrupt.
  2) Punch-in 110% on “costs” and “$18K”.
  3) Add 2–3 fast b-roll inserts: house keys, inspection checklist, mortgage papers.
- Caption style (example lines):
  - “Here’s the mistake nobody warns you about…”
  - “It’s NOT the down payment.”
  - “It’s the hidden costs + timing.”
- B-roll cues (overlay notes):
  - [B-ROLL: inspection photo]
  - [B-ROLL: closing disclosure doc]
- CTA (final 2–3s): “Want 10+ clips/month like this? See samples: https://6ff73b98-dcff-4f53-a10d-9ad392f63342-00-1n4h4nuiq6w8j.picard.replit.dev/sites/3 | agent_bob_replit+clip-factory@agentmail.to”

RE-02 (Listicle) — “3 things to check before you tour a home”
- Hook text: “Before your next house tour… check these 3 things”
- Structure: count-up 1/2/3 with quick visual transitions.
- Edit beats:
  - Add number pop animations.
  - Insert b-roll per item (neighborhood, foundation, HVAC).
- Caption lines:
  - “1) Neighborhood comps (don’t guess).”
  - “2) Big-ticket items (roof/HVAC).”
  - “3) Resale factors (light, layout, parking).”
- CTA: same as above (website + email).

- B-roll cues:
  - [B-ROLL: slow controlled squat]

FT-02 (Listicle) — “3 cues to instantly improve your squat”

- Edit beats:

  - Use freeze-frame + label for each cue.
- Caption lines:

  - “2) Knees track over toes.”
  - “3) Drive the floor away.”
- CTA: same as above.
