What you’ll receive (Delivery Pack)
For each batch/order, you’ll get a folder containing:
- Final vertical MP4 clips (ready to post)
- Captions file per clip (.SRT)
- B-roll / on-screen cue sheet (simple doc per clip)
- Posting suggestions (caption first line + CTA + hashtags + platform notes)

Our production flow
1) Pre-flight confirmation (before editing begins)
We confirm your platform targets, tone, CTA, branding, and any “do-not-use” words/topics. This prevents rework.

2) Edit + assemble
We create vertical clips with:
- Hook-first structure
- Clean jump cuts (tight pacing)
- Captions with high readability
- Safe framing for faces + UI overlays

3) Review
You’ll get a review link (Frame.io / Vimeo / unlisted YouTube).
- Please leave timestamped comments (e.g., 00:07 “change caption wording”, 00:12 “trim pause”).
- Consolidate feedback when possible.

4) Revisions (24-hour window)
- You have 24 hours from delivery of the review link to request revisions.
- After the revision window closes, we can still help, but changes may be treated as a new request.

QA standards (what we check before anything is sent)
Visual / framing
- Faces not cropped; eyes in upper third when appropriate
- “Safe zones” respected (no critical text in areas covered by UI on TikTok/Reels/Shorts)
- No accidental zoom/crop artifacts
- Consistent styling across the batch (fonts, size, color, highlight rules)

Captions
- High accuracy (names/terms verified)
- Readable line breaks (no awkward splits)
- Punctuation and capitalization consistent
- On-screen captions match the .SRT file (timing and wording)

Audio
- Speech is clear and consistently leveled clip-to-clip
- Loudness normalized; no clipping/distortion
- Obvious mouth noises minimized where feasible

Story/pacing
- Hook clarity in first 1–2 seconds
- Dead air removed; jump cuts feel natural
- Clear payoff (insight, step-by-step, or punchline)

Exports
- Vertical 9:16
- Platform-ready MP4 (H.264), high quality, sensible bitrate

Review tools (how to comment)
- Frame.io/Vimeo: click the frame to drop a comment with a timestamp
- YouTube (unlisted): include timestamps in your comment (e.g., 0:07, 0:12)

Revision policy (scope boundaries)
Included (free within 24h window)
- Caption fixes (typos, timing tweaks)
- Minor trimming/tightening, removing filler words
- Minor reframing adjustments (reposition, scale slightly)
- Audio level tweaks
- Swap a single b-roll insert with another of similar type

Not included (treated as new work)
- Changing the chosen topic/angle after delivery
- Adding new sections from the source that weren’t requested
- Major re-edit across multiple clips due to changed direction
- New branding package mid-project (fonts/colors/style overhaul)

Turnaround expectations
- We confirm the timeline after intake + source delivery.
- If anything blocks work (missing source/brand/CTA), we pause and notify you immediately.
