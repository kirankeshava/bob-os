=== 1) LINK PLACEMENT RULES (IMPORTANT) ===
Goal: Use the website URL as “portfolio/proof” while never asking for off-platform payment.

A) Upwork (safe approach)
- OK: Add legitimacy URL as a portfolio item link or in “Portfolio/Other Experiences” where Upwork allows external links.
- NOT recommended: Posting Calendly/WhatsApp links in initial messages or proposal if it can be interpreted as moving communication/payment off Upwork.
- Best practice: Ask the client to answer key questions inside Upwork. If they request a call, use Upwork’s built-in call/meeting features. Only share Calendly as a “scheduling convenience” after a contract is in place (or if Upwork explicitly allows it in that context).
- Always include: “Payment stays on Upwork.”

B) Fiverr (safe approach)
- Fiverr prefers communication and orders on-platform. External links can be risky.
- OK: Put the website URL as portfolio/proof if Fiverr allows it in your profile/portfolio section.
- Avoid: Telling buyers to book Calendly before ordering. Instead, use Fiverr’s inbox + order requirements.
- Use: Tally intake as an internal “question list” inside Fiverr requirements if possible. If you must send a link, do it only after buyer requests it and make clear the order stays on Fiverr.

C) Carrd
- Carrd can freely link to Upwork/Fiverr order pages and Calendly.
- Primary CTA hierarchy: (1) Order on Upwork (2) Order on Fiverr (3) Book 15-min.
