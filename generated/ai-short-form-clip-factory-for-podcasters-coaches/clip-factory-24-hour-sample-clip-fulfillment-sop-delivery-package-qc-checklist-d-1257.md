## Production SOP (24-hour clock)
### Step 1 — Create Client Folder + Tracker (10 minutes)
Create Drive folder:
- **ClientName_ClipFactory/**
  - 01_Source/
  - 02_Workfiles/
  - 03_Exports/
  - 04_Thumbs_Hooks/
  - 05_Notes_QA/

Naming convention:
- Source: Client_EpisodeTitle_Date.mp4
- Export: Client_Platform_Clip01_HookSlug_v1.mp4

Update tracker row (see tracker schema below).

### Step 2 — Pull Transcript + Identify 8–12 Candidate Moments (30–45 minutes)
Process:
1) Skim transcript or listen at 1.5x.
2) Mark timestamps for moments that match at least 2:
   - Clear takeaway / framework / list
   - Strong emotional contrast (“I used to… then…”) 
   - Controversial/contrarian statement
   - Specific numbers/results
   - “Mistake most people make”
3) Select best 1 moment for sample clip (or top 5–15 for paid batch).

**Selection heuristic for sample clip:** pick the moment with the clearest self-contained payoff in <45s.

### Step 3 — Edit for 9:16 Narrative (45–75 minutes)
- Create 9:16 sequence (1080x1920).
- Cut dead air; tighten pacing.
- Ensure first 1.0–1.5 seconds has a hook line.
- Add jump cuts every 1–3 seconds as needed.
- Add basic camera framing: center face, rule-of-thirds, avoid cutting chin.

**Hook formula options (choose one):**
- “If you’re [audience], stop doing this…”
- “Nobody tells you this about [topic]…”
- “3 signs you’re [problem]…”
- “I wasted X years until I learned this…”

### Step 4 — Captions + On-screen Text (30–45 minutes)
Caption rules:
- Max 2 lines on screen
- 3–6 words per line (roughly)
- Highlight 1–3 keywords per sentence (color or bold)
- Avoid covering face; keep captions lower-third

Add optional top hook text (first 2–3 seconds) and a subtle CTA at end.

### Step 5 — Light B-roll Cues (optional, 10 minutes)
If b-roll is not being inserted, add **b-roll cues as notes** in a text file:
- Timestamp → suggested b-roll keyword
Example:
- 00:07 “calendar blocked” → b-roll: calendar overlay
- 00:18 “client results” → b-roll: graph up/right

### Step 6 — QC + Export (15 minutes)
Export settings:
- 1080x1920
- H.264
- 15–30 Mbps (or “High Quality” preset)
- AAC audio 320 kbps if available

Listen for:
- No clipping/distortion
- Captions match audio (names/terms correct)
- No awkward cut mid-word

## QC Checklist (use every time)
**Hook & pacing**
- [ ] Hook appears within first 1.5 seconds
- [ ] Clip delivers a clear takeaway by the end
- [ ] Dead air removed; pacing feels “tight”

**Visuals**
- [ ] Subject framed well in 9:16 (no cut-off chin/forehead)
- [ ] Captions do not block face
- [ ] No distracting zooms or jitter

**Captions**
- [ ] Accurate spelling (names, brands)
- [ ] 1–2 lines max; readable on phone
- [ ] Keyword highlights used sparingly (not every word)

**Audio**
- [ ] Levels consistent, no peaking
- [ ] Background noise acceptable

**Platform readiness**
- [ ] 1080x1920 export
- [ ] File name follows convention
