3) QA SCORECARD (Internal; also usable with contractors)
========================
Title: DFY Clip Factory — Clip QA Scorecard (v1)
How to use: Each clip must PASS all “Hard Gates”. Then score the “Quality” section. Any clip below 12/18 gets revised before delivery.

A) HARD GATES (PASS/FAIL — must all pass)
1. Duration fits platform intent (typically 15–45s unless client requests otherwise).
2. Hook within first 1.5 seconds (visual + spoken or caption).
3. Captions present and readable on mobile (no UI overlap; safe margins).
4. No profanity/claims/compliance violations per client notes.
5. Audio levels consistent; no clipping; voice clearly intelligible.
6. Export is 1080x1920, high quality, correct frame rate (30fps unless otherwise set).
7. File name follows naming convention (client_episode_clip##_*).

B) QUALITY SCORE (0–3 each; total /18)
1. Hook strength (0 none / 1 okay / 2 good / 3 stops-scroll)
2. Pacing + pattern interrupts (0 dull / 1 some / 2 consistent / 3 excellent every 2–3s)
3. Caption timing accuracy (0 off / 1 minor drift / 2 accurate / 3 tight and polished)
4. Visual clarity (framing, zooms, jump cuts) (0 messy / 1 okay / 2 strong / 3 very strong)
5. Branding consistency (fonts/colors/end card/CTA) (0 none / 1 inconsistent / 2 consistent / 3 on-brand premium)
6. Ending/CTA effectiveness (0 abrupt / 1 weak / 2 clear / 3 compelling)

C) QUICK FIX CHECKLIST (If score <12)
- Tighten first 2 seconds: move strongest line earlier; add bold on-screen hook.
- Increase interrupt rate: zoom 105–115%, cut dead air, add 1–2 b-roll cues.
- Simplify captions: fewer words per line, stronger highlights, avoid covering faces.
- Improve ending: add end card (0.8–1.2s) with single CTA.

D) DELIVERY READINESS
- If all Hard Gates pass AND score ≥12/18 → Approved for delivery.
- If any Hard Gate fails → Must revise.

End of templates.
