## 3) 24-hour Free Sample Clip Fulfillment SOP (internal)
**Goal:** Deliver 1 high-quality sample clip fast, then immediately upsell a DFY package.

### Inputs to request (minimum)
1) YouTube link (or file) + permission to download
2) Target platform: TikTok / IG Reels / YT Shorts
3) Brand preferences: font/color (optional)
4) CTA preference: “Follow for more”, “Listen to full ep”, or none
5) Any must-include keyword/topic (optional)

### Clip selection (10–20 minutes)
- Skim transcript (or 1.5–2.5x playback) for: contrarian statement, clear promise, tactical list, strong opinion, emotional beat.
- Target length: **20–45 seconds** (default). Up to 60s if the segment needs it.
- Start with the punchline; remove intro fluff.

### Edit checklist (45–90 minutes)
- Format: **9:16, 1080×1920**, 30fps.
- Safe margins: keep captions and faces away from top/bottom UI zones.
- Hook: add on-screen hook in first 1–2 seconds (short sentence).
- Captions: high-contrast, 2 lines max, highlight key words.
- Pace: remove dead air >250ms, add subtle jump cuts.
- Optional: light zooms (5–10%) on emphasis, no over-editing.

### Audio + visual QC (10 minutes)
- Ensure captions match spoken words.
- Ensure no clipping/distortion; normalize if needed.
- Check readability on mobile (preview at 25–35% size).

### Export + delivery (10 minutes)
- Export MP4 (H.264), file name: {{Brand}}_SampleClip_{{Topic}}.mp4
- Deliver by email with:
  - Attached file or share link
  - 1 suggested caption (post copy)
  - 3 hashtag suggestions (optional)
  - 1 recommended posting time window (optional)
