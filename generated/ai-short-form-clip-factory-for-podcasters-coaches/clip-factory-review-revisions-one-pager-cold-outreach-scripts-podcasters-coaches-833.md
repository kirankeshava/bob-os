Title: How to Review Your Clips + Request Revisions (Fast)

Thanks for working with Clip Factory. To keep our 48-hour turnaround fast and painless, please review your clips using the steps below.

Official business page (for reference): https://6ff73b98-dcff-4f53-a10d-9ad392f63342-00-1n4h4nuiq6w8j.picard.replit.dev/sites/3
Support / delivery contact: agent_bob_replit+clip-factory@agentmail.to

1) What you’ll receive
- 10 vertical clips (9:16) ready to post
- Burned-in captions + styling
- Hook-forward edits (hook in first ~1.5 seconds)
- Platform-safe formatting (TikTok / IG Reels / YouTube Shorts)
- A “Clip Notes” text file or doc with:
  - Clip titles
  - Timecodes from the original episode
  - Suggested caption text
  - Optional hashtag ideas

2) How to review (7–10 minutes total)
Please answer these 3 questions per clip:
A) Would you post this? (Yes/No)
B) If “No,” what is the single biggest issue?
   - Hook weak / starts too slow
   - Wrong highlight / context missing
   - Captions hard to read
   - Too long / too short
   - Not on-brand
   - Other (describe)
C) What exact change do you want?
   - “Trim first 0.8s”
   - “Start at 12:43 instead”
   - “Change first caption line to: ____”
   - “Remove the word ‘____’ from captions”

3) Revision request format (required)
To request revisions, reply to the delivery email and paste this:

CLIP FILE NAME:
REQUEST TYPE: (Hook / Cut / Captions / Style)
EXACT CHANGE:
OPTIONAL: Reference timestamp in the clip (e.g., 00:03–00:05)

This format prevents back-and-forth and keeps turnaround fast.

4) What counts as a revision (what we’ll change)
Included revisions focus on execution fixes:
- Tightening the hook (trim/replace first seconds)
- Adjusting cut points to improve clarity
- Caption readability fixes (typos, line breaks, timing)
- Minor style tweaks within the chosen preset (position/size/color within reason)
- Volume leveling / removing obvious dead air

5) What does NOT count as a revision (new scope)
These requests typically require new selection/editing cycles:
- Switching to a different section/topic of the episode (new highlight)
- Major re-branding (new fonts/colors not in the chosen preset)
- Adding extensive custom b-roll sourcing
- Turning clips into fully-produced ads with new scripts/voiceover

If you want any of the above, we’ll quote it clearly before doing extra work.

6) Timing (so you can plan content)
- First delivery: within 48 hours of receiving working source files + brand/style inputs
- Revisions: typically 24 hours for up to 2 clips (batch revisions may take longer; we’ll confirm)

7) Approval = ready to post
Once you reply “Approved” (or confirm no revisions), you can post immediately. If you want, we can also suggest a posting schedule.
