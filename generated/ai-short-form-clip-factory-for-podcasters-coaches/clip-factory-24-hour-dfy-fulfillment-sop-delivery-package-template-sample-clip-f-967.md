## Goal
Deliver DFY vertical short-form clips for podcasters/coaches in <24 hours from asset receipt, with consistent quality, minimal revisions, and clear upsells to recurring packages. This SOP is designed to get to $5k DFY revenue fast (SaaS build remains gated).

## Inputs required (from client)
Request via email: agent_bob_replit+clip-factory@agentmail.to
1) Source: YouTube link OR uploaded file (MP4) OR podcast episode link
2) Target platforms: TikTok / Reels / Shorts (default all)
3) Brand kit: logo (optional), colors, fonts (optional)
4) Caption style preference: clean / bold / kinetic (default bold clean)
5) Hook style: curiosity, contrarian, list, story (default curiosity)
6) CTA + link: “Follow for more”, “Book a call”, etc.
7) Any forbidden topics/words

## Standard DFY deliverables (baseline)
- 5–15 clips (depending on package) in 9:16, 1080x1920, H.264 MP4
- Burned-in captions (word-highlight optional)
- Hook title (on-screen + in filename)
- Optional: b-roll cue notes (if client wants to self-edit long-term)
- Delivery message with posting suggestions

## Tooling (fast, low-cost)
- Transcription: Whisper (local or API via existing tools) OR Descript trial/free where possible
- Clip selection: manual scan + transcript search (keywords/heat moments)
- Editing: CapCut Desktop or Premiere (whatever is available) + template project
- Captions: CapCut auto captions, then quick cleanup
- Assets: Google Drive folder

## Time-boxed 24-hour workflow
### T+0 to T+30 min — Intake + Confirmation
1) Acknowledge receipt within 30 minutes.
2) Confirm: number of clips, turnaround, target platform, and caption style.
3) Create a Drive folder using template below.
4) Log job in CRM with: client name, episode link, deadline, package.

Client confirmation email (copy/paste):
Subject: Got it — clips in 24 hours
“Thanks! I’ve got the link/files. I’m creating [X] vertical clips for TikTok/Reels/Shorts with [caption style]. I’ll deliver first batch within 24 hours here: [Drive link]. If you have a preferred hook tone (curiosity/contrarian/story/list) reply with one word — otherwise I’ll default to curiosity.”

### T+30 min to T+2.5 hrs — Transcript + Moment Mining
1) Generate transcript.
2) Identify 8–25 candidate moments (depending on package) using:
   - Pattern cues: strong opinion, quick steps, surprising stat, story pivot, emotional emphasis, audience pain
   - Keyword search: “mistake”, “don’t”, “secret”, “framework”, “step”, “here’s how”, “the reason”, “most people”, “what I’d do”
3) For each candidate moment, write a 6–10 word hook title.
4) Pick final set prioritizing: standalone clarity + short setup + punchy payoff.

### T+2.5 to T+6 hrs — Assembly Edit (Template-First)
Use a consistent template to reduce time:
- Layout: speaker centered, safe margins for UI
- Hook text: top 15–20% of frame, 3–8 words
- Captions: lower third, 2 lines max
- Jump cuts: remove dead air >250–400ms
- Loudness: normalize to platform-friendly (avoid clipping)

Checklist per clip (2–6 minutes each, depending on complexity):
1) Set sequence to 9:16 1080x1920.
2) Crop/auto-reframe.
3) Tighten start: remove greetings, start on punchline or tension.
4) Add hook text overlay.
5) Add captions; fix names/terms; keep punctuation minimal.
6) Add subtle progress cuts; ensure payoff within first 15 seconds.
7) Export with naming conventions.

### T+6 to T+10 hrs — Caption Polish + QC
QC checklist:
- First 1.5 seconds: scroll-stopping (no slow intro)
- Captions: legible, not covering face, no more than 2 lines
- No typos on key terms
- Audio balanced
- Ends with clean cutoff or CTA
- File plays on mobile

### T+10 to T+24 hrs — Delivery + Light Revisions Window
1) Upload to Drive folders.
2) Send delivery note + posting guidance.
3) Offer 1 revision round: “minor text/caption timing tweaks” within 48 hours.

Filename rules:
- Use short hook slug in ALL CAPS words separated by hyphen
- Include clip number and aspect ratio
Example: ACME_Ep12_DONT-DO-THIS-IN-SALES_03_9x16.mp4

PostingNotes.md (copy/paste):
- Recommended caption: 1 sentence + CTA
- Hashtags: 3–8 niche tags
- Best practice: post 1/day for 7 days; pin top performer
- If one clip spikes, we’ll cut 3 follow-ups from the same episode


## Revision policy (protect time)

- Hook text rewording (same clip)
Not included (new scope):

- Heavy b-roll sourcing
- Full re-editing to different narrative
Offer paid add-on for new scope.
