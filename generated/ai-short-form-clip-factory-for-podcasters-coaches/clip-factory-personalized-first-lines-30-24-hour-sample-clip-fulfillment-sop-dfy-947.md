2) Intake (collect in one message)
Ask for:

- Target platform (TikTok/Reels/Shorts)
- Topic/episode title + audience
- Preferred CTA (follow, comment keyword, book call)

3) Selection Rules (choose the right moment)
Pick a 15–35s segment that has:
- A clear opening hook in the first 1–2 seconds (question, contrarian statement, surprising stat)
- One idea only (no rambling)
- A “why it matters” payoff by the end
If no perfect moment exists, create the hook as on-screen text (e.g., “Most [NICHE] get this wrong…”) but keep spoken audio intact.

4) Editing Spec (what ‘good’ looks like)
- Format: 1080x1920 (9:16)
- Safe margins: keep captions above bottom UI, avoid top notch areas
- Cuts: remove dead air; add subtle punch-ins every ~3–5 seconds if appropriate
- Captions: high-contrast, 2 lines max, highlight 1–3 keywords per sentence
- On-screen hook: 6–10 words max, matches the segment’s core claim
- Add a single b-roll cue note (timestamp + suggestion) if it improves clarity (e.g., “Add quick b-roll of X when mentioning Y”)
- Ending: 1-line CTA overlay (e.g., “Want the template? Comment ‘CLIP’”)
