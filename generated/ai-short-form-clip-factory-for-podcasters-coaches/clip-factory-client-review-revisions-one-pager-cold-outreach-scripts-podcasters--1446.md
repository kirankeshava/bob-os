## 1) Client One-Pager: How to Review Clips + Request Revisions (DFY Clip Factory)

**What you’ll receive (typical delivery):**
- 10 vertical clips (9:16) optimized for TikTok / IG Reels / YouTube Shorts
- Burned-in captions + hook-first edits + b-roll cues (when relevant)
- A “Clip List” doc with: clip titles, first-line hook, timestamps, and posting notes

**Where the files will be:**
- You’ll receive a download link or folder link in your delivery email.
- If you prefer, we can deliver via Google Drive/Dropbox.

### Your review process (fastest way)
Please review in this order to keep feedback quick and objective:
1) **Hook / first 1–2 seconds:** Would you stop scrolling?
2) **Message clarity:** Is the takeaway clear by ~8–12 seconds?
3) **Captions:** Any obvious errors or missing words?
4) **Pacing:** Any slow parts that should be tighter?
5) **Brand fit:** Does tone match your audience?

### How to request revisions (copy/paste format)
To keep the 48-hour pace, please submit revisions in this format (one message per clip):
- **Clip File Name:** (example: CF_JSMITH_EP12_C03_v1.mp4)
- **Timestamp(s):** (00:02–00:05)
- **Change Requested:** (example: “Remove first sentence, start at ‘Here’s the mistake…’”)
- **Priority:** Must-change / Nice-to-have

### What counts as an included revision
Included revisions are intended to improve the current clip without changing scope:
- Trimming start/end, tightening pauses, swapping a hook line from the same segment
- Caption typo fixes, punctuation, small caption timing adjustments
- Minor layout adjustments within the chosen style preset
- Volume normalization tweaks

### What is out-of-scope (may require a new clip or add-on)
- Choosing entirely new moments not in the delivered set (new searching)
- Re-cutting the clip around a new storyline or adding major new b-roll sourcing
- Changing the style preset after delivery (fonts/colors/layout) across all clips

### Turnaround + deadlines
- Please send revision notes within **48 hours of delivery** so we can keep momentum.
- We’ll return included revisions within **1 business day** (typically faster).
