- Proof/website: https://6ff73b98-dcff-4f53-a10d-9ad392f63342-00-1n4h4nuiq6w8j.picard.replit.dev/sites/3
- The email includes: (1) a folder link with all exports, (2) a text file/notes with hooks + timestamps, (3) optional captions (.srt) if requested.

What to check (60–90 seconds per clip)
1) Hook effectiveness (first 1–2 seconds): Does it start strong? Any dead air?
2) Captions: spelling, your name/company, key terms, numbers, and any sensitive claims.
3) Framing: face centered, not cropped awkwardly; readable captions on mobile.
4) Flow: pattern interrupts every ~2–3 seconds (zoom/cut/b-roll cue) so it doesn’t feel static.
5) Compliance: no disallowed music, no private info, no unapproved claims.

How to request revisions (the fastest format)
Reply to the delivery email and paste this exact structure:
- Clip File Name:
- Timecode (mm:ss–mm:ss):
- Change request (1 sentence):
- Optional: Replacement wording (exact text):

Examples:
- Clip File Name: CLIP_03_Hook-HowToStopScrolling_v01.mp4
  Timecode: 00:00–00:02
  Change request: Remove the first half-second of silence so it hits immediately.

- Clip File Name: CLIP_07_RealEstate-Offer_v01.mp4
  Timecode: 00:08–00:12
  Change request: Caption says “$50k” but should be “$15k.” Please correct.

What counts as an included revision (keeps turnaround fast)
Included (within revision allowance):
- Trim/cut adjustments, tighter hook, removing pauses
- Caption text corrections, minor styling tweaks (size/position)
- Swap b-roll cue to a different option, remove a cue
- Adjust end CTA text (subscribe/follow/book call)

Not included (would be a new request/next batch)
- Rewriting the entire clip concept or changing the core message
- Building new clips from new segments not in the original 10
- Complex motion graphics, custom animation, advanced sound design

Timing
- You’ll receive the initial batch within 48 hours of receiving usable long-form + brand inputs.
- Revisions are turned around ASAP; typical is same-day/next-day depending on volume and clarity.

If you want us to match your style more closely next batch
Reply with 1–2 example links you love (TikTok/Reels/Shorts) and note:
- Caption style (word-by-word vs phrase blocks)
- Energy level (calm vs aggressive)
- CTA preference (follow/comment/DM/book)
