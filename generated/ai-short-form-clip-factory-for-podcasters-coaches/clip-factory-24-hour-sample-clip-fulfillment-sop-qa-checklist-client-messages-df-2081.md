## Goal
Deliver a single, high-quality vertical clip within 24 hours of receiving assets to (1) prove value fast, (2) close DFY packages, and (3) collect product signals for the future $99–$299/mo SaaS—without building anything yet.

## Scope (What the sample includes)
- 1 vertical clip (9:16) from an existing podcast/video episode
- Length: 20–45 seconds (optimize for hook + payoff)
- Burned-in captions (word-by-word or phrase-by-phrase)
- Basic styling: clean template, readable typography, safe-zone compliant
- Export formats: MP4 (1080x1920), plus optional SRT caption file

## What we ask the client for (Asset checklist)
Client can reply to the email below and provide:
1) Source link (YouTube/Spotify/video file link) + timestamp range if they have it
2) Platform target (TikTok / IG Reels / YouTube Shorts)
3) Brand assets (optional): font preference, brand colors, logo (PNG), examples they like
4) Caption style preference: (A) word-by-word, (B) 2–4 word chunks, (C) sentence style
5) Call-to-action text (optional): “Follow for X”, “DM ‘CLIPS’”, etc.
6) Any words to avoid / compliance constraints

## 24-hour SLA Workflow (Internal)
### Step 0 — Intake + clock start (0–15 minutes)
- Log deal in CRM with: Name, handle, niche, episode link, deadline, platform target.
- Confirm assets received. If missing, send “missing assets” follow-up immediately.

### Step 1 — Pull best 10 minutes (15–45 minutes)
- Skim episode for 2–4 candidate moments:
  - Strong opinion, contrarian take, quick win, mistake-to-avoid, story pivot.
  - Prefer segments with clear nouns/verbs and minimal crosstalk.
- Choose 1 segment with:
  - Hook in first 1–2 seconds
  - Payoff within 20–45 seconds
  - Minimal required context

### Step 2 — Transcript + caption draft (45–90 minutes)
- Generate transcript (tool-agnostic): YouTube transcript / Whisper / existing captions.
- Edit for readability (remove filler, keep meaning).
- Create caption pacing: emphasize 1–3 keywords per line.
- Identify 1–3 “b-roll cues” (optional) to visually reset every ~2–3 seconds.

### Step 3 — Edit + template apply (90–240 minutes)
- Build 9:16 layout:
  - Speaker centered, headroom safe
  - Captions in lower third but above UI safe zone
  - Add progress bar (optional) and subtle background blur if needed
- Add simple motion: punch-in 5–8% on emphasis beats.
- Audio:
  - Normalize and lightly compress; remove harsh peaks.

### Step 4 — QA + exports (240–300 minutes)
- Run QA checklist (below).
- Export MP4 1080x1920.
- Export captions as SRT (optional but recommended).

### Step 5 — Delivery + conversion (300–360 minutes)
- Email the clip with:
  - Download link
  - Suggested caption copy (1–2 variants)
  - Suggested posting time (if known) and CTA
  - Ask: “Want 10–20 of these per month?” and propose package.

## Quality Assurance Checklist (must-pass)
### Visual
- [ ] 9:16, 1080x1920, no black bars unless stylistic and intentional
- [ ] Speaker face not cropped; eyes/forehead within safe area
- [ ] Captions do not overlap platform UI (bottom & right safe zones)
- [ ] Text contrast passes: readable on mobile at arm’s length
- [ ] No spelling errors in first 2 lines (highest drop-off risk)

### Hook + pacing
- [ ] Hook lands within 0–2 seconds (no long lead-in)
- [ ] Dead air removed; no awkward mid-word cuts
- [ ] Pattern interrupts every ~2–3 seconds (cut, zoom, b-roll, text emphasis)

### Captions
- [ ] Captions match audio (no major mismatches)
- [ ] Key words emphasized consistently (bold/color if used)
- [ ] Line length not overcrowded (ideally 2–6 words per line)

### Audio
- [ ] Loudness consistent; no clipping
- [ ] Music (if any) under dialogue; not distracting

### Export
- [ ] MP4 plays on mobile; file size reasonable (<150MB typically)
- [ ] Filename formatted: Client_Episode_Topic_9x16.mp4

## SaaS Validation Signals to Capture (1 minute after delivery)
Log in CRM:
- Did they ask for: automatic highlights, brand templates, speaker labeling/diarization, faster turnaround, bulk processing?
- How many clips/month do they want?
- Willingness to pay range: $99 / $199 / $299+ per month
- Must-have output: MP4 only vs MP4 + SRT + hook/caption suggestions
