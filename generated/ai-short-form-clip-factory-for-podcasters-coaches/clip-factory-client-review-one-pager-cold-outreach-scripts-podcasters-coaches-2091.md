# How to Review Your Clips + Request Revisions (Fast)

**Who this is for:** Clients receiving their first batch of clips.

**Your delivery will include**
- 10 vertical clips (MP4) sized for TikTok / IG Reels / YouTube Shorts
- Captions burned in (plus safe margins)
- A simple “Clip List” (Google Doc or message) with: clip titles, timestamps from the source, and 1-sentence posting notes

**How to review (10–15 minutes total)**
1) **Watch at 1.25x** if you’re short on time. Focus on the first 2 seconds of each clip.
2) For each clip, choose one of three outcomes:
   - **APPROVE** (ready to post)
   - **MINOR EDIT** (quick change, no new creative direction)
   - **KILL** (don’t use this clip)
3) **Batch feedback** in one message (preferred). This protects the 48-hour turnaround.

**What counts as a “Minor Edit” (included)**
- Fix a typo or caption wording
- Adjust caption position slightly (higher/lower)
- Replace 1–2 words in the on-screen hook text
- Remove a sentence or shorten the ending
- Swap emoji intensity (more/less)
- Quick audio level tweak

**What counts as a “Revision / New Direction” (not included in the free trial)**
- “Use a completely different moment/topic” (new clip selection)
- New brand style system (fonts/colors/layout redesign)
- Heavy re-editing (restructure, new storyline, extensive b-roll, custom animation)

**Revision format (copy/paste this)**
Subject/Message: **Revisions — [ClientName] — [EpisodeName]**

For each clip:
- **Filename:** (example: `CLIP-03_Tax-Loop_HOOK-A_v1.mp4`)
- **Decision:** APPROVE / MINOR EDIT / KILL
- **Timecode (if needed):** 00:00–00:14
- **Change request (one sentence):** “Change hook line to: ‘Here’s the #1 mistake…’ and move captions 10% higher.”

**Response time + turnaround**
- If we receive consolidated feedback within **24 hours**, minor edits ship within **24 hours**.
- If feedback comes in piece-by-piece across days, turnaround may extend (we’ll still be fast, but the clock can’t start until feedback is consolidated).
