# 24-Hour Fulfillment Timeline (SOP)
### T+0 to T+15 min — Payment Confirmed → Send Intake
1) Confirm payment (invoice/Stripe/manual). 
2) Immediately send “Intake Request” email (template below).
3) Create job in tracker with:
   - Client name + episode link
   - Due time (24h from asset receipt)
   - Package (5 clips / 10 clips / weekly)
   - Brand style notes (if any)

### T+15 to T+60 — Intake Validation + Source Capture
4) Validate assets received:
   - YouTube link OR raw file
   - Target platforms (TikTok/Reels/Shorts)
   - Audience + offer + CTA
   - Brand kit (fonts/colors) if applicable
5) Download source video/audio.
6) Generate transcript (fastest method available).

### T+60 to T+180 — Highlight Selection (fast decision rules)
7) Identify 8–12 candidate moments, choose best 5.
   - Look for: contrarian statement, list/steps, mistake-to-avoid, quick story with payoff, “most people get this wrong.”
8) Clip length targets:
   - TikTok/Reels: 20–45s preferred; up to 60s if удержание (retention) is strong
   - YouTube Shorts: 25–55s

### T+180 to T+600 — Edit Assembly (repeatable template)
9) For each clip:
   - Hard hook in first 1–2 seconds (either native quote or added text hook)
   - 9:16 framing, speaker centered, tighten silences
   - Captions on, with emphasis styling (2–5 words max highlighted at once)
   - Add b-roll cues only if it increases clarity (don’t over-produce)
   - Add simple progress bar only if it helps retention (optional)

### T+600 to T+780 — QC + Fixes
10) Run QC checklist (below).
11) Fix caption errors, crop issues, audio leveling.

### T+780 to T+900 — Export + Delivery
12) Export:
   - MP4 1080×1920, H.264, 10–16 Mbps
   - Optional: SRT
13) Upload to Drive folder, set sharing to “Anyone with link can view.”
14) Send delivery email with:
   - Folder link
   - Clip titles + suggested captions/hashtags
   - 1 question to reduce revisions (“Any preferred hook style for next batch?”)

# QC Checklist (must pass before delivery)
## Content/Retention
- Hook lands in first 2 seconds (clear curiosity or promise)
- No long dead air at start/end
- Clip has a single idea (not 3 topics smashed together)

## Visual
- Correct 9:16 crop; face not cut off during gestures
- Safe margins: captions not hidden by TikTok/Reels UI (keep low-third but above bottom UI)
- Consistent template across all clips in batch

## Captions
- Spelling/grammar clean (names/terms checked)
- Caption timing matches speech (no lag > ~250ms)
- Emphasis styling not distracting; highlights key words only

## Audio
- Level consistent across clips (no sudden jumps)
- Background noise acceptable; no clipping

## Exports
- File names consistent: Client_EpDate_Clip01_HookTitle.mp4
- Drive permissions correct
