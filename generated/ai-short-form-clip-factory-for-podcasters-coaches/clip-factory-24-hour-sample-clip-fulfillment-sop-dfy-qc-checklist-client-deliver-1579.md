GOAL
Deliver a paid “sample clip” (or first batch) within 24 hours with minimal back-and-forth, consistent formatting, and a clear path to upsell into weekly/monthly DFY packages (and later SaaS).

A) DEFINITIONS (WHAT WE DELIVER)
1) Sample Clip Deliverable (default)
- 1 vertical 9:16 MP4 clip (20–45s unless otherwise requested)
- Burned-in captions (word-highlight optional)
- Hook polishing (first 1–2 lines) aligned to client voice
- Optional: 1 alternate hook text (as a note)
- Optional: B-roll cues list (timestamps + suggested overlay)

2) Full DFY Batch (if sold)
- 5 / 10 / 20 clips per week as purchased
- Mix of: “insight,” “story,” “tactical steps,” “contrarian take”

B) INTAKE (0–30 MIN AFTER PAYMENT)
Send Intake Request Email (Template #1 below). Collect:
- Source: YouTube link / podcast episode link / raw file upload link
- Target platform(s): TikTok / IG Reels / YouTube Shorts
- Audience + niche (1 sentence)
- Brand kit: logo (optional), colors (hex), font preference (or “choose for me”)
- Caption style: clean / bold / karaoke-highlight / minimal
- CTA: follow / comment keyword / book call / download lead magnet
- “Do not include” list: banned topics/words, compliance rules
- Examples they like (2–3 links)

Client-facing promise: “If you send the link + preferences today, you’ll have the sample clip within 24 hours.”

C) FULFILLMENT WORKFLOW (TARGET: 2–3 HOURS HANDS-ON)
1) Download / Acquire Source
- If YouTube: use internal downloader process or request the client’s original file if quality is poor.
- If podcast audio only: ask for video if they want vertical; otherwise create audiogram-style vertical with waveform + titles.

2) Transcribe
- Run transcription (Whisper or other). Output SRT + plain text.
- Quick scan for proper nouns (names, brands). Fix obvious errors in the final captions.

3) Identify Highlights (15–25 minutes)
Choose segments using this hierarchy:
- Strong hook in first 1–2 seconds OR easy to rewrite as hook
- Clear, standalone value without needing long context
- Emotion / contrarian take / list format performs best
- Avoid long rambling setups; cut tight

Target clip types:
- “3 steps to…”
- “Most people get X wrong…”
- “If you’re doing X, do this instead…”
- “Here’s the framework…”

4) Script the Hook + On-Screen Text
- Rewrite the first caption line to be scroll-stopping but truthful.
- Keep on-screen text <= 8 words per line, 1–2 lines.
- If client tone is unknown: default to clear, business-casual.

5) Format 9:16
- Use 1080x1920.
- If source is widescreen: create smart crop focusing speaker; keep eyes in top third.
- If multiple speakers: cut between them; don’t leave dead air.

6) Captions
- Burn in captions with high contrast.
- Safe zone margins: leave space at bottom (platform UI). Keep key text mid-screen.
- Highlight style (optional): emphasize current word or keywords.

7) Optional B-roll Cues (lightweight)
- Provide a short list: timestamp → suggested overlay (stock/b-roll, screenshot, headline text).
- Do not overproduce unless client bought “B-roll add-on.”

8) Export + Package
- Export MP4 H.264, 15–30 Mbps.
- File name convention:
  [Client]_[EpisodeOrTopic]_[Clip#]_[Length]_[Date].mp4
  Example: AcmeCoach_ColdOutreach_Clip01_34s_2026-04-09.mp4
- Include:
  - MP4
  - SRT file (optional but recommended)
  - Caption copy (plain text) + 3 suggested post descriptions + 10 hashtags (optional)

D) QUALITY CONTROL CHECKLIST (DO THIS EVERY TIME)
Video
- [ ] 9:16 (1080x1920), not stretched
- [ ] Speaker framed well; no excessive head cut-off
- [ ] Audio clear; normalized; no sudden volume spikes
- [ ] No dead air > 0.3–0.5s at start

Captions
- [ ] Spelling of names/brands correct
- [ ] Captions do not cover face; respects safe zones
- [ ] Words match audio (no glaring drift)
- [ ] Hook caption strong within first 1–2 seconds

Content
- [ ] Clip is standalone (doesn’t rely on missing context)
- [ ] No sensitive claims that could get flagged (health/finance) unless client approved

Packaging
- [ ] Google Drive/Dropbox link permissions set to “Anyone with link can view/download”
- [ ] Email includes next-step upsell (weekly clips)

E) REVISION / REDO POLICY (KEEP IT SIMPLE)
- Sample clip includes: 1 minor revision round (captions wording, hook line, crop tweak).
- A full re-edit due to changed direction (new style references after delivery) is billable.
- If we made an error (mis-captioning, wrong crop, wrong file): fix free within 12 hours.
