## 3) 24-Hour “Sample Clip Offer” Fulfillment SOP (Internal)

Goal
Deliver 1 high-quality vertical clip within 24 hours of receiving a prospect’s link, then convert them into a paid DFY package.

Tools (free/low-cost preferred)
- Source: YouTube link / Drive link
- Editing: CapCut Desktop or Premiere/Resolve (whatever is available)
- Captions: CapCut auto-captions or Whisper-based captioning (if already available)
- Delivery: Google Drive folder link (or Dropbox)
- Comms: agent_bob_replit+clip-factory@agentmail.to

Intake checklist (must-have)
1) Episode link (YouTube/Drive)
2) Target platform(s): Reels / TikTok / Shorts
3) Audience & topic angle (1 sentence)
4) Style references (optional): 1–2 creators they like
5) Brand kit (optional): font/color/logo

Process (timeboxed)
0) Log request (5 min)
- Create a row in CRM: Name, email, link, niche, date/time received, deadline (+24h), status.

1) Pull transcript & identify 3 candidate moments (25–35 min)
- Skim transcript for:
  - contrarian point
  - clear “how-to” step
  - story with a punchline/lesson
  - a strong outcome (“I doubled…”, “This saved…”, “The mistake is…”) 
- Select the best moment with a clear first line.

2) Cut for retention (20–30 min)
- Target 20–45 seconds for the sample.
- Remove pauses, tighten sentences, cut dead air.
- Ensure the first 1–2 seconds contains the hook.

3) Format 9:16 (10–15 min)
- If talking head: center subject, crop for face, add safe margins.
- If podcast: use waveform + headline text + b-roll cues (simple).

4) Captions + emphasis (20–30 min)
- Burn-in captions.
- Use 2–4 word emphasis styling for key phrases.
- Keep line length short; avoid covering the face.

5) Lightweight polish (10–15 min)
- Normalize audio.
- Add subtle background blur if needed.
- Optional: add a simple title card (0.5–1 sec) if it doesn’t hurt retention.

6) Export + delivery (10 min)
- Export settings: 1080x1920, 30fps, H.264, high bitrate.
- Create a Drive folder: “{ClientName} — Sample Clip — {Date}”.
- File name: “{ClientName}_SampleClip_9x16.mp4”.

7) Delivery email + upsell (5–10 min)
Subject: Your sample clip is ready
Body:
- Share Drive link
- Ask 2 quick questions:
  1) Want captions more bold/minimal?
  2) Should we optimize for Reels, TikTok, or Shorts first?
- Offer next step:
  “If you want 10 clips like this from your next episode, we can start this week. Starter is $499 for 10 clips (48h turnaround).”
