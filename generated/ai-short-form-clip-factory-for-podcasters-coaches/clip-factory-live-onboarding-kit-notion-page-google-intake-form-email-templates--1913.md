1) What you’ll receive (Delivery Pack)
For each batch/order, you’ll receive a “Delivery Pack” containing:
- Final MP4 files (vertical, platform-ready)
- Captions file for each clip (SRT)
- B-roll / on-screen cue sheet (per clip) so you can re-create or enhance visuals later
- Posting suggestions (per clip):
  - Suggested first-line caption (hook)
  - Suggested hashtags
  - Best-fit platform(s) + posting notes

2) Our QA standards (what we check before you ever review)
We run a QA pass on every clip to reduce revisions and protect your brand.

Visual (safe zones + framing)
- No cropped faces (forehead/chin not cut; eyes remain clearly visible)
- Subject remains centered and readable on mobile
- Text stays within “safe zones” (not blocked by TikTok/Reels UI)
- No distracting jump framing or accidental zooms

Captions
- Captions match spoken words (no meaning-changing errors)
- Correct punctuation for readability
- Key words highlighted only when it improves comprehension
- Profanity/“banned words” handled per your intake preferences

Audio
- Voice is consistently audible (no sudden volume drops)
- Background noise reduced where possible
- No clipping/distortion

Edit rhythm
- Hook is clear in first 1–2 seconds
- Cuts feel intentional (remove dead air, repeated takes, long pauses)
- Clip ends cleanly (no awkward cut-off mid-word)

Export settings (platform-ready)
- Vertical 9:16 (1080x1920)
- H.264 MP4
- Target 30fps (or match source if necessary)

3) Review process (timestamp comments)
You’ll get a single review link where you can leave timestamped comments.

Accepted review methods (free + fast):
- Frame.io review link (preferred)
- Unlisted YouTube link
- Vimeo review link

How to leave feedback (required format)
- Use timestamps (example: “00:07 change caption line to…”, “00:12 cut 0.5s earlier”)
- Group feedback per clip if there are multiple clips
- If a note applies to the whole batch, say so explicitly

4) Revision window (24 hours)
To keep delivery fast and predictable, we include a 24-hour revision window:
- The 24-hour timer starts when the review link + delivery pack is sent.
- If we don’t receive revision notes within 24 hours, the delivery is considered accepted.

5) What counts as a FREE revision vs. a change request
Free revisions (included)
- Fixing our mistakes: typos, misheard captions, export glitches
- Framing adjustments if a face/text is cropped incorrectly
- Audio leveling fixes if voice volume is inconsistent
- Implementing reasonable timestamped edit tweaks that keep the same source segment/topic

Change requests (not included in the free revision window)
- Swapping to different source segments (“use a different 2 minutes”)
- New hooks, new script direction, new CTA not in the original request
- Re-cutting the entire clip after you change your mind on the concept
- Adding brand-new assets not provided during intake (logos, brand kits, b-roll libraries)

If you request changes outside the included scope, we’ll confirm the new scope in writing before proceeding.
