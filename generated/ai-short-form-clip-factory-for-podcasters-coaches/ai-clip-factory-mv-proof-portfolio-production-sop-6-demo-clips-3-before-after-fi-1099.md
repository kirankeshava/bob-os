A) Source requirements (permission-friendly)
1) Use one longform source per niche (3 total). Must be Creative Commons, public domain, or explicitly permissioned by the creator. Save the source URL in a text file in the Drive folder.
2) Pick two moments per source (30–60s each). Choose segments with: (a) one clear claim, (b) one actionable takeaway, (c) one emotional beat (surprise/relief/curiosity).
3) Keep the speaker’s message intact; this is a demo of editing/captioning/packaging quality.

B) Export standards (non-negotiable)
Canvas: 1080x1920 (9:16). FPS: 30. Codec: H.264. Audio: 48kHz AAC.
Safe zones: Keep all text inside center 80% height to avoid UI overlays.
Captions: Burned-in, high contrast, 2 lines max, 4–7 words per line.
Pacing: Cut every 1.0–2.5 seconds (jump cuts, punch-ins, or b-roll swaps).
Branding: Small footer watermark text: “AI Clip Factory (Demo)” + email.
CTA end card (last 1.0–1.5s): “Want 30 clips/mo? See samples: [website URL] | Email: [email]”.

C) Clip types (ensure variety across the 6)
You must cover these 4 styles across the set: 
1) Storytelling hook (open loop → payoff)
2) Listicle (3–5 rapid points)
3) Contrarian take (challenge a common belief)
4) Testimonial-style (results framing; if no actual testimonial exists, format it as a ‘case-style narration’ using the speaker’s own phrasing)
Recommended distribution across niches:
- Real Estate: Story hook + Contrarian
- Fitness: Listicle + Contrarian
- B2B: Story hook + Testimonial-style/case-style

D) Edit recipe (CapCut Desktop, free)
1) Create project → set ratio 9:16.
2) Import the longform video → duplicate the clip on timeline (track A) for safety.
3) Cut to the selected 30–60s excerpt.
4) Build hook stack (first 2 seconds):
   - Add big headline text (6–10 words). Example pattern: “Most agents lose deals because…” / “Stop doing this in your workouts…” / “Your cold emails fail because…”
   - Add subtle punch-in (110% → 120%) and a whoosh or click SFX at 3–6% volume.
5) Captions:
   - Auto-captions if available; otherwise manual. Correct 100% of numbers, names, and key terms.
   - Emphasize 1–2 keywords per sentence using color or bold.
6) B-roll cues:
   - Every 5–8 seconds, insert a cutaway (stock, screenshots, charts, UI mockups) or use a dynamic punch-in + blur background layer.
   - If you can’t use stock, simulate b-roll with kinetic text cards (1–2 seconds): “LISTING PHOTOS”, “PROTEIN TARGET”, “REPLY RATE”.
7) Pattern interrupts (minimum 2 per clip):
   - Quick zoom cut, 2-frame glitch, or text pop.
   - On-screen icon (arrow/circle) to point at a phrase.
8) Loudness:
   - Normalize voice to consistent perceived level; avoid peaking. Add light background bed at very low volume if needed.
9) End card CTA (last 1–1.5s) with website + email.
10) Export using standards in section B.
