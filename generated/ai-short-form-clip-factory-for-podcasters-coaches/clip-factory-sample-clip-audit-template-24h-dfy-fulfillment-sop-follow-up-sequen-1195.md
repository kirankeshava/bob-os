========================
B) 24-HOUR DFY FULFILLMENT SOP (INTERNAL)
========================
Goal: deliver clips fast and consistently, minimize back-and-forth, and create a repeatable factory.

0) Inputs required (do not start without these)
• Source: YouTube link / Drive link / uploaded MP4
• Brand kit (optional): fonts/colors/logo
• Caption style preference: {{Clean / Bold / Karaoke / Minimal}}
• Clip goals: {{lead gen / authority / course sales / podcast growth}}
• Do-not-say list (optional)
• CTA preference: {{follow / comment / DM keyword / link in bio}}

1) Create job ticket (5 minutes)
• Job name: {{Client}}_{{Date}}_{{SourceName}}
• Folder structure:
  /01_source
  /02_transcript
  /03_selects
  /04_exports
  /05_delivery
• Log in CRM: client, package, due time, source link, notes.

2) Transcript + rough segmentation (20–40 minutes)
• Generate transcript (Whisper/auto-caption tool).
• Mark 8–15 candidate moments with timestamps.
• Filter using quick rules:
  - must have a clear claim, story, or actionable steps
  - remove long prefaces; prefer moments that can stand alone
  - avoid inside jokes/context-heavy segments

3) Select final clips (15 minutes)
• Pick the best 3–8 moments depending on package.
• For each clip, write:
  - 1 primary hook line (first 1–2 seconds)
  - 1 backup hook variant
  - 1 suggested title/filename
  - 1 b-roll cue (optional)

4) Edit + format (60–120 minutes depending on volume)
• Convert to 9:16 (smart crop + manual adjustments for faces).
• Tighten pacing:
  - cut dead air
  - add jump cuts
  - remove filler
• Captions:
  - burned in
  - max 2 lines
  - highlight keywords
  - ensure safe margins (no UI overlap)
• Add light polish:
  - subtle zooms/emphasis
  - simple intro text only if requested

5) QC checklist (10 minutes)
• Audio levels consistent
• Captions accurate (names/terms)
• First 1–2 seconds punchy (no slow ramp)
• No visual cropping errors
• Export settings correct (1080x1920, H.264, 30fps unless source dictates)

6) Delivery (5 minutes)
• Upload to Drive/Dropbox folder.
• Provide:
  - MP4s named: {{Client}}_{{Clip#}}_{{Title}}.mp4
  - Optional SRTs
  - A short “posting suggestions” note (best platform + caption suggestion)
