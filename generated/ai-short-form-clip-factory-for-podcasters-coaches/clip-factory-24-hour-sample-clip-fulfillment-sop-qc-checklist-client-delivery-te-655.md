A) What we deliver (default sample clip)
Deliverables (minimum):
1) 1 vertical video (9:16) exported in 1080x1920, H.264 MP4, 30fps (or source fps), target bitrate 8–16 Mbps.
2) Burned-in captions (word-level or phrase-level). Style: high-contrast, 2–3 lines max, big enough for mobile.
3) Hook options: 3 hook text variants to use as on-screen headline + caption copy.
4) Posting bundle: recommended title/caption, 5–12 hashtags, and CTA suggestion.
Optional if available quickly: 1 alternate crop version (face-safe vs. content-safe) or 1 alternate caption style.

6) Do-not-do list (words, tone, topics, any compliance constraints).

C) Fulfillment workflow (time-boxed for 24 hours)
0) Project setup (10 minutes)
- Create a folder structure: /ClientName/Source/Exports/Notes
- Save client’s preferred handle + platforms.

1) Source retrieval (15 minutes)
- Download or access source video.
- Confirm audio quality and speaker visibility.
- If link access fails, email immediately from agent_bob_replit+clip-factory@agentmail.to.

2) Transcript + highlight selection (45–75 minutes)
- Generate transcript (any workflow: manual + auto tools). 
- Pick 1 segment using these criteria:
  - Strong opening in first 1–2 seconds (a claim, question, contrarian take).
  - Single coherent idea in 20–45 seconds (ok up to 60s if dense).
  - Minimal dead air; clear payoff.
- Write 3 hook variants:
  - Hook A (curiosity): “Most people get X wrong…”
  - Hook B (specific): “If you’re [audience], do this in 5 minutes…”
  - Hook C (contrarian): “Stop doing X. Do Y instead.”

3) Edit + formatting (60–120 minutes)
- Canvas: 1080x1920 (9:16).
- Crop: center speaker; keep eyes on upper third; ensure safe margins for captions.
- Add dynamic motion (subtle punch-in cuts) every 2–4 seconds if needed.
- Add captions:
  - 2–3 lines max, avoid covering mouth.
  - Highlight 1–3 keywords per sentence (optional).
  - Remove filler words only if it improves clarity (don’t distort meaning).
- Add simple b-roll cue notes (even if not inserted): e.g., “(B-roll: screenshot of X)” in a separate doc.

4) Export (10–20 minutes)
- MP4, H.264, 1080x1920.
- Check file size and playback.

5) QC pass (10 minutes)
Use the QC checklist below before sending.

6) Delivery + upsell (10 minutes)
- Send delivery email template (below).
- Ask for 1 round of feedback (small changes) + offer next package (weekly clips).

D) QC Checklist (non-negotiable)
Format & technical
- [ ] 1080x1920 (9:16) vertical.
- [ ] Audio is clear, no clipping; voice loud enough on phone speakers.
- [ ] No black bars unless stylistic and approved.

Hook & pacing
- [ ] The first 1–2 seconds contain a clear hook (text and/or spoken).
- [ ] Cuts are tight; no long pauses >0.5–0.8s unless intentional.

Captions
- [ ] Captions are readable on mobile (high contrast, large size).
- [ ] Captions do not cover key facial area; respect safe margins.
- [ ] No major transcription errors (names/claims/numbers double-checked).

Brand & compliance
- [ ] No restricted/unsafe claims for the niche (especially health/finance).
- [ ] Any client do-not-do items respected.

Posting readiness
- [ ] Provide suggested caption + hashtags + CTA.
- [ ] Filename is clear: Client_Platform_Topic_Date_v1.mp4
