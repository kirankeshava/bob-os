1) CLIENT ONE-PAGER — HOW TO REVIEW & REQUEST REVISIONS
==============================
Title: How to Review Your Clips + Request Revisions (Fast)

Where to review
- You’ll receive either (a) a share link to a review page (preferred) OR (b) a folder link containing MP4s.
- Each clip file name includes the clip number and a short topic label (example: “C03_ColdOpen_Offer.mp4”).

What “done” looks like (what we optimized for)
- Hook lands within the first 1.5 seconds.
- Captions are readable on mobile and safe from UI overlays.
- Pacing includes pattern interrupts every ~2–3 seconds (zoom/cut/b-roll cue/word emphasis).
- Ends with a clear CTA or curiosity loop.

How to leave feedback (use this format)
For each clip you want revised, copy/paste:
- Clip #: (example: C03)
- Timestamp: (example: 00:07–00:11)
- Change request (one sentence): (example: “Replace the hook line with: ‘Most ads fail because…’”)
- Optional: Add a reference (link) if you want a specific style.

What counts as a revision (included)
- Caption text corrections (typos, punctuation, missed words).
- Hook swap using nearby wording from the same source segment.
- Tighten pacing (trim pauses, remove filler).
- Minor framing adjustments (safe zone, subtitle position, scale).
- Replace/adjust b-roll cues with similar alternatives.

What does NOT count as a revision (scope boundaries)
- Changing the topic/angle to something not in the original segment.
- New scriptwriting, new voiceover, or adding net-new content.
- Rebuilding clips from completely different timestamps (that’s a new clip request).

Revision rounds + turnaround
- Included: 1 revision round for up to 3 clips (per batch of 10).
- Turnaround: revisions delivered within 24 hours of receiving your notes in the required format.

Fastest approval workflow (recommended)
1) Mark 7 clips “Approved” immediately if they’re good.
2) Request revisions on up to 3 clips using the template above.
3) We deliver revisions within 24 hours.
