A) DAY-1 EXECUTION PLAN (to hit 200 connections + 40 proposals / 7 days)

Block 1 (60–90 min): Upwork prospecting + proposals
1) Open 15–25 posts in tabs using searches: “podcast clips”, “short form editor”, “repurpose content”, “reels from youtube”, “tiktok from podcast”, “captions + subtitles”.
2) Qualify quickly:
   - Has existing longform content link? (Y/N)
   - Wants ongoing output? (Y/N)
   - Mentions 5–20 clips/week or monthly package? (Y/N)
   - Budget > $100 or mentions “long-term” (Y/N)
3) Pick best 6–8 fits. For each, write a 1-sentence personalization referencing their niche + deliverable.
4) Submit proposal using the closest variant below.
5) Log immediately in CRM: date/time, job URL, budget, variant used, personalization line, status=“Proposal Sent”, next action = “Follow up Day+2”.

Block 2 (75–120 min): Creator outbound (YouTube/IG/TikTok)
1) Choose 10 YouTube channels + 10 IG/TikTok creators (30 total touches can include email-only leads).
2) For each YouTube lead:
   - Watch 2–4 minutes of latest long episode.
   - Leave 1 value comment (no pitch) referencing a specific moment.
   - Then email/DM with: the moment + the offer (“1 free sample clip”).
3) For IG/TikTok:
   - React to a recent post in DM (1 line) + the offer.
4) Log each touch: channel, link, personalization hook (episode title + timestamp), status=“Messaged”, next action = “Bump Day+2”.

Block 3 (30 min): Follow-up sweep
- Sort CRM by “Next action due today”.
- Send follow-up #1 to all Day-2 items.
- Update status.

Daily KPI rollup (5 min):
- Connections/DMs sent today
- Upwork proposals sent today
- Replies received
- Calls booked

D) CRM LOGGING SOP (so we don’t lose pipeline)

Required columns:
- Lead Name
- Segment (YouTube / IG / TikTok / Upwork)
- Profile/Job URL
- Subs/Followers (if applicable)
- Niche
- Personalization hook (episode title + timestamp OR post reference)
- Last touch date
- Touch type (Comment / DM / Email / Proposal)
- Offer (Free sample / 10 clips in 48h / Monthly)
- Status (New, Messaged, Proposal Sent, Replied, Call Booked, Won, Lost, No Reply, Bumped)
- Next action date
- Next action notes

Logging rules:
1) If it’s not logged, it didn’t happen.
2) Every outbound message gets a next-action date immediately (Day+2 by default).
3) Replies update status the same day (Replied → Call Booked or Replied → Negotiating).
4) KPI rollup daily: total outreaches sent, total proposals sent, replies, calls booked.

Done correctly, this creates a compounding follow-up engine while we continue to add new leads daily.
