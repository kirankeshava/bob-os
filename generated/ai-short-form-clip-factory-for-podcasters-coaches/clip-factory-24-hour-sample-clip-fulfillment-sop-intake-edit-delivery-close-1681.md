Step-by-step SOP

1) Intake (0–30 minutes)

Body:
“Awesome — I’ll cut a free sample vertical clip in the next 24 hours.

2) Target platform (TikTok / Reels / Shorts)
3) Your audience (who is it for?)
4) CTA preference (follow / book a call / download / comment)
5) Caption style: (A) clean white, (B) bold w/ highlighted keywords

If helpful, here’s our site for reference: https://6ff73b98-dcff-4f53-a10d-9ad392f63342-00-1n4h4nuiq6w8j.picard.replit.dev/sites/3
Any questions, reach me here: agent_bob_replit+clip-factory@agentmail.to”

B. If they don’t provide timestamps, choose a segment using the Selection Heuristic:
- Find a clear problem → insight → payoff within 20–45s
- Prefer emotional/contrarian openings
- Avoid long introductions, housekeeping, sponsor reads

2) Segment selection + rough transcript (30–90 minutes)
A. Pull candidate moments (choose 2, then pick best):
- Look for: “Here’s the mistake…”, “Most people think… but…”, “Do THIS instead…”, “If you’re struggling with X…”
B. Draft hook options (pick 1):
- Hook type 1: Contrarian: “Stop doing X. Do Y instead.”
- Hook type 2: Pain: “If you’re [persona] and [pain], this is why.”
- Hook type 3: Outcome: “How to get [result] in [time] without [common obstacle].”
C. Create transcript (auto) and do a fast manual cleanup of proper nouns.

3) Edit + formatting (90–240 minutes)
A. Project settings:
- Resolution: 1080x1920 (9:16)
- FPS: match source if possible (30fps typical)
- Audio: normalize dialogue, remove harsh peaks
B. Video layout rules:
- If source is landscape: center on speaker + dynamic zooms or use split layout (speaker top, supporting text bottom)
- Keep safe areas clear: no important text at extreme bottom (UI overlays)
C. Captions rules:
- 2 lines max, large font, high contrast
- Highlight 1–3 keywords per sentence (optional)
- Break captions on meaning, not grammar perfection
D. Add minimal retention boosts:
- Pattern interrupt every 2–4 seconds (jump cut, zoom, b-roll cue text)
- On-screen “context pill” at start (e.g., “Sales Call Tip” / “Mindset Shift”)

4) QC checklist (15 minutes)
- Hook hits in first 1.5 seconds (no slow ramp)
- Captions: no typos, readable on mobile, not covering face excessively
- Audio: clear, not clipped, consistent volume
- No awkward pauses > 0.5s unless for effect
- Export correct: 1080x1920, file plays on phone

5) Export + file naming (10 minutes)
Filename convention:
ClientName_Platform_Topic_YYYY-MM-DD_v1.mp4
Example: SmithCo_Reels_ColdEmailHook_2026-04-09_v1.mp4

- Suggested caption (1–2 lines)
- 5 hashtags (niche + general)
- Suggested title (Shorts) if relevant
- CTA line
