# Clip Factory — How to Review Clips & Request Revisions (Client One-Pager)

**Project:** 10 short-form vertical clips delivered within 48 hours (Week 1: free service)

**Where you’ll receive clips:** We deliver via a share link (Drive/Dropbox) or a review link (platform share link) plus a summary email from **agent_bob_replit+clip-factory@agentmail.to**.

## 1) What you’ll get in your delivery
For each clip, you’ll receive:
- **MP4 vertical (9:16)** ready to post
- **Burned-in captions** (styled)
- **Clip title + hook line** (for posting copy)
- **Suggested caption/post text** (optional, if requested)
- **“B-roll cue” notes** embedded in the timeline or included as a note (when applicable)

## 2) How to review quickly (recommended 15-minute flow)
1. **Watch all clips once** without stopping (get the overall feel).
2. Rewatch only the top contenders and mark:
   - **Keep** (ready to post)
   - **Minor tweak** (caption wording, timing, crop)
   - **Replace** (different moment from the episode)
3. Paste your feedback using the exact format below (fastest turnaround).

## 3) What counts as a revision
Revisions are meant for **small fixes**, not re-editing the whole set.

**Included (typical):**
- Caption text corrections (typos, wording)
- Caption timing tweaks (±0.5–1.5s shifts)
- Cropping/speaker framing adjustments
- Hook line swap (from the same moment)
- Audio leveling adjustments
- Removing a sentence for clarity (within the same selected moment)

**Not included (scope changes):**
- Replacing multiple clips with totally new segments beyond the agreed selection notes
- Changing the style preset after delivery (fonts/colors/layout)
- Rebuilding clips with new b-roll sourcing requirements
- Adding custom motion graphics packages beyond the preset

If you need scope changes, reply and we’ll confirm what’s possible and the new timeline.

## 4) How to request revisions (copy/paste format)
Send one message with all notes. Please use this format:

**REVISION REQUEST — [Client] — [Episode/Date]**
- Clip: `CLIP-03` (filename or link)
  - Timestamp: `00:00–00:12`
  - Change: “Replace caption line 2 with: ‘_____’”
  - Change: “Zoom out slightly at 00:04–00:08”
- Clip: `CLIP-07`
  - Timestamp: `00:09`
  - Change: “Remove the word ‘basically’ from captions”

**Approval note:** If you reply “Approved to post,” we treat the batch as final.

## 5) Turnaround on revisions
- **Minor revisions:** typically **24 hours** (often faster)
- **If you request clip replacements:** timeline may extend; we’ll confirm.

## 6) If you want the next batch
Reply: **“Next 10”** and send the next episode link/file + any new focus topics.
