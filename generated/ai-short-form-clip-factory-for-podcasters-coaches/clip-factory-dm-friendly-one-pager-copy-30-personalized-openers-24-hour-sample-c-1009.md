## 3) 24-hour “Sample Clip Offer” fulfillment SOP (internal)

**Goal:** Deliver value fast, prove quality, and convert to a DFY package (toward the $5k revenue gate). Target turnaround: **24 hours** from receiving link/assets.

### A) Intake (0–30 minutes)
1) Confirm episode source: YouTube link / drive upload.
2) Collect brand basics (if available): colors, fonts, logo, preferred caption style.
3) Collect objective: followers vs leads vs course sales.
4) Ask for 1–2 examples they like (links).
5) Confirm delivery email: agent_bob_replit+clip-factory@agentmail.to for comms; deliver to client’s preferred email/Drive.

**If client has nothing:** default to clean captions (white text, black shadow), minimal lower-third branding.

### B) Clip selection (30–90 minutes)
Pick **1 highlight** using these rules:
- Strong hook in first 1–2 seconds (or we can write/overlay one)
- Clear payoff within 20–45 seconds
- One idea per clip
- Avoid deep context dependencies

Write 2–3 hook options:
- “Stop doing X…”
- “Most people get X wrong…”
- “If you’re [persona], do this…”

### C) Edit spec (90–240 minutes)
1) Format: **9:16**, center crop, face tracking if needed.
2) Captions: burned-in, high-contrast, 2 lines max, highlight keywords.
3) Add on-screen hook text (first 1.5 seconds).
4) Add simple b-roll cues: either (a) quick cutaways or (b) text notes like “B-roll: [keyword]”.
5) Audio: normalize; remove long silences; keep pacing tight.

### D) Delivery package (same day)
Deliver:
- 1 MP4 clip (9:16)
- Caption text (copy/paste)
- 3 hook variants (for A/B test)
- Posting suggestion: TikTok/Reels title + 3 hashtags

### E) Revision policy
- One revision included for the sample (captions style, hook text, crop).
- Additional revisions roll into paid package.

### F) Upsell script (send with delivery)
“Delivered your sample clip + 3 hook options. If you want, I can turn each episode into **12 ready-to-post shorts** (captions + hooks + formatting) so you can post consistently without editing. Want me to quote the **5 / 12 / 20 clips** package for your next episode? Also—quick question: would you pay $99–$299/mo for a tool that lets you upload a link and exports highlight clips with captions automatically?”

### G) Track outcomes (CRM fields)
- Date received / delivered
- Niche
- Clip length
- Client feedback (1–2 sentences)
- Close status: sample→Starter/Growth/Volume
- SaaS signals: willingness-to-pay range + top requested features
