B) 24-HOUR “SAMPLE CLIP” FULFILLMENT SOP (DFY-FIRST, SaaS-VALIDATION INCLUDED)
Goal: Deliver 1–2 ready-to-post 9:16 clips within 24 hours of receiving the source link. Use the process to (1) earn trust, (2) upsell a clip package, (3) capture what SaaS features they’d pay for later.

Inputs required (via email to agent_bob_replit+clip-factory@agentmail.to):
- Source: YouTube link / Drive link / uploaded file
- Platform priority: TikTok / IG Reels / YT Shorts
- Target audience + offer (1 sentence)
- Brand kit (optional): fonts/colors/logo
- CTA (follow, comment keyword, book call, etc.)
- Any “do not include” topics

SOP Steps (timeboxed)
1) Intake + confirm (10 minutes)
   - Reply confirming receipt, delivery time (within 24h), and ask 1 clarifying question max (platform + CTA if missing).
   - Save prospect name + asset links in CRM.

2) Transcript creation (20–40 minutes)
   - Generate transcript (tool-agnostic; can be Whisper, Descript, etc.).
   - Quick skim for obvious sections: stories, frameworks, contrarian takes, lists, emotional moments.

3) Highlight selection (25 minutes)
   Pick 2 candidate segments using this scoring:
   - Hook potential in first 1–2 seconds (0–2)
   - Clear standalone context (0–2)
   - Strong payoff/lesson (0–2)
   - Length 12–45 seconds (0–2)
   - CTA fit (0–2)
   Choose the best 1–2 segments (aim: 18–35s for the sample).

4) Hook rewrite + on-screen text plan (25 minutes)
   - Write 3 hook options in the creator’s tone.
   - Decide caption style: punchy 1–2 line, or word-by-word karaoke captions.
   - Add b-roll cue notes (simple): e.g., “B-roll: ‘calendar’, ‘busy schedule’, ‘zoom call’, ‘client wins’.”

5) Edit + format (60–120 minutes)
   - Create 9:16 vertical layout (speaker centered, headroom correct).
   - Captions: high-contrast, 2–3 lines max, highlight keywords.
   - Add subtle progress bar or template frame if it matches brand.
   - Audio cleanup: normalize, remove harsh peaks.
   - Export platform presets:
     - Reels/TikTok: 1080x1920, H.264, 30fps, AAC audio.
     - Shorts: same, ensure safe margins.

6) QA checklist (10 minutes)
   - Captions accurate (names/terms checked)
   - First frame visually strong
   - No cut-off text in safe areas
   - Clip ends clean (no awkward mid-word cut)

7) Delivery email (10 minutes)
   Subject: “Your sample clips are ready (9:16 + captions) — Clip Factory”
   Body:
   - Link to downloads
   - 1–2 sentences explaining what moments were chosen and why
   - 3 hook variations included in text
   - Ask for a simple yes/no: “Want 12 more like this from the same episode?”

8) Upsell + SaaS validation (10 minutes)
