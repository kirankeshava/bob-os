## 1) 24‑Hour “Sample Clip” Fulfillment SOP (DFY-first)

**Goal:** Deliver 1–2 vertical clips in **≤24 hours** from receipt of source link/files. Use the sample as a wedge to close a DFY package. Do **not** mention building SaaS; SaaS remains gated until $5k DFY collected.

**Promise to prospect:** “If you send me one episode link, I’ll return 1–2 ready-to-post vertical clips with captions + hooks within 24 hours. If you like them, we can scale to a weekly clip pack.”

### A. Intake (T0–T1 hour)
1) Prospect replies to outreach OR books call.
2) Send **Intake Request** (copy below) from: **agent_bob_replit+clip-factory@agentmail.to**.
3) Required inputs:
   - Source: YouTube link / podcast video link / Drive file
   - Preferred platforms: TikTok, Reels, Shorts (choose)
   - Brand cues: fonts/colors (if any), emoji/no emoji, caption style (clean vs punchy)
   - CTA: follow, comment, book call, download lead magnet
   - Topic focus + audience (1 sentence)
   - Any “must-include” phrases or disclaimers
4) Create a job row in CRM with timestamps:
   - Lead name, niche, link, deadline, status (Intake / In Edit / QA / Delivered / Follow-up)

### B. Clip Selection (T1–T3 hours)
1) Skim the episode and mark **3 candidate moments** (15–45 seconds) using these rules:
   - Strong opinion, actionable steps, surprising stat, or story pivot
   - Minimal context needed (stands alone)
   - Clear emotional beat (pain → insight → solution)
2) Choose **1–2 moments** based on:
   - Hook potential (first 1.5 seconds)
   - Audio clarity (no crosstalk)
   - Visual adequacy (face visible) or b‑roll feasibility
3) Write a **hook line** for each clip:
   - Pattern: “Stop doing X—do this instead” / “Most people get X wrong” / “If you’re struggling with X, watch this”

### C. Edit Spec (T3–T5 hours)
**Format defaults (unless client specifies):**
- 9:16 vertical, 1080x1920, 30fps
- Safe margins for UI (keep text center)
- Captions: big, high-contrast, 2–5 words per line
- Highlight keywords (1 color) for emphasis
- Add progress bar (optional)

**Template choices (pick 1):**
1) **Clean Pro:** centered captions, subtle highlight, no emojis
2) **Punchy Creator:** quick cuts, keyword highlights, light SFX emphasis
3) **Authority Coach:** minimal cuts, lower-third name/title, clean captions

**B-roll cues (optional):** Add b-roll only if it improves comprehension (e.g., “Google Trends screenshot”, “calendar”, “before/after graph”). If no b-roll available, use quick zooms/cuts.

### D. Production (T5–T18 hours)
1) Generate transcript (tooling is flexible; for sample, manual + auto is fine).
2) Create caption lines; remove filler words; preserve meaning.
3) Assemble vertical layout:
   - Face/video primary
   - Captions center or lower 3rd
4) Add hook text within first 0–1.5 seconds.
5) Render two deliverables per clip:
   - **MP4** (ready-to-post)
   - **SRT** (optional add)

### E. QA Checklist (T18–T22 hours)
- Hook appears immediately
- Captions readable on phone (test at 50% zoom)
- No spelling errors / wrong words
- No caption covering face
- Audio consistent (no clipping)
- Clip length matches platform preference (15–45s default)
- Export naming convention correct

**Naming convention:**
- `ClientName_EpTitle_Clip1_YYYYMMDD.mp4`
- `ClientName_EpTitle_Clip1_YYYYMMDD.srt`

### F. Delivery (T22–T24 hours)
1) Upload to Google Drive/Dropbox folder.
2) Send **Delivery Email** (template below) with:
   - 2 bullets explaining what you optimized (hook, pacing, captions)
   - Ask for 1–10 rating + what to change
   - Soft close into DFY package
3) Log delivery + schedule follow-up in 48 hours.

### G. Conversion Follow-Up (48 hours after delivery)
- If rating ≥8: propose weekly pack (8/16/30 clips) and start date.
- If rating <8: offer one revision quickly; then re-ask.
