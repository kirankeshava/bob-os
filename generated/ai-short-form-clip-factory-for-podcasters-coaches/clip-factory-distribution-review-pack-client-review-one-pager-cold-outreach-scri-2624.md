# Clip Factory — How to Review Clips + Request Revisions (Fast)

**Where to send feedback:** Reply to the delivery email or email **agent_bob_replit+clip-factory@agentmail.to**.

**What you’ll receive** (typical delivery):
- 10 vertical clips (MP4) ready for TikTok/IG Reels/YouTube Shorts
- Burned-in captions (and optional SRT if requested)
- A “Clip Notes” file with hooks used + any b-roll/cut cues
- A preview/review link (if provided)

## 1) The fastest way to review (10–12 minutes)
1. Watch clips at **1.25x** speed.
2. Mark each clip with one status:
   - **APPROVE** (ready to post)
   - **TWEAK** (small edit)
   - **REPLACE** (swap for a different moment)
3. For any clip marked **TWEAK/REPLACE**, include:
   - **Clip filename** (example: `EP12_C03_TikTok_v1.mp4`)
   - **Timestamp** of issue (example: `00:06–00:09`)
   - **Exact change** (choose from the menu below)

## 2) Revision request format (copy/paste)
Subject: Revisions for EP12 Batch

- EP12_C03_TikTok_v1.mp4 — TWEAK — 00:06–00:09 — change caption line to: “Stop doing X if you want Y.”
- EP12_C07_IG_v1.mp4 — TWEAK — 00:00–00:02 — swap hook to “Most people get this wrong…”
- EP12_C09_YTS_v1.mp4 — REPLACE — prefer a clip about pricing objection, pull from 12:40–13:15

## 3) “Tweak menu” (pick 1–2 to keep turnaround fast)
**Captions**
- Fix a word / add punctuation
- Make captions bigger/smaller
- Move captions higher/lower (avoid UI)

**Hook/Structure**
- Alternate hook line (we can provide 1–2 options)
- Trim intro silence / tighten first 2 seconds
- Add one pattern interrupt (zoom/cut) around 2–4 seconds

**Visuals**
- Swap to different framing (face bigger/smaller)
- Add/remove b-roll cue (light)
- Adjust brightness/contrast slightly

**CTA**
- Change end-card CTA text (example: “Follow for daily X tips”)

## 4) What counts as a revision vs. a new clip
**Included revision (fast):** small text, timing, hook variant, minor trimming, caption style adjustments.

**New clip request:** choosing a completely different moment/topic not in the original selects. (We can do it, but it’s treated as a replacement request and may affect timing depending on scope.)

## 5) Turnaround for revisions
- **1 revision round included** for up to **3 clips** (or equivalent effort) if requested within **48 hours of delivery**.
- We aim to return revisions within **24 hours**.

## 6) Posting checklist (client)
- Confirm safe margins: captions not blocked by TikTok/IG UI
- Add your caption + 3–5 hashtags
- Pin a top comment that restates the hook/CTA
