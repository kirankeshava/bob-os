## 2) WTP + Priority Scoring Rubric (copy/paste into a sheet)

Score each feature from **0–3** in each column. Total = Priority score.

Columns:
1) **Pain (0–3)**
- 0 = nice-to-have
- 1 = mild annoyance
- 2 = frequent frustration
- 3 = top 1–2 bottleneck

2) **WTP Signal (0–3)**
- 0 = won’t pay
- 1 = only if bundled/free
- 2 = would pay at $99–$199
- 3 = would pay at $199–$299 or said “must-have”

3) **Reach (0–3)** (how common across prospects)
- 0 = niche edge case
- 1 = occasional
- 2 = common
- 3 = nearly universal

4) **Build Effort (inverse) (0–3)**
- 0 = hard/long/fragile
- 1 = medium
- 2 = straightforward
- 3 = trivial

5) **Retention Value (0–3)**
- 0 = one-time novelty
- 1 = minor stickiness
- 2 = weekly reliance
- 3 = mission-critical weekly workflow

**Priority Score = Pain + WTP + Reach + Effort + Retention (max 15).**

Recommended interpretation:
- **13–15:** Must-have for paid MVP (after $5k gate)
- **10–12:** Should-have (next sprint)
- **7–9:** Could-have (later)
- **0–6:** Don’t build yet

Feature list to score (starter set):
- Upload / YouTube link ingestion
- Auto transcription (accuracy)
- Speaker diarization (who said what)
- Highlight detection (best moments)
- Editable clip boundaries (start/end)
- Hook/Headline suggestions in niche voice
- Auto captions with brand styling
- B-roll cues / cutaway suggestions
- Platform presets (TikTok/Reels/Shorts)
- Auto exports + scheduled delivery
- Team collaboration (agency mode)
- Brand kit (fonts/colors/logo)

## 3) Feature → Test → Acceptance Criteria Mapping (revenue-gated)

Use this to prevent scope creep once $5k DFY is collected.

**Feature:** Highlight detection
- Test: From a 30–60 min episode, produce 8–15 candidate highlights.
- Acceptance criteria: User can review candidates in <10 minutes and export at least 5 clips they’d actually post.

**Feature:** Captions export (burned-in + SRT)
- Test: Generate burned-in captions with word-level timing.
- Acceptance criteria: Captions are readable on mobile, minimal drift, and export both MP4 (burned) and SRT.

**Feature:** 9:16 templates
- Test: Apply 2–3 layout templates (face cam + subtitles + title bar).
- Acceptance criteria: Clips render correctly at 1080x1920, safe margins, consistent typography.
