Title: How to Review Your Clips + Request Revisions (Fast)

Where to review
1) You’ll receive either:
   - A folder link (Google Drive/Dropbox) with all exports, or
   - A review link (Frame.io / similar) with timestamped comments enabled.

What you’re receiving
- 10 vertical clips (9:16) ready to post
- Burned-in captions (styled to your chosen preset)
- Platform-ready exports (TikTok / IG Reels / YouTube Shorts)
- A simple “Posting Notes” file (titles/hooks + suggested captions/hashtags if included)

How to review (fastest method)
1) Watch each clip once without pausing.
2) Rewatch only the clips you want changes on.
3) Leave notes in ONE of these formats:
   A) Timestamp comments directly on the review link, OR
   B) Reply by email with: Clip Name + Timecode + Change Request

Revision request format (copy/paste)
Subject: Revisions — [Client Name] — [Episode/Recording Name]

Clip: [CLIP NAME]
Timecode: [00:00–00:00]
Request: [Exact change]
Reason/Goal: [Optional: clarity, compliance, tone]

Common revision types we can do quickly
- Caption corrections (typos, speaker names, phrasing)
- Hook swap (alternate first line / alternate opener if we created multiple options)
- Tightening (trim 0.5–2s, remove filler)
- CTA swap (end-card text change)
- Minor reframing (subject centered, safe margins)

What counts as a new edit (out of scope for the included revision)
- Rebuilding clip selection strategy (new 10 clips from scratch)
- Major content rewrite / new narrative arc
- Adding new brand assets we didn’t have at intake
- Changing style preset after delivery across all clips

Included revision policy (standard)
- 1 revision round included per batch of 10 clips.
- Submit all revision requests in one message within 48 hours of delivery.
- We return revisions within 24 hours (business days) after receiving consolidated notes.

Approval = done
If you reply with “Approved” (or no revision request within 48 hours), we’ll mark the batch complete and archive the project.

Need help choosing what to post first?
Reply with: “Pick my top 3” and we’ll recommend the best-performing openers based on hook strength + clarity.
