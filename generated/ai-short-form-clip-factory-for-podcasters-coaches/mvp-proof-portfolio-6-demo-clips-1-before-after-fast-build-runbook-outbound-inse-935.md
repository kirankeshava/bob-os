Why this scope: 6 clips demonstrate breadth across niches and formats without spending days rendering 12+3. The before/after shows transformation (editing value) which is what prospects buy.

A) Clip set specification (6 clips)
Pick 1 permission-friendly longform source per niche (Creative Commons on YouTube OR explicit creator-permitted reuse OR your own recorded script). From each source, cut 2 clips:
1) “Story hook” style (20–35s)
   - Structure: Pattern interrupt hook (0–2s) → context (2–6s) → 2–3 beats (6–25s) → payoff + CTA (last 5s)
   - Captions: 3–6 words per line, highlight 1 keyword per beat.
   - Edits: jump cut every 1.5–2.5s; subtle punch-in on emphasis.
2) “Listicle” style (25–45s)
   - Structure: Hook → “3 things” → rapid bullets → short CTA
   - Captions: numbered list on-screen; bold the number.
   - Edits: quick cuts; add b-roll cues over each bullet.

Niche guidance:
- Real Estate: hooks around pricing myths, inspection surprises, first-time buyer mistakes.
- Fitness: hooks around common form mistakes, simple nutrition swaps, “stop doing X”.
- B2B: hooks around pipeline, offer clarity, outreach personalization, positioning.

B) Formatting + export settings (platform-safe)
- Canvas: 1080x1920 (9:16)
- Frame safe: keep faces/text in center 80% (avoid UI overlays)
- Captions: burned-in; high contrast; 2-line max; 55–70% width
- Loudness: target around -14 LUFS integrated (or “normalize to platform” if tool supports)
- File naming convention:
  - [Niche]_[Style]_[SourceShortName]_v1.mp4
  - Example: B2B_StoryHook_SaaSFounderTalk_v1.mp4

C) Before/After comparison (1 video)
Purpose: show raw excerpt vs final edited clip.
Format (sequential is fastest):
- 0–2s: Title card “Before → After (Clip Factory Sample)”
- 2–17s: BEFORE segment (raw excerpt)
  - Label top-left: “BEFORE: raw excerpt”
  - No captions, no zooms.
- 17–40s: AFTER segment (final version)
  - Label top-left: “AFTER: final short”
  - Include captions, hook text, punch-ins, b-roll cues.
Export: 1080x1920, same caption styling.
