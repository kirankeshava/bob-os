2) What You’ll Receive (Delivery Pack)
For each clip, you’ll get:
- Final MP4 (vertical, ready to post)
- Captions file (.SRT)
- B-roll cue sheet (timestamped suggestions)
- Posting suggestions: first-line caption options + hashtag set + recommended cover frame notes

3) QA Standards We Check (Before You See It)
Visual:
- No cropped faces; safe zones respected (UI areas top/bottom)
- Stable framing; no accidental zoom/crop artifacts
- Readable captions (size/contrast) and consistent styling
Captions:
- Names/brands spelled correctly; punctuation for clarity
- No “hallucinated” words; captions match the audio
Audio:
- Consistent loudness; no clipping/distortion
- Voice prioritized over music; noise reduced as needed
Editing:
- Hook clarity in first 1–2 seconds
- Clean jump cuts; no awkward mid-word cuts
- Pacing suited for short-form retention
Export:
- 1080x1920 (9:16), MP4 H.264, 30fps (unless source requires 24/60)
- High bitrate for text clarity; platform-safe formatting

4) Review Flow (Timestamp Comments)
- You’ll receive a review link (Frame.io / Vimeo / unlisted YouTube) with each batch.
- Please leave comments using timestamps (e.g., “00:07 change caption to …”).
- If multiple clips: label feedback by clip name (e.g., CF_03 + timestamp).

5) Revision Window & Turnaround
- Revision window: 24 hours after we send the review link.
- Typical revision turnaround: within 24 hours after feedback is received (unless major scope change).

6) What Counts as a Free Revision vs Paid Change
Free revisions (included):
- Caption typo fixes; punctuation/line breaks
- Minor timing adjustments to captions
- Small reframes to prevent face cropping
- Audio level tweaks (voice/music balance)
- Swapping a b-roll suggestion with another provided asset
Paid changes (scope changes):
- New clip selections outside the agreed moments
- Major re-edit / restructuring beyond notes
- New branding package after approval (fonts/colors/overlays)
- Additional versions for multiple platforms beyond original agreement
- New content requests after 24h window closes

7) How to Avoid Rework (Client Checklist)
Before we start, please confirm:
- Platforms (TikTok/Reels/Shorts) + @handles
- CTA + link destination
- Any banned words/topics
- Preferred tone (serious/educational/high-energy)
- Brand colors/fonts/logo (if applicable)
