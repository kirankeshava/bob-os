A) DFY 24-Hour Fulfillment SOP (end-to-end)
Goal: deliver 3–5 ready-to-post vertical clips within 24 hours of receiving intake + source link, with consistent spec and minimal back-and-forth.

1) Order kickoff (T+0–30 min)
- Confirm payment received (or written confirmation if invoicing).
- Send “Intake + next steps” email.
- Create job in tracker with: Client, link(s), deadline (24h), package, platform (TikTok/Reels/Shorts), brand notes.

2) Intake checklist (must-have to start)
Collect via email or copy/paste form:
- Source: YouTube link / podcast link / raw file link (Drive/Dropbox)
- Primary goal: leads, awareness, course sales, etc.
- Audience + niche keywords
- Tone: authoritative / friendly / spicy / calm
- CTA text (and destination URL)
- Brand kit: fonts/colors/logo (optional; if missing, default clean captions)
- “No-go” topics + compliance notes
- 3 example creators they like (optional)
If intake is incomplete after 2 hours: proceed with defaults and note “v1 assumptions” to avoid slipping the 24h SLA.

3) Clip candidate selection (T+30–120 min)
- Watch/listen at 1.5–2x speed; mark 8–12 candidate moments.
- Selection rules (choose 3–5):
  a) Self-contained: makes sense without full context.
  b) Clear hook within first 1.0 seconds.
  c) Payoff: a tactic, contrarian take, list, story turn, or strong emotion.
  d) Length: 20–45s ideal; up to 60s if story warrants.
- Create a quick “clip map” doc: timestamps + 1-line hook idea per clip.

4) Editing spec (standard defaults)
- Format: 1080x1920 (9:16), 30fps, H.264, AAC.
- Safe margins: keep subtitles within 10% padding from edges.
- Framing: speaker centered; crop for eyes/face when possible.
- Captions:
  - Burned-in, high contrast, 2–3 lines max.
  - Highlight keywords (1–3 words) per sentence.
  - Punctuation + line breaks for readability.
- Hook treatment:
  - Add on-screen hook text for first 1–2 seconds if it increases clarity.
  - Optional pattern interrupt: quick zoom, cut, or b-roll insert.
- B-roll cues (lightweight): add 1–3 b-roll inserts per clip only when it increases comprehension (don’t over-edit).
- Audio: normalize to -14 LUFS integrated; remove long silences.

5) Clip assembly workflow (T+2–10 hours)
For each clip:
- Cut to strongest start (remove throat clears).
- Add captions and keyword emphasis.
- Add minimal motion (subtle zooms/cuts) for retention.
- Add CTA end card (last 1–2 seconds) if client wants it.
- Export file naming: Client_Clip01_Topic_YYYYMMDD.mp4

6) QA checklist (T+10–12 hours)
- Captions: no obvious typos; names spelled correctly.
- No cutoff words at beginning/end.
- Audio balanced; no peaking.
- Visual: face not cropped awkwardly; captions not blocked by UI areas.
- Length appropriate; hook appears immediately.

7) Delivery (T+12–24 hours)
- Deliver via Google Drive/Dropbox folder + a simple “Posting notes” message:
  - 1 suggested caption per clip
  - 5 hashtags (niche + broad)
  - Best platform suggestion (Reels vs TikTok vs Shorts)
- Email subject: “Your clips are ready (v1) — [Client Name]”

8) Revisions policy (protect scope)
- Include 1 revision round per clip (caption tweaks + trim adjustments).
- Excludes: new clip selection from scratch, major reframing, rebranding unless brand kit provided.
- Turnaround for revisions: 24–48 hours.

9) Upsell/retainer transition (immediately after delivery)
- Ask 2 questions:
  1) Which clip performed best and why?
  2) Want this weekly so you never miss posting?
- Offer next step: weekly package or monthly retainer (ties to later SaaS once $5k DFY is reached).
