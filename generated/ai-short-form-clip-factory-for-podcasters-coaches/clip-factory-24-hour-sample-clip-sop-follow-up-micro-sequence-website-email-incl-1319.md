SOP: 24-HOUR SAMPLE CLIP (DFY) — START TO DELIVERY
Goal: deliver 1 high-quality vertical clip that proves the service, reduces perceived risk, and tees up a package sale.

1) Intake (0–15 min)
Ask for:
- Source link (YouTube/podcast/Vimeo/Drive)
- Target platform (TikTok/Reels/Shorts)
- Niche + audience (1 sentence)
- Preferred tone (direct, contrarian, empathetic, hype)
- CTA (follow, book call, download lead magnet)
- Any “do-not” words and any must-use terms
All replies/assets go to: agent_bob_replit+clip-factory@agentmail.to

2) Clip selection (15–45 min)
Pick ONE segment that meets all:
- Standalone value in <45 seconds (or 45–60 if exceptionally strong)
- Has a clear hook within first 1–2 seconds
- Contains one concrete payoff (framework, mistake, step-by-step, punchline)
- Minimal context dependency
Write down:
- Proposed hook options (3)
- Suggested clip title (1)
- Exact start/end timestamps

3) Edit spec (45–180 min)
Output format:
- 9:16 vertical
- Burned-in captions (high contrast, 2 lines max, large font)
- Safe margins for UI (keep critical text centered)
Caption style rules:
- Remove filler words; keep meaning
- Add emphasis (CAPS) for 1–2 key words per sentence max
- Split lines on natural pauses
Hook overlays (choose 1):
- “If you’re doing X, stop.”
- “Most people get X wrong.”
- “Here’s the fastest way to X.”
- “Do THIS before you try X.”
B-roll cues (lightweight): add 2–4 simple callouts like [B-ROLL: screenshot of…] or [ON-SCREEN: 3 bullet list].

4) QC checklist (last 15 min)
- Hook is readable in 1 second
- Captions accurate, no weird line breaks
- Audio is clear, normalized, no clipping
- End frame has CTA (follow / comment keyword / book)
- File plays on mobile, no black bars

5) Delivery email (send at <=24h)
Subject: Your sample clip is ready (vertical + captions)
Body:
- 1 sentence: “Here’s the first cut based on your [episode/topic].”
- Link to file (or attachment) + note on duration + platform
- 3 bullet notes: hook used, CTA, what to test next
- Ask: “Want 8–12 clips/month like this? Reply ‘YES’ and I’ll send 2 package options.”
Include credibility block + website URL again.

UPSELL PATH (immediately after delivery)
- If positive reply: offer monthly DFY package (8/12/20 clips) with turnaround and a simple onboarding checklist.
- If revisions requested: allow 1 fast revision; then ask for package decision.
