## MVP Clip Set (what to produce)
**Per niche (2 clips each):**
1) **Storytelling Hook** (30–45s)
- Objective: show hook crafting + pacing + captions.
- Edit beats: hook in first 1.5s, cut every 1.5–2.5s, subtle zooms, keyword emphasis.
- Captions: 2 lines max, high-contrast, highlighted keywords.

2) **Listicle OR Contrarian Take** (25–40s)
- Objective: show structure + retention pattern.
- Edit beats: numbered beats (1/2/3) or contrarian opener (“Stop doing X…”).
- Add quick b-roll cues (onscreen text like: “B-ROLL: neighborhood aerial”, “B-ROLL: workout demo”, “B-ROLL: dashboard screenshot”).

**Recommended MVP mix**
- Real Estate: Story hook + Listicle
- Fitness: Story hook + Contrarian
- B2B: Story hook + Listicle

**Before/After (1 export, 20–30s)**
- Format: sequential (preferred for clarity)
  - 0–3s: Title card “BEFORE: Raw excerpt”
  - 3–13s: raw excerpt (no captions)
  - 13–16s: Title card “AFTER: Final clip (captions + cuts + b-roll cues)”
  - 16–30s: finished version segment
- On-screen footer throughout: “SAMPLE EDIT • Clip Factory”

## Edit style guide (consistent across niches)
- Canvas: **1080x1920 (9:16)**
- Safe margins: 10% top/bottom; avoid UI overlap.
- Captions: burned-in, 2 lines max, bold sans (Inter/Anton/Montserrat), high-contrast box.
- Pace: remove dead air; jump cuts; zoom 105–115% on emphasis.
- Audio: normalize to ~ -14 LUFS (or platform-safe default), light noise reduction.
- Visual: add simple progress bar or subtle motion to reduce static talking head.
- Branding: small corner bug “Clip Factory • SAMPLE”
