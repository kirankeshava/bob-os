B) 10-QUESTION MICRO-SURVEY (COPY/PASTE)
Send after a reply or after a call. This quantifies willingness-to-pay and ranks features.

1) What best describes you? (podcaster / coach / agency / brand)
2) Where do you want to post consistently? (TikTok / IG Reels / YT Shorts / all)
3) Current cadence: how many short clips/week do you publish?
4) What’s the #1 bottleneck? (finding highlights / editing / captions / hooks / posting consistency)
5) How do you edit today? (in-house editor / freelancer / DIY / agency)
6) If a tool could take a link/upload and output 5–15 ready-to-post 9:16 clips with captions, what would that be worth monthly?
   - $0–$49 / $99 / $149 / $199 / $299 / $499+
7) Must-have features (pick top 3):
   - Auto highlight detection
   - Accurate captions
   - Speaker labels (diarization)
   - Brand templates (fonts/colors)
   - Hook suggestions (brand voice)
   - B-roll cue suggestions
   - Platform-safe cropping (face tracking)
   - Export presets per platform
8) Nice-to-have features (pick top 3 from same list)
9) What would stop you from paying? (accuracy / time to learn / exports look generic / price / compliance / other)
10) If we did a DFY package first (we cut clips for you), would you want to transition to DIY tooling later? (yes/no/maybe)

C) REPLY-TO-ROADMAP TRACKING RUBRIC (NO BUILD REQUIRED)
Log each lead/call outcome in a sheet using these tags and quick scores.

Lead status tags:
- DFY-NOW (wants clips done-for-you in next 7 days)
- SaaS-LATER (interested in tool; not ready to buy DFY)
- NOT-FIT (no short-form intent, no budget, or no content volume)

Minimum fields to capture:
- Contact + niche
- Current weekly long-form volume (episodes/videos per week)
- Current short-form cadence
- Editing method (DIY/editor/freelancer)
- Budget range (one-time DFY and monthly SaaS)
- Feature top 3 + must-have notes in their language
- Biggest objection
- Next step date

Fast scoring (1–5 each):
- Urgency (needs clips now)
- Volume (enough content to justify subscription)
- Budget (can pay DFY and/or $99–$299/mo)
- Pain (editing bottleneck severity)

Interpretation:
- If Urgency + Pain are high → push DFY close.
- If Volume + Budget are high but Urgency low → push waitlist + micro-survey.
- If Budget low or volume low → deprioritize.

All assets above intentionally reinforce the revenue gate: we validate and collect waitlist signals, but do not start building the SaaS MVP until $5k DFY sales are collected.
