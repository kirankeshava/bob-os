2) Listicle Clip (per niche)
- Structure: Hook (0–2s) → 3 bullets (2–40s) → Quick recap (40–50s) → CTA
- Captions: numbered bullets on-screen; keep each bullet ≤ 8 words
- Edits: hard cuts between bullets; add a light SFX ‘tick’ (optional) and micro-zooms
- B-roll cues: 1 overlay per bullet
