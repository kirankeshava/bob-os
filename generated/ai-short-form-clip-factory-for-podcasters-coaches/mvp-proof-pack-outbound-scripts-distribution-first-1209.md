## Goal (Distribution-First)
Stop waiting on a perfect 12+3 portfolio. Ship a **minimum viable Proof Pack** that is good enough to close calls: **6 demo clips (2 per niche) + 2 before/after comparisons**, hosted on Google Drive and embedded on the portfolio page. Start outbound as soon as the first 2 clips are live.

## MVP Proof Pack Scope
**Niches:** Real Estate, Fitness, B2B (podcasts/coaches)

**Per niche (2 clips each):**
1) **Story/Hook clip** (high retention opener + pattern interrupt)
2) **Listicle or Contrarian clip** (clear value density + strong captions)

**Before/After (2 total):**
- **BA-1:** Raw excerpt (no captions, horizontal/uncut) → Final (9:16, captions, zooms, b-roll cues)
- **BA-2:** Raw excerpt → Final with different caption style (karaoke highlight + punch-in cadence)

**Why this is enough:** Prospects only need to see (a) we can find moments, (b) captions look pro, (c) pacing is modern, (d) formatting is platform-ready.

## 48-Hour Proof Sprint (Free Tooling)
**Hour 0–2: Source + timestamps**
- Choose **1 license-safe longform video per niche** (Creative Commons / creator-permitted reuse).
- Pull **2 segments per niche** (25–45 seconds each) with strong audio.

**Hour 2–10: Edit + caption**
- Format: **1080x1920**, safe margins, large captions.
- Caption rules: 3–7 words per line, emphasize keywords, remove filler.
- Add simple b-roll cues as text overlays like: “[B-ROLL: house exterior]” or quick cutaways.

**Hour 10–12: Before/after exports**
- Export the “raw” excerpt (light trim only) and the “final” version.
- Assemble BA videos as either:
  - Split-screen labeled RAW vs FINAL, or
  - Sequential: 3s RAW label → 3–5s raw → 3s FINAL label → final clip.

**Hour 12–14: Hosting**
- Google Drive folder structure:
  - /Clip Factory Proof Pack/
    - /01 Real Estate/
    - /02 Fitness/
    - /03 B2B/
    - /04 Before-After/
    - /README - links & notes (Google Doc)
- Set folder to “Anyone with link can view.”

**Hour 14–16: Portfolio embeds**
- Embed Drive previews or (preferred) YouTube Shorts embeds.
- Add a “Request a free sample clip” CTA that routes to website + email.

**Hour 16–24: Public proof links**
- Upload all samples to YouTube Shorts and TikTok as “SAMPLE/DEMO” to generate clickable public links for DMs.
