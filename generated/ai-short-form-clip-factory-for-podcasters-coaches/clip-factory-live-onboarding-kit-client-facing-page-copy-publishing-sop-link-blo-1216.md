## 1) What you’re getting
For each long-form episode/video you provide, we generate **ready-to-post vertical short clips** optimized for TikTok/Instagram Reels/YouTube Shorts.

Each clip delivery includes:
- **Final video:** MP4 (vertical 9:16) with burned-in captions
- **Caption file:** SRT (so you can edit/import if desired)
- **B-roll cue sheet:** timestamped suggestions for overlays/cutaways
- **Posting suggestions:** first-line caption ideas + hashtag set + CTA suggestion

## 2) Our QA standard (what we check before you ever see it)
We run every clip through a consistent checklist so quality is predictable:

### A) Safe zones + framing
- Speaker’s face is not cropped (forehead/chin/edges safe)
- Text/captions stay inside platform safe areas (no UI overlap)
- No blocked/covered logos, titles, or key on-screen elements
- Any intentional punch-ins are smooth and not distracting

### B) Captions accuracy
- Captions match spoken words (names/brands corrected)
- Readable styling: contrast, outline/shadow, consistent placement
- Timing is tight (no long delays; no early words)
- No accidental profanity substitutions or meaning changes

### C) Audio leveling
- Voice is clear and consistently audible
- No clipping/distortion; no harsh sibilance
- Loudness feels consistent within the clip (no random dips/spikes)
- Music (if used) sits under the voice and never masks it

### D) Edit clarity (hook + pacing)
- First 1–2 seconds has a clear hook (visual + spoken/caption)
- Jump cuts feel intentional (no awkward mid-word cuts)
- Clip ends cleanly (no cut-off sentence; has a natural endpoint)
- CTA (if requested) is visible and non-spammy

### E) Export + platform formatting
- Vertical 1080x1920
- H.264 MP4, high quality, stable frame rate
- No black bars; no stretched aspect ratio
- File names are consistent and easy to identify

## 3) Review flow (fast + timestamped)
We provide a single review link per batch using **Frame.io / Vimeo / Unlisted YouTube** (depending on what’s available) so you can leave **timestamped comments**.

**How to request revisions:**
1) Open the review link
2) Leave timestamped comments (e.g., “00:07 change caption to…”)
3) If multiple changes, group them per clip to avoid confusion

**Revision window:**
- You have **24 hours** from delivery time to request included revisions.
- After 24 hours, we can still help, but changes may be scheduled and/or billed depending on scope.

## 4) Delivery pack (what you receive)
You’ll receive a folder link (typically Google Drive) with this structure:

```
/CLIENTNAME_PROJECT_YYYY-MM-DD/
  /01_FINAL_MP4/
  /02_SRT_CAPTIONS/
  /03_BROLL_CUE_SHEETS/
```

### File naming
Each clip is numbered so review notes are unambiguous:
- `CLIP_01_HookTopic_IGREELS.mp4`
- `CLIP_01_HookTopic.srt`
- `CLIP_01_BROLL_CUES.pdf` (or .docx)

## 5) What counts as a free revision vs. paid change (triage rules)
We want you happy, but we also protect scope so timelines stay reliable.

### Included (free) revisions — within 24 hours
These are fixes to meet the agreed spec:
- Caption typos or mis-heard words
- Timing tweaks to captions
- Minor framing/safe-zone adjustments
- Audio level balancing
- Small hook clarity edits (e.g., swapping first caption line) if it doesn’t require a full restructure
- Minor on-screen text corrections (names, titles, URLs)

### Paid changes (or new scope)
These are new creative directions or extra work beyond original intent:
- Changing the clip selection (“make a different moment into Clip 01”)
- Major rewrite of captions/hook concept across the clip
- Adding new graphics packages, heavy motion design, or brand kits not provided upfront
- New aspect ratios/extra platform versions not included in the original order
- Multiple rounds of revisions beyond what was agreed

**If something is urgent:** email agent_bob_replit+clip-factory@agentmail.to with “URGENT” + the clip number.
