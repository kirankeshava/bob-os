## Goal
Deliver a “Sample Clip in 24 Hours” (or first batch) with consistent quality and minimal back-and-forth. This SOP is optimized for speed (DFY revenue now) and collects signals for the later $99–$299/mo SaaS—without building anything.

## Standard Deliverable (Sample Clip Offer)

- Length: 20–45s (unless content demands 45–60s)
- Burned-in captions + hook line at top
- Export formats: MP4 (H.264), 1080x1920, 30fps, 10–16 Mbps
- Optional (if provided): logo bug + brand colors + preferred font

## Production Workflow (Step-by-step)
### Step 1 — Download & Prep (10–15 min)
- Pull source from link/upload.
- Confirm audio clarity; if unusable, flag immediately.
- Create project folder using naming convention (below).

### Step 2 — Find 3–5 Candidate Moments (20–30 min)
Selection rules:
- Must have a clear payoff within first 3–5 seconds OR can be reframed with an on-screen hook.
- Prefer: contrarian take, specific number, mistake/correction, “here’s the framework,” strong emotion, or a clean story beat.
- Avoid: long context setup, inside jokes, heavy jargon without payoff.

### Step 3 — Choose 1 Clip + Write Hook Overlay (10–15 min)
- Pick the moment with best “standalone clarity.”
- Write 1 hook line (max 8–12 words). Examples:
  - “Most coaches price wrong because of this…”
  - “3 questions that instantly qualify a client.”
  - “Stop doing this on podcasts—it kills retention.”
- Hook should NOT repeat the first spoken sentence word-for-word; it should amplify it.

### Step 4 — Captions (15–25 min)
Caption requirements:
- Accuracy > style. Fix obvious transcription errors.
- Keep lines short; break on natural pauses.
- Emphasize 1–3 keywords per sentence (color or weight) if brand style supports.
- Profanity: follow client preference (default: no censor unless asked).

### Step 5 — Edit & Format (25–45 min)
- Vertical 9:16 composition: speaker centered, eye-line around upper third.
- If source is horizontal: use smart crop + gentle motion (pan/zoom) to keep energy.
- Add subtle background music ONLY if client requested; otherwise keep clean.
- Add b-roll cues only if they’re available quickly; DFY speed > overproduction.

### Step 6 — QC Checklist (10 min)
Playback full clip with sound. Confirm:
- [ ] Hook overlay readable on mobile (safe margins)
- [ ] Captions synced; no glaring typos
- [ ] No jump cuts that break meaning
- [ ] First 1 second has motion (no dead air)
- [ ] Audio levels consistent; no clipping
- [ ] File exports correctly (1080x1920)
- [ ] Title/filename correct

### Step 7 — Deliver (5–10 min)
- Upload MP4 to Drive/Dropbox.
- Send delivery email (template below).
- Ask 2 questions to capture SaaS signals: desired volume/week and must-have features.
