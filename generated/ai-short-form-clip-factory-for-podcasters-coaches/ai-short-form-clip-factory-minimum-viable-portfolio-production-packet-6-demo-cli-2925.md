Niche A — Real Estate (Longform Source A)
- Source criteria: educational agent/broker talk, market tips, buyer psychology.
- Clip A1 (Storytelling hook):
  On-screen hook text: “The #1 mistake first-time buyers make…”
  Structure (30–45s):
  0–3s: state the mistake + why it’s costly.
  3–25s: explain with one example.
  25–40s: fix/next step.
  B-roll cues: quick overlay of “offer letter / inspection / interest rate chart”.
  CTA: “Want me to turn your podcast into 10 clips/week? Link in portfolio page.”
- Clip A2 (Contrarian take):
  On-screen hook text: “Stop chasing ‘perfect’ staging—do this instead.”
  Beats (35–55s): problem -> contrarian claim -> 2 supporting points -> one actionable step.
  Visual: 2 punch-ins, 1 b-roll insert of “before/after room photo” (stock).

Niche B — Fitness (Longform Source B)
- Source criteria: trainer/coach explaining technique, habit-building, nutrition myths.
- Clip B1 (Listicle/steps):
  Hook text: “3 fixes if you can’t lose fat (without more cardio)”
  Beats (40–55s): step 1/2/3 with on-screen numbered captions; highlight keywords.
  B-roll cues: “meal prep / steps counter / dumbbells”.
- Clip B2 (Testimonial-style edit from educational content):
  Hook text: “A client stalled for 8 weeks—here’s what changed.”
  Beats (35–50s): quick context -> what was wrong -> change made -> outcome.
  Visual: add subtle ‘results’ overlay card (no specific claims; keep generic: “energy up / consistency up”).

Niche C — B2B / Consulting (Longform Source C)
- Source criteria: founder/coach talking positioning, lead gen, pricing.
- Clip C1 (Storytelling hook):
  Hook text: “I thought I needed more leads… I needed THIS.”
  Beats (35–50s): false assumption -> turning point -> new framework (1–2 bullets).
  B-roll cues: “calendar full / CRM pipeline / proposal doc”.
- Clip C2 (Listicle):
  Hook text: “2 changes that doubled replies to my cold DMs”
  Beats (30–45s): change #1 (example line) -> change #2 (example line) -> quick recap.
  Visual: overlay the example DM line as a caption card for 2 seconds.

BEFORE/AFTER (3 examples) — exact formats
Goal: show transformation (raw -> polished) in a way prospects instantly ‘get’.

- On-screen labels:
  - RAW (no captions, no cuts)
  - FINAL (captions + cuts + b-roll cues)
- Use the same excerpt. Keep raw audio identical.

Before/After #2 (Fitness):

- Final: add numbered steps captions + punch-ins on key words (e.g., “protein”, “steps”, “sleep”).

Before/After #3 (B2B):
- Raw: founder explains a lesson.
- Final: add ‘framework card’ overlay (2 bullets) + zoom cuts every 2–3 seconds.
