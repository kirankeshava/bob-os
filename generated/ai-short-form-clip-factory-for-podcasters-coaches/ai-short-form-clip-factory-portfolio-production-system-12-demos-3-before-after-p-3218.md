## 3) Clip editing template (reusable EDL)
**Timeline setup (CapCut desktop / free):**
- Canvas: **9:16**
- Resolution: **1080x1920**
- FPS: match source (30fps ok)
- Loudness: normalize voice to ~-14 LUFS (or CapCut “Loudness normalization”)

**Structure (30–55s):**
1) **0.0–1.2s Pattern interrupt**
   - Hard cut into the most emotionally charged phrase.
   - Add 2–4 word headline text (top third).
2) **1.2–5.0s Hook**
   - One clear promise: “If you’re [audience], do this…”
3) **5.0–40s Value**
   - Cut all pauses >250ms.
   - Add jump cuts every 1.5–3.0s.
   - Add subtle punch-in/punch-out (105–115%) alternating.
4) **40–55s CTA**
   - Soft CTA: “Want 10 clips/week like this? See samples:” + website URL.

**Captions style:**
- Burned-in captions, 2-line max, 5–8 words/line.
- Highlight 1–2 keywords in a contrasting color.
- Keep captions inside safe margins (avoid bottom 15% where UI covers).

**B-roll cues (as text overlays):**
- Add short bracketed cues (not too many):
  - Example: “[B-ROLL: listing photos / gym B-roll / SaaS dashboard]”
- Place b-roll cues near the bottom-left (still within safe margins).
