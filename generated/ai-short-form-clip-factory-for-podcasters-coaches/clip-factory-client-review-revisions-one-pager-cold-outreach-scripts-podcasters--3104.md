## 1) Client One-Pager: How to Review Clips + Request Revisions (48-Hour Workflow)

**Thanks for working with Clip Factory.** This page shows exactly how to review your clips quickly, what “counts” as a revision, and how to request changes without slowing delivery.

**Where to review:**
- You’ll receive a download link (or a review link if we used a review tool) in the delivery email.
- Each clip is named clearly (example): `CLIENT_Ep12_C03_IGReels_v1.mp4`.

### What to check (fast review: 10–15 minutes total)
For each clip, confirm these 6 items:
1) **Hook lands fast:** first line hits within ~1.5 seconds.
2) **Captions readable:** no words covered by UI (TikTok/IG buttons), correct spelling for names/brands.
3) **Pacing feels right:** no long dead air; pattern interrupts roughly every 2–3 seconds (zoom/cut/b-roll/text emphasis).
4) **Meaning stays accurate:** no caption mistakes that change the message.
5) **Brand fit:** style matches the preset (font/color/position) you chose.
6) **CTA/end card:** clear and non-spammy (follow/DM/book/etc.).

### How to request revisions (copy/paste format)
Please reply to the delivery email (or email us) using this exact template so we can fix fast:

**Subject:** Revisions — [Client] — [Episode/Asset Name]

For each clip needing changes:
- **Clip name:** (e.g., `ACME_Ep12_C03_TT_v1.mp4`)
- **Timestamp(s):** (e.g., 00:04–00:07)
- **Change requested:** (e.g., “Replace ‘pipeline’ with ‘sales pipeline’ in captions”)
- **Priority:** Must-fix / Nice-to-have

### What counts as a revision (included)
Included in the free trial / standard delivery:
- Caption typo fixes, punctuation, on-screen text corrections
- Timing tweaks (caption timing, jump-cut tightening)
- Minor style adjustments within the chosen preset (caption position, emphasis styling)
- Swap a b-roll cue for a similar cue (same idea, different visual)

### What is NOT included (would be a new clip or new batch)
To protect speed and the 48-hour SLA, these are treated as new requests:
- Changing the selected moment/topic entirely (new highlight)
- Major re-editing (rebuilding the narrative, new structure)
- New brand kit development after the batch is produced

### Revision window + turnaround
- **Revision request window:** within **72 hours** of delivery.
- **Turnaround:** most revisions returned within **24 hours**.
- **Rounds:** up to **1 round** included per clip in the free trial (unless we made a factual/caption error, which we fix regardless).
