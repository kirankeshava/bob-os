## 1) Client One-Pager: How to Review & Request Revisions (Copy/Paste)

**Project:** AI Short-Form Clip Factory (10 vertical clips)

**Where you’ll review:** You’ll receive a delivery email from us with: (1) a folder link containing final MP4s, (2) a simple review sheet (or timestamp notes), and (3) an optional preview playlist.

### What “done” means (so we review the same way)
Each clip is considered complete when it has:
- A clear **hook in the first ~1.5 seconds** (text + spoken moment aligned)
- **Readable captions** (high contrast, mobile-safe, not covering faces)
- **Pattern interrupts** every ~2–3 seconds (zoom/cut/b-roll cue/graphic) to maintain retention
- Clean audio levels (no clipping; voice intelligible)
- Export that matches the requested platform spec (TikTok / IG Reels / YT Shorts)

### How to review fast (recommended)
1) Watch clips at **1.25x speed**.
2) Mark each as: **APPROVE** / **REVISION**.
3) If revision: leave **one sentence describing the goal**, plus timestamps.

### Revision request format (copy/paste)
Use this exact format to avoid delays:
- **Clip File Name:** (example: EP12_CL03_Hook-Rate-Cuts_v1.mp4)
- **Issue Type (pick one):** Hook / Captions / Pacing / B-roll / CTA / Brand Style / Other
- **Timestamp(s):** 00:03–00:07
- **Requested change:** “Replace the on-screen hook text with: ‘3 mistakes costing you deals’ and tighten the pause.”
- **Optional:** link to a reference clip you like

### What counts as a revision (included)
Included revisions are **small, specific changes** that don’t require re-concepting:
- Hook text rewrite (1–2 options)
- Caption readability fixes (typos, timing, line breaks)
- Minor pacing trims (remove dead air)
- Swap a b-roll cue or remove a distracting visual
- CTA end-card text tweak

### What does NOT count as a revision (new scope)
These are effectively a new clip direction:
- “Pick totally different moments” or “make 10 different clips”
- Major restructure requiring new selects across the episode
- New brand kit introduced after editing is done

### Revision window + turnaround
- **Revision window:** Please send revision notes within **48 hours of delivery**.
- **Turnaround:** We return revisions within **24 hours** of receiving complete notes.
- If we don’t receive notes within the window, the delivery is assumed approved so we can keep turnaround fast for everyone.

### Fastest way to get the exact style you want
Send 2–3 links to clips you love and say what you like about each (captions, pace, b-roll, hook style). If you don’t have references, we’ll use our default style preset for your niche.
