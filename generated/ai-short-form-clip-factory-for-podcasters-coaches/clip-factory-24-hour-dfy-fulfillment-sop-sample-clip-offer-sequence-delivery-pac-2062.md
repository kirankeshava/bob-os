===========================
A) 24-HOUR DFY FULFILLMENT SOP (Intake → Delivery)
===========================
Goal: Deliver 3 ready-to-post vertical clips (9:16) from a single long-form source within 24 hours of receiving assets. Optimize for speed + consistency; capture learnings for eventual SaaS.

Tools (free/low-cost options only):
- Transcription: Whisper (local) or free trial where available; fallback: YouTube auto-captions download.
- Editing: CapCut Desktop (free) or DaVinci Resolve (free).
- Captions: CapCut auto-captions + manual cleanup.
- B-roll: Client-provided assets first; fallback: free stock (Pexels/Unsplash) if allowed.
- Delivery: Google Drive folder + email.

SOP Timeline (T=0 is when client sends link/files + brand info):
1) T=0–30 min — Intake + Setup
   - Create project folder: /ClientName/YYYY-MM-DD_SourceTitle/
   - Save source link/file.
   - Create subfolders:
     01_Source
     02_Transcripts
     03_Clips_Drafts
     04_Clips_Final
     05_Thumbnails (optional)
     06_Captions_SRT
     07_Notes
   - Confirm objective: platform (TikTok/Reels/Shorts), target audience, CTA, taboo topics.
   - Confirm style: captions (word-by-word vs line-by-line), pacing, emojis yes/no, on-screen framing.

2) T=30–120 min — Transcript + Highlight Mining
   - Generate transcript.
   - Identify 8–12 candidate moments using this filter:
     (a) Strong claim or contrarian insight in first 1–2 sentences
     (b) Clear takeaway in <45 seconds
     (c) Emotion: surprise, pain, urgency, curiosity
     (d) Minimal context required
   - Select best 3 moments; log in a simple table in /07_Notes/highlights.md:
     - Timestamp start/end
     - 1-line hook idea
     - CTA suggestion
     - B-roll overlay idea

3) T=120–240 min — Build Drafts (3 clips)
   For each clip:
   - Create 9:16 sequence (1080x1920).
   - Reframe speaker (center, eye-line top third; add subtle zoom cuts every 3–6s).
   - Add captions (high-contrast, safe margins for UI).
   - Add hook text on-screen for first 1.5 seconds.
   - Add simple visual pattern interrupts every 4–8 seconds (b-roll cutaway, headline overlay, zoom, quick jump cut).
   - Export draft MP4.

4) T=240–330 min — QC + Brand Pass
   - Captions: fix names, numbers, acronyms.
   - Audio: normalize to consistent loudness; remove long silences.
   - Visual: ensure no UI text is covered by captions; check for jitter.
   - Compliance: remove copyrighted b-roll unless licensed; avoid medical/financial claims wording if risky.

5) T=330–360 min — Delivery + Next-Step Upsell
   - Export finals + optional SRT.
   - Upload to Google Drive.
   - Send delivery email (template below).
   - Ask for: 1) which clip performed best 2) style tweaks 3) next episode link.

Revision Policy (keep it simple):
- 1 revision round included per clip within 72 hours.
- Revisions include captions fixes, hook text tweak, minor trim. Not included: changing topic selection after delivery, full re-edit from scratch.
