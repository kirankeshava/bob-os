B) UNIVERSAL EDIT SPEC (use for all 9 exports)
- Canvas: 1080x1920 (9:16)
- FPS: 30
- Audio: normalize to -14 LUFS (CapCut auto enhance ok)
- Captions: large, high-contrast, 2 lines max; highlight keywords in yellow; add subtle shadow.
- Safe margins: keep text above bottom 15% (avoid UI overlays on TikTok/Reels).
- Visual rhythm: cut/zoom every 1.5–2.5 seconds; punch-in on key phrases.
- B-roll cues: use stock/onscreen keywords (can be CapCut built-in stock; if not available, use simple emoji/keyword popups as “b-roll cues” text).
- CTA overlay (last 1.5s): “Want clips like this? See samples: [website URL] | Email: [email]”
