E) CRM LOGGING SOP (to hit 200 connections + 40 proposals)
========================

Required CRM fields to log per lead:
1) Lead Name
2) Segment (YouTube / Coach / Brand / Upwork)
3) Platform + URL (channel/job link)
4) Followers/Subs (if creator)
5) Contact method (email/IG DM/TikTok DM/Upwork)
6) Personalization hook (episode title + timestamp OR job requirement)
7) Offer used (e.g., “10 clips/48h + free sample”)
8) Last touch date/time
9) Status (New / Contacted / Follow-up 1 / Follow-up 2 / Replied / Meeting set / Won / Lost)
10) Next action date/time (always required)
11) Notes (budget signals, style prefs, turnaround)

Daily execution cadence (recommended):
• AM block (60–90 min): 6–8 Upwork proposals + 10 creator outreaches
• Midday (30 min): log everything + set next actions
• PM block (60 min): 20 creator outreaches + follow-ups due today

Daily KPI rollup (top of sheet):
• Connections/Contacts sent today
• Proposals sent today (Upwork)
• Replies today
• Meetings booked today
• Samples requested / samples delivered

Minimum logging rule: if it’s not in the CRM with a next-action date, it didn’t happen.
