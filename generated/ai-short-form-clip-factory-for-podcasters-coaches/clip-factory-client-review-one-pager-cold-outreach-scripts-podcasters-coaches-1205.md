# 1) Client One-Pager: How to Review Clips + Request Revisions (Copy/Paste)

**What you received**
You’ll receive **10 ready-to-post vertical clips** optimized for TikTok / Reels / YouTube Shorts.

**Fastest way to review (10–15 minutes)**
1) Watch each clip once with sound ON.
2) On the second pass, only note **must-fix** items (typos, misheard words, wrong names, missing context).
3) Approve the rest as-is. Short-form wins through volume and speed.

**What counts as a revision (included)**
Included revisions are limited to:
- Caption fixes (typos, punctuation, misheard words)
- Timing tweaks (shift captions earlier/later)
- Minor hook tweak using **existing words from the clip**
- Remove/replace b-roll cue with another **simple cue**
- Export fix (wrong aspect ratio, cropping, audio level)

**What does NOT count as a revision (not included)**
- Rewriting the clip narrative or adding new scripting
- Creating new graphics packages from scratch
- Sourcing highly specific b-roll assets (custom footage)
- Re-cutting an entirely different moment not in the delivered selects

**Revision window + turnaround**
- Please request revisions within **72 hours** of delivery.
- We turn included revisions around in **24 hours**.

**How to request revisions (copy/paste format)**
Send one email with all changes using this structure:

**Subject:** Revisions — [ClientName] — [Episode/Video Title]

**Clip ID:** CLIP-01
**Timecode:** 00:07–00:11
**Change:** Caption word “their” → “there”

**Clip ID:** CLIP-04
**Timecode:** 00:00–00:02
**Change:** Use hook alt: “Stop doing X—do this instead” (keep same moment)

**Clip ID:** CLIP-09
**Timecode:** 00:12–00:15
**Change:** Remove b-roll cue, keep clean talking head

**If you want us to cut a different moment**
Reply with:
- Episode/video link or file
- **Exact timestamp range** you want (ex: 12:40–13:15)
- Goal (ex: “lead gen for coaching,” “podcast growth,” “sell listing consults”)
We’ll confirm if it’s within scope or quote it.
