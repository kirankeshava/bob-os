- Hook angle: “3 fixes if you’re training hard but not seeing results.”

- Edit feel: quick rhythm cuts; add 1 beat-synced text pop per item
- B-roll cues: “meal prep”, “sleep tracker”, “progressive overload chart”

**4) Testimonial / Case Result (25–40s)**
- Hook angle: “I had a client stuck for 6 months—this changed everything.”
- Captions: emphasize timeframes + outcomes (*6 months / changed / results*)
