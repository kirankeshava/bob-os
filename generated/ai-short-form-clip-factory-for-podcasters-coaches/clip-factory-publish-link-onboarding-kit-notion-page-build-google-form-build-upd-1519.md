E) Review process (timestamped comments only)
- You will receive a review link to a watermarked preview.
- Please leave feedback as timestamped comments, e.g., “00:07 change caption line” or “00:12 remove jump cut.”
- Consolidate notes into one pass when possible to keep turnaround fast.

F) QA standards (what we check before sending)
- Safe zones: no cropped faces, no critical text near edges, platform UI overlays considered
- Captions: accurate words, correct punctuation, no repeated/garbled segments, speaker labels if needed
- Audio: consistent loudness, no clipping, noise reduction not overdone, music bed not masking speech
- Jump cuts: purposeful pacing, no mid-word cuts unless stylistic and intentional
- Hook clarity: first 1–2 seconds establish curiosity/benefit; no slow ramp
- Branding: correct @handles, CTA, on-screen text spelling
- Export settings: vertical 9:16, correct codec, readable bitrate; file names consistent

G) Delivery method (how you’ll get files)
- Final deliveries provided via a shared Google Drive folder, structured by clip number.
- You will also receive a single “Posting Suggestions” doc for quick copy/paste.

H) Revisions policy (simple + firm)
Free revisions (within 24 hours of delivery):
- Fix caption typos/misheard words
- Adjust safe-zone framing / minor crop fixes
- Minor audio leveling tweaks
- Replace a b-roll cue if it’s clearly mismatched
- Small timing adjustments (tighten/extend by a few frames/seconds)

Paid changes (quoted before work starts):
- New direction/creative pivot after delivery (different hook angle, different topic focus)
- Adding new clips beyond the original order count
- Major re-edit / restructuring (new selects, new story arc, new CTA across all clips)
- Requests based on new source footage provided after editing is done

Section 1 — Contact + project basics
1) Full name (Short answer) REQUIRED
2) Email for delivery/review (Short answer) REQUIRED
3) Brand / channel name (Short answer) REQUIRED
4) Source link (Long answer) REQUIRED
   - Prompt: “Paste Google Drive/Dropbox/unlisted YouTube/Vimeo link. If multiple files, paste folder link.”
5) What did you purchase? (Multiple choice) REQUIRED
   - “X clips package” / “Custom”
6) Target platforms (Checkboxes) REQUIRED
   - TikTok / Instagram Reels / YouTube Shorts / LinkedIn (vertical)

Section 2 — Style + constraints
7) Clip vibe (Multiple choice) REQUIRED

8) Caption style (Multiple choice) REQUIRED
   - Clean subtitles / Bold karaoke / Minimal / No captions
9) On-screen text allowance (Multiple choice) REQUIRED
   - Yes (light) / Yes (heavy hooks + bullets) / No
10) Brand colors/fonts (Long answer) OPTIONAL
11) Your @handle(s) + website link (Long answer) REQUIRED
12) Banned words/topics (Long answer) OPTIONAL
13) Must-include CTA (Short answer) OPTIONAL

Section 3 — Content selection
14) Best moments (Long answer) OPTIONAL
   - Prompt: “If you have timestamps, paste them. Example: 12:30–13:10.”
15) Avoid moments (Long answer) OPTIONAL
   - Prompt: “Anything we should NOT clip? Provide timestamps if possible.”
16) Competitors/creators you like (Long answer) OPTIONAL

Section 4 — Approval + confirmation
17) Confirmation checkbox (Checkboxes) REQUIRED
