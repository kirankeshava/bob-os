## 4) 24‑hour “sample clip” fulfillment SOP (zero-cost first week)

**Goal:** Deliver 1 high-quality sample clip in ≤24 hours to close DFY packages.

### A) Intake (5 minutes)
Ask for:
- YouTube link (preferred) or Drive upload
- Target platform (Reels/TikTok/Shorts)
- Preferred caption style (clean vs bold)
- 1–2 creators they like (for pacing)
- Any words to avoid + CTA
Route everything to: agent_bob_replit+clip-factory@agentmail.to

### B) Toolchain (free options)
- Download source: yt-dlp (free) or ask for Drive file.
- Transcribe: Whisper (local if available) OR use a free-tier transcription tool if already accessible.
- Rough cut: CapCut desktop (free) or DaVinci Resolve (free).
- Captions: CapCut auto-captions + manual cleanup.

### C) Clip selection criteria (10 minutes)
Pick 1 segment that has:
- A clear hook in first 1–2 seconds
- A complete idea in 20–45 seconds
- Emotional spike / contrarian take / strong tactical list

### D) Edit checklist (30–60 minutes)
- Format: 9:16
- Tighten pauses; remove fluff
- Add hook text in first frame (big, 6–10 words)
- Captions: 1–2 lines, high contrast, punctuation cleaned
- Add subtle zoom/cuts every 2–4 seconds if needed
- Add b‑roll cue notes (in a text file) if b‑roll would clarify
- Export settings: 1080x1920, H.264, 15–30 Mbps

### E) Delivery (5 minutes)
Send:
- MP4 file
- 3 alternative hook lines
- Suggested caption (post text) + hashtags (5–10 niche)
- Ask for “approve this style? (Y/N) what to change?”
