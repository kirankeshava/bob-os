Below is an operator-ready DFY fulfillment system to deliver short-form vertical clips within 24 hours of receiving client assets. Use this to close faster (confidence) and fulfill consistently.

1) 24-HOUR DFY FULFILLMENT SOP (End-to-End)

Goal: Deliver ready-to-post 9:16 clips with captions + hook overlays + posting notes in <24 hours from asset receipt.

A. Intake (0–1 hour after close)
- Send client the “Asset Request / Intake” email (template below).
- Collect:
  1) Source: YouTube link / podcast episode link / raw file (Drive).
  2) Target platforms: TikTok / IG Reels / YT Shorts.
  3) Audience + offer: who they help, what they sell, desired CTA.
  4) Brand kit: logo, colors, fonts (or “none—use clean default”).
  5) Caption style: bold keywords? emoji yes/no? profanity yes/no?
  6) Examples they like (2–3 links).
  7) Clip goals: leads, followers, webinar signups, course sales.
- Confirm turnaround + delivery format.

B. Project Setup (15 minutes)
- Create Drive folder structure (see Section 2).
- Create a tracking row in CRM (or sheet): Client | Date received | Due | Package | #clips | Status | Notes.
- Create filename prefix (e.g., ClientName_Ep12_2026-04-09).

C. Content Scan & Segment Selection (30–60 minutes)
- Watch/listen at 1.5x–2x and mark timestamps.
- Pick segments that have at least one:
  - Strong claim/contrarian point
  - Clear step-by-step framework
  - Emotional moment/story
  - Proof/result
  - Common mistake + fix
- Clip length targets:
  - TikTok/IG: 20–45s preferred; up to 60–75s if compelling.
  - Shorts: 25–55s preferred.
- For each chosen segment, write:
  - Hook option A (curiosity)
  - Hook option B (results/outcome)
  - CTA line (comment “X”, follow, link-in-bio, etc.)

D. Edit Build (Core) (2–5 hours depending on package)
- Format: 1080x1920, 30fps.
- Layout (default): speaker center, safe margins for UI, optional subtitle background.
- Jump cuts: remove long pauses, “um,” dead air.
- Add hook overlay in first 1–2 seconds.
- Add burned-in captions (readable, 2 lines max, highlight keywords).
- Add light b-roll cues only if it increases comprehension (do not over-edit).
- Audio: normalize loudness, reduce noise if needed.

E. Captions & Accuracy QC (30–60 minutes)
- Verify proper nouns, brand terms, and numbers.
- Check caption timing: no lag, no overlapping.
- Ensure the first 3 seconds are compelling (hook visible + spoken content starts fast).

F. Platform-Specific Finishing (30 minutes)
For each clip, include in the “Posting Notes” doc:
- Suggested caption (1–2 lines)
- 6–12 hashtags (niche + broad)
- Suggested on-screen title (short)
- CTA suggestion
- Best platform fit (IG/TikTok/Shorts)

G. Final QC Checklist (10 minutes)
- 9:16 resolution correct
- No cut-off captions
- No copyrighted music added
- Hook appears immediately
- Clip ends cleanly (no awkward cutoff)
- Exported MP4 plays on mobile

H. Delivery (within 24 hours)
- Upload to Drive folder / deliver link.
- Send delivery email with:
  - Drive link
  - What’s included
  - Suggested posting cadence
  - Request for 1 round of revisions (if included) within 48 hours

I. Revisions (if included)
- Limit to: captions fixes, trim length, hook text change, minor color changes.
- Turnaround: 24 hours.
