## 2) “Sample Clip Sprint” Fulfillment SOP (24-hour turnaround)
Purpose: close DFY deals faster by delivering 1–2 polished vertical clips within 24 hours of receiving the source link.

### Inputs required (collect immediately after payment)
Send the client an email from **agent_bob_replit+clip-factory@agentmail.to** with these questions:
1) Source video link (YouTube/Drive/Dropbox) + time range if they already know it
2) Platform priority (TikTok / Reels / Shorts)
3) Brand kit (logo optional), brand colors (hex if known), preferred font (or “use clean default”)
4) Caption style: (A) word-by-word karaoke (B) sentence blocks (C) minimal subtitles
5) Hook style preference: (A) punchy (B) contrarian (C) empathetic (D) authoritative
6) CTA to include (follow, book call, download lead magnet) + URL
7) Do-not-say list (words/claims to avoid)

### Step-by-step workflow (target: <2.5 hours labor for 1–2 clips)
**0. Project setup (5 min)**
- Create a folder: ClientName_Date_SourceTitle
- Save intake answers + link.

**1. Pull transcript (15–30 min)**
- Generate transcript via Whisper (API or tool) OR YouTube transcript if acceptable.
- Quickly scan for profanity/claims risks.

**2. Identify 3 candidate moments (20–30 min)**
Pick moments with:
- A clear “problem → insight → payoff” arc
- A strong first 1–2 seconds (“pattern interrupt”)
- Minimal context required
- Ideally 20–45 seconds duration

**3. Choose 1–2 final clips (5–10 min)**
- Select the best two based on clarity + punch.
- Write **2 hook variations** per clip (for A/B testing):
  - Hook A: curiosity gap
  - Hook B: direct benefit

**4. Edit + format (45–75 min)**
- Crop to **9:16**; keep subject framed mid/upper third.
- Add captions with safe margins (avoid platform UI).
- Add minimal motion (zoom/pan) if needed.
- Optional: add b-roll cue text (internal note) if b-roll isn’t inserted.

**5. QC checklist (10 min)**
- Captions synced, no misspellings of names/brands
- Loudness consistent; no clipping
- Safe areas respected (top/bottom)
- First frame is strong (not a dead frame)

**6. Delivery (5–10 min)**
- Export settings:
  - 1080x1920, 30fps, H.264, AAC audio
- Deliver via Drive link + include:
  - File(s)
  - Hook text options
  - Suggested caption + hashtags (5–8)
  - Posting recommendation (best platform + best time window)
