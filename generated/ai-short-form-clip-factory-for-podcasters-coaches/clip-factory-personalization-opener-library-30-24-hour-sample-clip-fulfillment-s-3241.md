## Part B — 24-Hour “Sample Clip” Fulfillment SOP (DFY-first; supports $5k gate)
Goal: deliver one high-quality 9:16 clip within 24 hours of receiving a link + permission. This is the fastest path to trust → paid DFY package.

### 0) Tools (free/low-cost defaults)
- Source: YouTube link / podcast video link / Drive upload.
- Transcript: Whisper (local if available) or platform transcript + manual cleanup.
- Editing: CapCut Desktop (free) or DaVinci Resolve (free).
- Captions: CapCut auto-captions + manual correction.
- Delivery: Google Drive link (view/download) + optional native upload if client provides access.

### 1) Intake (Timebox: 10 minutes)
Request via email to agent_bob_replit+clip-factory@agentmail.to:
- Source link + episode title
- Target platform (TikTok / Reels / Shorts)
- Audience (who is it for?)
- Preferred tone (direct/empathetic/high-energy)
- Brand colors/fonts (optional)
- CTA (follow, comment keyword, book call, watch full episode)
- Any “must include” talking point

Reply template should include legitimacy:
- “Website: https://6ff73b98-dcff-4f53-a10d-9ad392f63342-00-1n4h4nuiq6w8j.picard.replit.dev/sites/3”

### 2) Find highlights (Timebox: 25 minutes)
Process:
1) Skim transcript for:
   - Contrarian take
   - Specific numbers/steps
   - A strong story moment
   - A pain-first statement
2) Identify 3 candidate segments (15–45s each).
3) Select 1 best segment using these quick criteria:
   - Hook potential in first 1–2 seconds
   - Clarity without extra context
   - Single idea per clip

### 3) Write hook + on-screen structure (Timebox: 10 minutes)
Create:
- Hook text for first frame (max 8 words)
- Subtitle style: 2 lines max, high contrast
- Pattern interrupts: 1 punch-in + 1 b-roll cue
- End CTA card (2 seconds)

### 4) Edit vertical cut (Timebox: 35 minutes)
Steps:
1) Set project to 9:16 1080x1920.
2) Auto reframe speaker (center face, avoid cropping mouth).
3) Add punch-ins on emphasis words.
4) Add b-roll cues (either placeholders text like “B-ROLL: [keyword]” or simple stock-free visuals if available).
5) Add subtle background music (very low, -25 to -35 LUFS feel) only if it improves pace.

### 5) Captions (Timebox: 25 minutes)
- Generate auto captions, then correct:
  - Names, niche terms
  - Numbers and dates
- Style:
  - Highlight 1–3 keywords per sentence
  - Keep reading speed comfortable

### 6) Export + QA (Timebox: 10 minutes)
Export settings:
- MP4, H.264
- 1080x1920
- 30fps
- 10–20 Mbps
QA checklist:
- Hook visible immediately
- No caption typos in first 3 seconds
- Audio clear, no clipping
- CTA present

### 7) Deliver (Timebox: 5 minutes)
Email deliverables:
- Drive link to MP4
- 2 caption options (post copy)
- 5–10 hashtag suggestions (optional)
- “If you want, I can produce [X clips/week] from each episode with consistent style.”
Include:
