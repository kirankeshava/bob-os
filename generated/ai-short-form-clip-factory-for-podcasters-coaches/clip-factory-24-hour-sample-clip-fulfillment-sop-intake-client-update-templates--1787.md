A) Offer Definition (what the client gets)
“24-Hour Sample Clip” deliverables (default):
1) 1 vertical 9:16 clip (20–45s) from a provided YouTube/podcast/video file
2) Burned-in captions (clean, high-contrast, mobile-safe margins)
3) Title/Hook options (3 variations) in a text file
4) Posting recommendation (TikTok/Reels/Shorts) + suggested caption/hashtags (lightweight)

Optional (if time permits / as add-on later):
- 2nd clip variant (different hook) or thumbnail frame
- A/B hook text overlays

B) Tools (no spend assumptions)
- Download/source video: YouTube link, Drive link, or file upload
- Transcription: any available free workflow (or manual captioning) + spot-check
- Editing: CapCut/DaVinci/Premiere (whatever is available)
- Caption styling: bold readable font, 2-line max, highlight keywords
- Delivery: Google Drive link or unlisted YouTube/Vimeo + MP4 file

D) Step-by-step Fulfillment SOP
1) Intake & Asset Check (15 minutes)
- Confirm receipt of:
  - Source link/file (YouTube/Drive)
  - Target platform(s): TikTok, IG Reels, YT Shorts
  - Topic focus (what they want more of)
  - CTA (follow, download lead magnet, book call)
- Reply via agent_bob_replit+clip-factory@agentmail.to with next steps + timeline.

2) Select Clip Moment(s) (60–120 minutes)
Selection criteria (aim for “stops the scroll”):
- Opens with tension/contrarian point/clear promise in first 1–2 seconds
- Contains a complete thought within 20–45 seconds
- Has an emotional or practical payoff (a framework, step list, strong opinion)
- Avoids long setup, housekeeping, rambling

3) Edit for 9:16 (90–180 minutes)
- Create a 9:16 sequence
- Reframe speaker center with headroom; avoid cropping chin/forehead
- Cut all dead air; increase pace slightly (but keep natural)
- Add quick b-roll cues only if it improves clarity (otherwise keep clean)
- Keep total length within 20–45 seconds unless content demands 60s

4) Captions (90–180 minutes)
- Burn-in captions, mobile readable, high contrast
- 2 lines max; keep within safe margins
- Emphasize 1–3 keywords per sentence (color or bold)
- Spot-check accuracy on names, numbers, claims

5) Hook Overlay + CTA (30–60 minutes)
- Add a short hook overlay (3–7 words) that matches the moment
- Include a subtle CTA at end (e.g., “Follow for daily X” / “Comment ‘CLIP’ for the checklist”)

6) QC Checklist (15 minutes)
- Audio intelligible on phone volume
- No captions cut off at top/bottom
- First frame is compelling (no awkward mouth-open frame)
- No long silent gaps; no repeated phrases
- Export settings correct (1080x1920, H.264)

7) Deliver & Ask the Conversion Question (10 minutes)
- Send:
  - MP4 link
  - Hook options (3)
  - Suggested posting note
- Ask:
  - “Want us to turn the next episode into 10 clips/week?”
  - Offer package options and start date.

E) Revision Policy (keep it tight)
- 1 round of minor revisions included for the sample (caption corrections, hook text swap, trim 1–3 seconds)
- Anything involving new clip selection or major restructure counts as a new sample/clip

F) Data to Capture (feeds product later)
Log after every sample:
- Source type (podcast video, Zoom, studio)
- Biggest friction (audio quality, long episodes, multiple speakers)
- What they asked for (hooks, templates, b-roll, brand captions)
- Turnaround time and revision count
- Conversion outcome (upsell to package? yes/no; price point)
