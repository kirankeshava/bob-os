  1) Final clips (MP4 vertical)
  2) Optional “no-captions” versions (if requested)
  3) A notes file listing the hook + CTA for each clip

How to review (fastest method)
1) Watch each clip once with sound ON (this catches 90% of issues).
2) Then skim captions for: names, numbers, claims, and brand terms.
3) Approve or request changes using the format below (copy/paste).

What counts as a revision (included)
Each batch includes 1 revision round covering:
- Caption corrections (typos, wording, punctuation)
- Hook text adjustment (shorten/clarify)
- Cropping/safe-area fixes (face cut off, too low/high)
- Basic timing tweaks (start 0.2–1.0s earlier/later)
- CTA end-card text swap (same layout)

What does NOT count as a revision (new scope)
To keep 48-hour delivery, the following are treated as new requests:
- New clip topics not in the delivered set
- Re-cutting the clip from a different segment/timecode
- Changing the style preset (fonts/colors/layout) after delivery
- Extensive content changes (rewriting the spoken message)

Revision request format (copy/paste)
Subject: Revisions — [ClientName] — [EpisodeName]

Clip ID(s): [e.g., EP12_C03]
Timecode / moment: [e.g., 00:04–00:09]
Change needed: [e.g., “Fix caption: ‘CRM’ not ‘CEM’”]
Priority: [Must-fix / Nice-to-have]

If multiple clips:
- EP12_C03: …
- EP12_C07: …

Approval rule
- If we don’t hear back within 48 hours of delivery, we assume approval and we’ll archive the project as complete (you can still request new batches anytime).
