B) 12 demo clips: production matrix (what to cut + how to edit)
Export standards (all clips)
- Format: 9:16, 1080x1920
- FPS: match source; deliver 30fps if possible
- Length targets: 20–35s for TikTok/Reels; keep under 60s
- Captions: burned-in, high-contrast, 2 lines max, key words highlighted
- Safe margins: keep text within 10% inset (avoid UI overlays)
- Audio: normalize voice ~ -14 LUFS, light compression
- Visual: fast cuts every 1.0–2.0s; punch-in zoom 105–115% on emphasis
- B-roll cues: on-screen prompts like [B-ROLL: Zillow map] (for demo), or actually overlay royalty-free b-roll if available
