## 4) 24-Hour Sample Clip Fulfillment SOP (internal)

**Objective:** Deliver 1 high-quality sample clip in ≤24 hours to maximize conversion to a DFY package.

### Inputs required (must collect)
1) Source link (YouTube/Spotify video/Drive)
2) Target platforms (TikTok/Reels/Shorts)
3) Audience + offer (what’s being sold or promoted)
4) CTA (follow, comment keyword, book call, download)
5) Brand notes (optional): colors, font preferences, examples they like

### Process steps
1) **Acknowledge receipt (within 1 hour):** confirm delivery timeline; ask one clarifying question if needed (CTA or target audience).
2) **Select moment (10–20 min):** find a 20–45s segment with a clear claim, story beat, or actionable tip.
3) **Create 3 hook options (5 min):** write 3 hook lines; pick best for caption top line.
4) **Caption style rules (consistency):**
   - Large, high-contrast captions; 2 lines max on-screen
   - Highlight keywords (1 color) if brand color known; otherwise white + yellow accents
   - Remove filler words lightly; keep meaning intact
5) **Export rules:**
   - 9:16, 1080x1920, 30fps
   - Safe margins for UI (keep captions above bottom 15%)
   - Loudness normalized; remove long silences
6) **Delivery package:**
   - 1 MP4 file
   - 1 text file (or email) with: suggested caption, 5 hashtags, CTA line
   - Ask for “yes/no” feedback + one change request
7) **Conversion follow-up (same message):** propose package and next steps.
