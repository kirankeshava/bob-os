  5) Locations (# + addresses if multi-location)

- Add specialized profiles (optional): “Reputation Management” / “Google Business Profile”.
- Add portfolio item: “AI Review Reply & Reputation Autopilot” with link to website above.
- Create 3 saved searches (keywords):
  1) “google business profile reviews”
  2) “respond to reviews”
  3) “reputation management”
- Apply only to jobs that:
  - Mention Google reviews/GBP/Yelp or “reply to reviews”
  - Have clear scope (X reviews/week)

C. Fiverr setup (20 minutes)

- Add “Available now / 24hr response” in FAQ.
- Use simple thumbnails (Canva free): one headline + 3 bullets.
