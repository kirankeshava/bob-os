6) Daily sending ops checklist (14-day ramp)
Assumptions: 1 inbox to start; expand only after positive reply rates and low bounces.
- Day 1–2: 10–15 new emails/day + follow-ups; verify formatting; watch bounces.
- Day 3–4: 20–25/day.
- Day 5–7: 30–40/day.
- Week 2: 50/day if bounce rate <3% and spam complaints ~0.
Daily routine (60–90 minutes):
1) Pull 25–50 leads from Priority A/B first.
2) QA 10% sample: correct category, not a franchise directory listing, has recent reviews, has website/phone.
3) Personalize 1 line using review snippet or paraphrase.
4) Send initial emails in a 2–3 hour window (avoid blasts).
5) Handle replies within 2–4 hours. Book calls immediately.
6) Log outcomes in CRM.
Bounce/complaint thresholds:
- If hard bounces >3% in a day: stop sends, fix list quality.
- If any spam complaints: reduce volume, tighten targeting, adjust copy.
