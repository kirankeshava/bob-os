## Goal
Activate a new **paid location** (or trial in week 1 policy) within **48 hours**: access granted → brand voice set → approvals done → first responses drafted → weekly KPI report baseline created.

## Day 0 (Immediately after verbal “yes”)
1) **Send Kickoff Email (Day 0)**
   - Include: legitimacy URL + support inbox.
   - Attach/link: intake form, access-request instructions (GBP + Yelp), proposed start date.

2) **Send Intake Form** (customer completes in <10 minutes)
   - Brand voice + do/don’t list
   - Escalation contacts
   - Locations + links
   - Approval mode (auto-post vs approval-first)

3) **Send Access-Request Emails** (forwardable)
   - GBP: request Manager access to the right listing(s).
   - Yelp: request access/role needed (or “send us screenshots/export” fallback if access is blocked).

4) **Create Internal Customer Record (CRM Sheet)**
   - Stage: Closed → Paid → Onboarding
   - Fill: location count, vertical, review volume, current response time, escalation contact.

## Day 1 (Within 24 hours)
5) **Confirm Access + Ingestion**
   - Verify we can view reviews and respond (GBP/Yelp).
   - If access blocked: set fallback workflow (customer forwards review links or screenshots daily; we draft responses within SLA).

6) **Set Brand Voice + Rules**
   - Create “Voice Card” (5 bullets): tone, greeting/sign-off, personalization rules, refund/comp policy language, prohibited phrases.

7) **Build Escalation Triggers**
   - Examples: 1–2 stars, allegations (fraud/safety), staff name mentioned, legal threat, health/safety claim.
   - Confirm escalation contact + response time target.

8) **Draft Sample Pack for Approval (10–15 minutes)**
   - Draft 3 replies from recent reviews:
     - 1 positive
     - 1 neutral
     - 1 negative (with de-escalation)
   - Send for approval if in “approval-first” mode.

## Day 2 (Within 48 hours)
9) **Go Live**
   - If approved: begin posting replies per the agreed mode.
   - Start cadence: daily review sweep (or Mon/Wed/Fri for low volume).

10) **Baseline Weekly KPI Report Setup**
   - Capture baseline:
     - Current star rating
     - # new reviews last 7/30 days
     - % responded
     - Median response time (estimate if platform doesn’t expose)
     - # negatives escalated

11) **First-Week Success Criteria (Share with customer)**
   - Within 7 days:
     - 90–100% of new reviews responded (unless customer requests selective)
     - 0 un-escalated critical negatives
     - Consistent brand voice
     - Weekly KPI email delivered with “what changed + next actions”
