# D) Week-1 Execution Cadence (daily quotas)
Daily (Mon–Fri):
1) **Upwork:** 10–20 proposals/day using Template Variant 1 (and 1–2 niche-specific variants if needed).
2) **Cold email:** 20–40 new emails/day (hand-built list from Google Maps + business websites).
3) **Follow-ups:** 10–30 follow-ups/day (Day 3 + Day 7).
4) **Speed-to-lead:** Respond to any inbound within 15 minutes when possible.

Day 7 review (hard cut rule):
- If a channel produces **0 booked calls in 7 days**, stop time/spend on it for the next week and reallocate volume to the best-performing channel.

# E) Tracking Sheet Spec (Google Sheets)
Tabs:
1) Leads
2) Conversations
3) Calls
4) Deals
5) Weekly Summary

Key columns (Leads tab):
- Date Added
- Channel (Upwork / Fiverr / Cold Email / Referral)
- Campaign (e.g., CE-Dentists-W1)
- Company
- Contact Name
- Email/URL
- Niche
- Rating (if visible)
- Review Count (if visible)
- Status (Not Contacted / Contacted / Replied / Call Booked / Closed Won / Closed Lost)
- Notes

Weekly Summary formulas:
- Booked calls by channel (COUNTIF)
- Reply rate by channel (Replies / Contacted)
- CAC (Week 2+): Spend / Closed Won (or Spend / Booked Calls as leading indicator)
