## 3) Fulfillment scope (week-1 deliverable; no API dependency)
### Review sources
- **Google Business Profile (GBP)** reviews (primary)
- **Yelp** reviews

### Response creation + QA
- Draft responses using brand voice rules + approved offer language.
- Human QA for: policy compliance, brand safety, HIPAA/medical/legal sensitivity, admissions of fault, refund promises, prohibited incentives.

### SLA rules
- **4–5★:** auto-draft + post (or queue for client approval depending on access model) within SLA.
- **1–3★ / sensitive keywords:** escalation workflow + approval required before posting.

### Escalation rules (minimum)
Escalate when any of the following occur:
- 1–3★ rating
- Mentions: “lawsuit”, “fraud”, “scam”, “BBB”, “attorney”, “chargeback”, “HIPAA”, “injury”, “discrimination”, “harassment”
- Staff named + allegation
- Refund demands or threat to post elsewhere

Escalation output: recommended public response + private resolution script + who to contact + “next action due date.”

### Posting method (deliverable immediately)
- **Default (preferred):** client grants **GBP Manager access** and (if possible) Yelp Business access.
- **Fallback (always available):** **client-post workflow** — we draft responses in a shared doc/email; client posts using their own login. This avoids API and avoids us holding credentials.

### Weekly KPI report (sent every Monday)
Includes:
- New reviews (Google/Yelp)
- Average rating trend (7/30 days)
- Response rate (%)
- Median response time
- Negative review count (1–3★)
- Escalations opened vs resolved
- Top themes (3 bullets)
- Next-week recommendations (3 bullets)
