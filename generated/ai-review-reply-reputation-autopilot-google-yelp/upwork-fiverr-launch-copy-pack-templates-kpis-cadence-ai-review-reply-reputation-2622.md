C) Channel KPIs + Weekly Review Cadence (7-day cut rule)

North Star: Booked qualified calls (and then conversions) per channel.

KPIs by channel (track weekly)
Upwork
- Proposals sent
- Reply rate (% of proposals with response)
- Booked calls
- Cost per booked call (once connects/boost spend begins)
- Close rate

Fiverr
- Impressions
- Clicks
- Inquiries
- Orders (or booked calls if routing to consult)
- Cost per inquiry/booked call (once promoted gigs begins)

Cold Email
- Emails sent
- Deliverability (bounce rate)
- Reply rate
- Positive reply rate
- Booked calls

Proof generation
- # pilots started
- # pilots completed
- # testimonials/case studies obtained
- Average rating change / response time change (per pilot)

7-day cut rule
- If any paid channel produces 0 booked calls within 7 days of spend start → pause spend and reallocate.
- If booked calls happen but show-rate < 50% → tighten qualification + reminders.
- If show-rate good but close-rate < 20% → adjust offer/ICP and add proof.

Weekly review cadence (30 minutes, every Monday)
1) Pull metrics by channel and campaign name
2) Identify bottlenecks (reply rate vs booked calls vs show rate)
3) Reallocate effort/spend to the best “booked calls per week” channel
4) Update scripts (best-performing subject lines, proposals, gig thumbnails)
