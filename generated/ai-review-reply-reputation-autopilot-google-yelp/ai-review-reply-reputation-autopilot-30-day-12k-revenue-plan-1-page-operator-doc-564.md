## (3) Simple fulfillment scope (week-1 deliverable)
### Review sources
- Google Business Profile reviews
- Yelp reviews

### SLA (service level)
- DFY Growth: respond within **24 business hours**
- DFY Pro: **12 business hours**
- Agency Starter: **48 business hours**
- Agency Pro: **24 business hours**

### Escalation rules (negative review handling)
Escalate (notify client + create ticket) when any review matches:
- Rating **1–2 stars** (always)
- 3-star with negative sentiment keywords: “refund”, “rude”, “scam”, “lawsuit”, “HIPAA”, “unsafe”, “mold”, “injury”, “fraud”, “police”, “attorney”, etc.
- Mentions of protected health/legal/financial claims

Escalation ticket includes:
- Review text + platform + date
- Proposed response (public) + proposed offline action
- Required owner input fields (what happened? offer? contact person?)

### Posting method (no API dependency)
**Default (preferred):**
- Client grants **Google Business Profile manager access** (or location manager) so we can post responses natively.

**Yelp constraint workaround (week 1):**
- If Yelp access is available via Business Owner access: we post.
- If not: we use **client-post workflow**: we draft responses and send “copy/paste” blocks in a daily email.

### Human-in-the-loop QA (brand-safe)
- We draft responses using the client’s voice guide.
- A human QA pass checks:
  - No admissions of liability
  - No incentives for reviews
  - No protected health info
  - No competitor mentions
  - No aggressive language
- Auto-approval policy:
  - **4–5 stars**: can auto-post unless flagged keywords present
  - **1–3 stars**: require client approval OR escalation rules apply

### Weekly KPI report (every Monday)
- New reviews (Google/Yelp)
- Average rating change (7/30-day)
- Response rate %
- Median response time
- Negative review count + escalations opened
- Resolved escalations count
- Top 3 recurring themes (service, wait time, pricing, staff)
