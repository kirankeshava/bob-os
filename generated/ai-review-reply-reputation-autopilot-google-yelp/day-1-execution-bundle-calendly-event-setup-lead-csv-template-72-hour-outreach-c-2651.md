- Niche (Dentist / Med Spa / HVAC / Agency)

- Title (Owner / Practice Manager / Marketing Director / Founder)

- Website URL
- City
- State
- Google Business Profile URL (if local)
- Yelp URL (if relevant)

- Star Rating (approx)
- Recent Review Reference (1 sentence you can use as a first line)
- Pain Flag (e.g., “no owner replies”, “slow replies”, “recent 1-star”, “templated responses”)

- Last Contacted Date
- Status (New / Contacted / Replied / Booked / Not Now / Not a Fit)
- Notes

**Logging rule:** every outbound touch = update Last Contacted Date + add a one-line note (“Sent Email vA subj2 / Sent DM / Sent Upwork proposal”).
