**Type:** plan
**Agent:** marketer
**Created by:** Marketing Agent

---

# Goal
Close the first 3–10 paying local businesses in 30 days using organic outreach first, then only add minimal spend if we hit clear traction thresholds.

# Core Offer (single CTA across all channels)
**CTA:** “Get a free 10-review response audit + 3 ready-to-post replies. If you like the voice, I can set up an autopilot workflow in 48 hours.”

**Starter Package (fixed price):** Review Response Autopilot Setup + 7 days of responses
- Brand voice/tone guide (1 page)
- Escalation rules for 1–2 star reviews
- Draft responses for new reviews (7 days)
- Weekly KPI snapshot (rating trend, response rate/time)

**Monthly Retainer (after starter):** Reply drafting + escalation + weekly KPI email.

# Free Tool Stack (cheapest options)
- **CRM/Tracker:** Google Sheets (pipeline + daily metrics)
- **Booking:** 
  - Option A: Calendly Free (1 event type) + Google Calendar
  - Option B: Google Calendar “Appointment schedule” + share booking link
- **Intake:** Google Form (collect GBP/Yelp links, tone, escalation contact)
- **Email:** Gmail (no domain initially)

# Spend Policy (strict gates)
Default is **$0**. Only spend when these gates are met:
- **Gate 1 (Upwork Connects):** Buy Connects only after you have **≥3 replies** or **≥1 call booked** from Upwork in the last 7 days.
- **Gate 2 (Any promotion/ads):** Only after **≥1 closed deal** and you can point ads to a proven offer + booking link.

# Lean Budget Sheet (30 days)
**Week 1 (Days 1–7): $0**
- No Connects purchases; use existing Connects only
- No Fiverr Promoted Gigs
- No paid tools

**Week 2 (Days 8–14): $0–$25 (optional, only if Gate 1 hit)**
- Upwork Connects small top-up *only if* replies/calls prove the market

**Week 3 (Days 15–21): $0–$25 (optional, only if Gate 1 hit)**
- Same as week 2

**Week 4 (Days 22–30): $0–$50 (optional)**
- Upwork Connects scaled slightly if close rate is healthy

**Total target spend: $0–$100** (instead of $850)

# KPIs (weekly targets)
- Upwork: 5–10 proposals/day (using best-fit jobs only)
- Fiverr: 3 gigs live, response time < 1 hour while online, 5 buyer requests/day (if available)
- Facebook Groups: 3 posts/week + 10–20 DMs/week to engaged commenters
- Pipeline: 5 calls/week target by week 2

# Daily Operating Routine (45–75 minutes)
## Upwork (25–40 min)
1) Check saved searches (GBP, Yelp, reputation mgmt, review responses, local SEO).
2) Apply to 5–10 jobs that match:
   - Local business / agency managing local clients
   - Mentions reviews, reputation, GBP, Yelp, customer feedback
   - Clear pain (bad reviews, no time, low response rate)
3) Use the niche proposal template + add **one custom audit bullet** from their listing.
4) Log each proposal in Google Sheets (date, job URL, niche, budget, status).
5) Follow-ups: T+24h and T+72h for unanswered proposals.

## Fiverr (10–15 min)
1) Refresh availability/online status during working hours.
2) Improve FAQ answers based on any new inquiries.
3) If Buyer Requests exists: send 5 offers/day (tight scope, fast turnaround).

## Facebook Groups (10–20 min)
1) Comment value-first on 5 posts (no pitching).
2) Post 1 audit-style post 3x/week.
3) DM only people who comment/like asking for the audit.

# Weekly Cadence
- Monday: Post an audit in 1 group + send 10 DMs to engagers
- Tue/Wed/Thu: Upwork proposals daily + 1 group post mid-week
- Friday: “Weekly reputation KPI” educational post + soft CTA
- Sunday: Update tracker, refine openers based on replies

# 10-Review Audit Deliverable (what you send in DM/email)
Deliver in 10 minutes:
1) 3 missed response opportunities (paste the review snippets)
2) 3 ready-to-post replies in a brand-safe tone
3) 1 escalation rule (what to do next time a 1-star arrives)
4) Offer: “If you want, I’ll set up the autopilot + handle replies for 7 days.”

# Owner Execution Checklist (go live)
1) Choose booking option (Calendly Free or Google Appointment Schedule).
2) Create Google Form intake with fields:
   - Business name
   - GBP link
   - Yelp link (if any)
   - Industry
   - Locations
   - Preferred tone (friendly/professional/concise)
   - Escalation contact (email/phone)
3) Publish Fiverr gigs (use existing copy; **no Promoted Gigs**).
4) Set Upwork saved searches and start bidding using existing Connects.
5) Join 10–15 FB owner groups and post first 3 audits this week.

# Spend Gates Recap (simple)
- No spend until you see conversations.
- Small Upwork Connects top-up only after measurable replies/calls.
- No paid promotion until 1 closed deal confirms the offer converts.
