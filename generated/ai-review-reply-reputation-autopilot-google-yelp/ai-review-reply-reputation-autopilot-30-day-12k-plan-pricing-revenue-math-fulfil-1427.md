# 4) Fulfillment scope + SOP (week-1 deliverable; no API dependency)

## Review sources
- Google Business Profile (GBP)
- Yelp Business

## Access & posting methods (deliverable immediately)
1) **Preferred**: client adds us as manager on GBP; provides Yelp owner access where possible.
2) **Fallback (default allowed)**: **client-post workflow**
   - We draft responses in a shared doc/portal + send daily “ready-to-post” batch email.
   - Client posts within agreed window; we track compliance.

## SLA & working cadence
- Check for new reviews: **daily** (Mon–Fri; weekends optional add-on)
- Draft response time: **≤24h (Growth/Agency Starter)**, **≤12h (Pro/Agency Pro)**
- Weekly KPI report: sent **every Monday** for prior week.

## Response rules (brand-safe)
- Always: thank reviewer, reflect key detail, concise, no defensiveness, invite offline resolution when needed.
- Never: mention HIPAA/PHI (health), disclose private info, argue facts, request deletion, incentivize, “review gating.”
- Include business signature line approved in onboarding.

## Escalation rules (1–3 star + risk)
Escalate if any of:
- 1–3 stars
- Mentions: fraud, lawsuit, injury, discrimination, health outcome, harassment
- Staff named negatively
- Requests refund/chargeback

Escalation workflow:
1) Create escalation ticket (template below)
2) Notify owner/manager (email + optional SMS if provided)
3) Draft public reply + private outreach script
4) Require approval before posting
5) Track status (Open → Contacted → Resolved/Unresolved)

## Weekly KPI report (minimum viable)
- New reviews (Google/Yelp)
- Average rating (current + delta)
- Response rate %
- Median response time
- Negative reviews count + escalation count
- “Top 3 themes” (what customers praise/complain about)
- Next-week recommendations (1–3 bullets)
