FULFILLMENT SCOPE (WEEK 1 DELIVERABLE, NO API DEPENDENCY)
Review sources supported now:
- Google Business Profile (GBP)
- Yelp

Posting method (week 1):
1) Preferred: Client adds us as Manager on GBP + grants Yelp Business Owner access.
2) Fallback (no access): We draft responses + send a daily “post-ready” list; client copies/pastes. We track compliance and response-time KPI from timestamps/screenshots.

Core workflow (daily):
1) Capture: pull new reviews from GBP/Yelp notifications or manual check (daily cadence).
2) Draft: generate 1–2 response options using brand voice profile + banned phrase list.
3) QA: human-in-the-loop check for tone, policy, medical/legal claims, incentives, privacy.
4) Approval rules:
   - Auto-approve 4–5★ unless flagged keywords are present.
   - Require client approval for 1–3★, threats/legal, medical/financial claims, discrimination/hate, doxxing, refund demands.
5) Post: post directly (if access) OR deliver copy to client (fallback).
6) Escalation ticket (for negative reviews): open ticket with required fields (review link, customer name if known, issue theme, recommended next step, owner contact).
7) Weekly KPI report (every Monday): rating trend, new review count, response rate, median response time, negative review count, escalations opened/closed, top themes, recommended actions.

Escalation rules (negative reviews):
- Trigger: 1–3★ OR contains keywords (refund, lawsuit, fraud, scam, injury, discrimination, HIPAA/PHI, police, BBB, attorney).
- SLA: escalation sent within 4 business hours (Pro) / 8 business hours (Growth).
- Outcome tracking: status = Open / Waiting on owner / Resolved / Public follow-up posted.

Explicit constraints / compliance:
- No review gating (do not selectively request reviews only from happy customers).
- No incentivizing reviews (money/discounts for reviews).
- We do not promise removal of reviews.
- Disputes/flags submitted only with written client authorization.
- For sensitive verticals (medical/legal/financial): default to higher approval threshold.
