4) LEAD COLLECTION SOP (no tools required)
Goal: 100 leads in 2 hours, then send 30/day.
Pick ONE city + ONE niche (start with high LTV + review sensitivity): dentists, med spas, HVAC/plumbing, law firms.
Steps:
1. Google Maps search: “{{city}} {{niche}}” → open top 50 results.
2. Capture into a sheet: Business name, owner/manager name if visible, phone, website, Google rating, review count, last review date, email (from website/contact page), GBP link.
3. Prioritize:
- rating 3.5–4.6 (most upside)
- recent reviews in last 14 days
- low response rate (visible unanswered reviews)
4. Send sequence: cold email variant A → follow-up #1 (48h) → follow-up #2 (day 5).

5) INTERNAL FULFILLMENT SOP (pilot delivery)
Inputs: forwarded review notification email(s) → agent_bob_replit+review-bot@agentmail.to
Process per review:
1. Log: client + location + platform + star rating + review text + timestamp.
2. Classify sentiment (positive/neutral/negative) + detect escalation keywords.
3. Draft response (brand voice + compliance guardrails):
- Positive: thank + specific reference + invite back
- Neutral: thank + address detail + invite offline if needed
- Negative: apologize + take offline + no admissions of liability; escalate if keywords/<3 stars.
4. Send to client:
- If approval required: email draft + ask for “APPROVE” or edits; include link-out posting instructions.
- If no approval: send suggested reply + link-out.
5. SLA/escalation:
- If <3 stars, strong negative, or keywords (refund/legal/health/safety/injury): email escalation contact immediately.
- If unhandled > SLA hours: reminder email.
6. Weekly KPI email (every 7 days):
- New reviews by platform/location
- Average rating (current vs last week)
- % responded within SLA
- Median response time
- # negative reviews + # escalations + top recurring themes
