## 5) CRM Spreadsheet Schema (Google Sheets-ready)
**Tabs:** Leads | Pipeline | Metrics

**Leads columns:**
- Date Added
- Practice Name
- Location (City, State)
- Website
- Phone
- Contact Name
- Role
- Email
- Source (Google Maps / Website / Referral / LinkedIn)
- ICP Fit (High/Med/Low)
- Notes

**Pipeline columns:**
- Practice Name (data validation from Leads)
- Stage (Prospecting / Contacted / Interested / Demo Booked / Demo Done / Closed Won / Closed Lost)
- Last Touch Date
- Next Step Date
- Next Step (Call/Email/DM/Demo)
- Pain (No-shows / late cancels / hygiene gaps)
- Locations Count
- Pricing Quoted
- Setup Fee (Y/N)
- Expected Close Date
- Outcome Notes

**Metrics tab (simple formulas):**
- Total leads added (COUNT)
- Contacted rate = Contacted+ / Total
- Demo booked rate = Demo Booked+ / Contacted+
- Close rate = Closed Won / Demo Done
- Collected-to-date = SUMIF(Stage=Closed Won, Setup+Month1)

**Collection formula suggestion:**
- Add numeric columns: Setup $ (e.g., 399), Month1 $ (e.g., 399 * Locations Count)
- Collected = Setup $ + Month1 $
