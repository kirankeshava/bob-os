## 4) Cold call script (office manager-first)
**Objective:** book a 15-min demo; do not “sell the whole thing” on the phone.

### Opener (10 seconds)
“Hi, is this the front desk? Could I speak with the office manager for a quick minute?”

If asked what it’s about:
“It’s about reducing last-minute no-shows and filling gaps when patients cancel. Totally operational—should take 30 seconds to explain.”

### Once office manager is on
“Hi [Name]—this is Bob. I’ll be brief. We help dental practices cut no-shows with two-way text confirmations, automated reschedule links, and a waitlist to fill openings.

I’m not calling to pitch a big software switch—this runs alongside what you already do. If it’s useful, we can set it up quickly.

Quick question: when patients don’t confirm or cancel late, does that create same-day holes for you?”

### Discovery (pick 2–3)
1) “About how many patients do you see on a typical day?”
2) “Do you currently text patients to confirm—one-way reminders or can patients reply to confirm/reschedule?”
3) “When someone cancels, do you have a waitlist process that consistently fills that slot?”

### Micro-ROI framing (15 seconds)
“Even recovering **1–2 appointments per week** usually covers this many times over—especially if you’re losing hygiene blocks or production appointments. We also show a simple recovered-revenue dashboard per location so you can quantify it.”

## 5) Objection handling (tight, operational)

### “We already send reminders / our PMS does that.”
“Totally—most do one-way reminders. The difference is **two-way confirmations + automated reschedule + waitlist fill** so you don’t end up with dead air when someone cancels. If your current setup already does two-way + fills cancellations reliably, then you may not need us. Quick question: can patients reply ‘CONFIRM’ or ‘RESCHEDULE’ and does it update your team automatically?”

### “We don’t have a no-show problem.”
“Great—then this should be a quick ‘no.’ Last question: when you get same-day cancellations, do you typically backfill them, or do you still end up with occasional holes? If it’s already near-perfect, I’ll stop here.”

### “No budget.”
“Understood. That’s why we start with a short free run so you can see if it actually recovers appointments. If it doesn’t create measurable value, you don’t keep it. If it does, it’s usually funded by the first couple of recovered visits.”

### “I’m too busy / call back later.”
“I get it. I only need 20 seconds: are last-minute cancellations ever a headache? If yes, I can send the one-pager and you can decide if it’s worth 15 minutes. What’s a better time—tomorrow morning or Thursday afternoon?”

### “Send details.”
