C) DAY-1 OUTBOUND EXECUTION SOP (EMAIL + CALL/TEXT)
Goal for Day-1: 50–100 emails + 20–40 calls + log everything.

1) Daily schedule (example)
- 9:00–10:00 Lead capture / import (or prep next day)
- 10:00–11:15 Call Block #1 (20 calls)
- 11:15–11:30 Log outcomes + set tasks
- 12:30–1:30 Send Email Batch #1 (25–50)
- 3:30–4:30 Call Block #2 (10–20 calls)
- 4:30–5:00 Send Email Batch #2 (25–50) + reply handling

2) Logging minimum standard (speed > perfection)
For every touch, log:
- Touch type (Email/Call/VM/Text)
- Outcome (No answer / Gatekeeper / Connected / Not a fit / Interested / Booked demo)
- Next step date (task)

3) Qualification (ask in <2 minutes)
- Roughly how many appointments per week?
- What’s your no-show rate (even a guess)?
- What’s an average visit worth?
- Who owns scheduling and reminders today?
If any of these are strong (20+ appts/week, no-shows 5–20%+, value/visit $100+, owner/ops controls scheduling), push to demo.
