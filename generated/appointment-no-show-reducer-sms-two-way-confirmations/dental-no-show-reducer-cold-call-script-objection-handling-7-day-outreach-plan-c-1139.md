3) OBJECTION HANDLING (short, crisp)
A) “We already send reminders.”
“Totally. The difference is two-way confirmation + automated reschedule + waitlist fill. Reminders alone still leave you with late cancels that become dead chair time.”

B) “Patients don’t like texts.”
“We only message patients who have opted into SMS, keep it minimal, and the value to the patient is convenience: confirm or reschedule in one reply.”

C) “We’re too busy / no time to set up.”
“That’s exactly why we keep it narrow. We do concierge setup and start with one provider or one location. Goal is to take work off the front desk, not add to it.”

D) “How much is it?”
“For most single locations it’s $399/month per location plus SMS pass-through, and a limited-time concierge setup fee. If it doesn’t measurably reduce no-shows, we’ll make it right—happy to walk through guarantees on a 15-minute demo.”

E) “Send me info.”
“Will do—what email should I send it to? I’ll include the proof page: https://6ff73b98-dcff-4f53-a10d-9ad392f63342-00-1n4h4nuiq6w8j-dbyyfbxp.picard.replit.dev/sites/2 and you can reply to agent_bob_replit+no-show-bot@agentmail.to. When should I follow up—later today or tomorrow?”

“We keep it light to start. Many offices begin via simple appointment exports and a defined daily workflow. If deeper integration is needed later, we can scope it—but the wedge offer is designed to be fast to launch.”

4) 7-DAY DISTRIBUTION EXECUTION PLAN (free-first)
Day 1: Pick 2–3 metro areas; define list criteria; build first 50 leads.
- Criteria: single location or 2–5 locations; general dentistry + high hygiene volume; has online booking or active phone line; appears to have multiple hygienists.
- Titles to target: Office Manager, Practice Manager, Operations Manager, Owner Dentist.

Day 2: Send 50 emails + 20 LinkedIn DMs + 20 calls.
- Goal: 3–5 replies; 1–2 demos booked.

Day 3: Send 50 more emails + 30 calls; follow up to all Day-2 non-responders.
- Goal: 2–3 demos booked.

Day 4: Run demos; send proposals same-day; keep outbound volume.
- Goal: close 1 paid setup.

Day 5: Continue outbound + follow-ups; close 1 more paid setup.

Day 6–7: Weekend admin: tighten script based on objections; publish 1 short LinkedIn post; prep onboarding for new customers.

Daily KPI targets (minimum viable)
- 50 cold emails/day
- 20–30 calls/day
- 20 LinkedIn DMs/day
- 2 demos booked/week per 250 outbound touches (initial target; improve with iteration)
