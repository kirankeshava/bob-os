4) Daily Outbound Checklist + KPI Log (copy/paste each day)
Daily targets:
- Emails sent: 50–100
- Calls placed: 20–40
- Craigslist: 1 post per city cluster (1–2x/week)
- FB groups: 5–10 value comments/posts per week

Execution blocks:
Block 1 (60–90 min): build 20–40 new leads + emails.
Block 2 (60 min): send 25–50 emails + log Status=Emailed, Touch Count +1.
Block 3 (60 min): call 10–20 leads; if no answer, leave voicemail; log outcomes.
Block 4 (45 min): send remaining emails; handle replies; book demos.
