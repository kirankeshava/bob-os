A) TARGET VERTICAL (to speed to 400–800 leads)
Primary ICP: Independent dental + orthodontic practices (1–5 locations)
Decision makers: Owner dentist, practice manager, office manager, operations manager.
Pain signals: online booking, multiple hygienists/chairs, reviews mentioning “waited/late/cancelled,” open hours, “new patient special,” staffing constraints.
Why this vertical: high value per appointment; no-shows directly translate to material lost revenue; often under-automated reminder workflows.

B) LEAD LIST SCHEMA (copy into Google Sheets)
Columns (minimum viable):
1. Business Name
2. Location Name (if multi)
3. Address
4. City
5. State
6. ZIP
7. Phone
8. Website
9. Decision Maker Name
10. Decision Maker Title
11. Email (verified if possible)
12. Source (Google Maps/Yelp/Directory)
13. Notes (booking link? software? hours?)
14. Outreach Stage (see CRM)
15. Last Touch Date
16. Next Touch Date
17. Next Step
18. Outcome (Interested / Not now / Wrong contact / No response)

Lead sources & method:
- Google Maps: search “dentist”, “orthodontist”, “family dentistry”, filter by city/metro; open profiles; capture phone + website.
- Yelp: similar searches; often has owner/manager names in responses; capture categories.
- Practice websites: look for “Meet the Team” + “Contact” + footer emails; sometimes practice manager listed.
- State dental association directories: list practices by county/city; use site contact pages for email.

Daily quota to hit 400–800 in 30 days:
- Target 25–35 new leads/day compiled + 15–25 enriched/day (decision maker + email).
- By day 14 you should have 350–500; by day 30 you should have 700–900.

C) CRM PIPELINE (stages + rules)
Tool: HubSpot Free or a single Google Sheet.

Stages:
0. New (not contacted)
1. Enriched (DM identified + email/phone confirmed)
2. Contacted – Email 1 sent
3. Contacted – Follow-up sequence active
4. Engaged (replied / call connected)
5. Demo Scheduled
6. Trial / Pilot Active
7. Closed Won (paid)
8. Closed Lost

Required fields per record:
- DM Name + Title, Email, Phone, Source, Stage, Last Touch, Next Touch, Notes.

Next-step rules:
- Every lead must always have a Next Touch Date.
- If no reply after Email 3, switch channel: call + voicemail + short email bump.
- If “Not now,” set reminder in 45–60 days and tag reason.
- If “Wrong contact,” immediately request correct contact and try LinkedIn/website team page.

E) COLD EMAIL SETUP CHECKLIST (FREE-FIRST)
Goal: deliverability + basic tracking without paid tools.

1) Domain & inboxes
- Prefer sending from a dedicated domain (e.g., get<brand>.com) with 1–2 inboxes (bob@ / hello@). 
- Free alternative: use existing AgentMail inbox for initial manual outreach; note: deliverability may be lower vs dedicated domain.

2) DNS authentication (once domain exists)
- SPF: authorize sending provider.
- DKIM: enable in provider and publish DKIM TXT.
- DMARC: start with p=none, rua mailbox, then tighten later.
- Add a custom tracking domain only if using a tool; otherwise skip.

3) Warmup plan (manual, free)
Days 1–3: 10 emails/day per inbox (real, personalized) 
Days 4–7: 20/day 
Week 2: 30–40/day
Rules: keep reply rates healthy (ask a simple question), avoid links in early days, keep copy short, plain-text.

4) Tracking (free)
- Use unique subject lines + CRM logging instead of open tracking pixels (pixels can hurt deliverability).
- Use UTM links only after warmup.

5) List hygiene
- Never blast unverified lists.
- Prefer role-based emails only if necessary (office@, info@) but prioritize named DM emails.
