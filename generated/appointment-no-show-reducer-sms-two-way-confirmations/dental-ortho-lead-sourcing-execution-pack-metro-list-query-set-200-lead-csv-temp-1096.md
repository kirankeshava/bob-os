2) Copy/Paste Query Set (Google Maps + Yelp)
Goal: identify independent (1–5 locations) dental/ortho practices with a visible phone number and a website.

Google Maps queries (paste into Google):
- dentist in {city} {state}
- dental clinic in {city} {state}
- family dentistry in {city} {state}
- cosmetic dentist in {city} {state}
- orthodontist in {city} {state}
- braces orthodontist in {city} {state}
- invisalign provider in {city} {state}

Yelp queries (use yelp.com search):
- category: Dentists; location: {city, state}
- category: Orthodontists; location: {city, state}

Filtering heuristics (fast screens):
- Include if: phone is listed AND website exists AND looks like a real clinic (photos/reviews/office hours).
- Prefer if: online booking button exists (often higher no-show sensitivity).
- Deprioritize if: DSO chain / “Corporate Dental” with many locations (unless you want multi-location deals).

3) Decision-Maker Contact Capture Checklist (Owner/Manager email)
From each practice website, check in this order:
A) Contact page: look for admin emails (office@, info@, reception@) and any named manager email.
B) Footer: sometimes lists office manager email.
C) Team/About page: look for “Practice Manager”, “Office Manager”, “Administrator”. Capture name.
D) If no email is listed: use a contact form? Still capture it as “Contact Form URL” and proceed—email can be enriched later.
E) If doctor name is visible: capture doctor full name + credentials (DDS/DMD). Useful for enrichment and personalization.

Common email patterns to try later (do NOT guess-send without verification):
- first@domain.com
- firstlast@domain.com
- drlast@domain.com
- officemanager@domain.com

Recommended dropdown values:

- QA_Status: Unverified, Verified

5) QA Rules (minimum bar before outreach)
Mark a lead “Verified” only if ALL are true:
- Phone number matches across Google Maps and website (or at least appears valid and consistent).
- Website loads and clearly belongs to the practice.
- City/State is correct.
- At least one contact path exists: email OR contact form URL.

Personalization minimum (to increase reply rate):
Add ONE short hook in Notes (5–12 words), e.g.:
- “Has online booking button (likely last-minute cancellations)”
- “Open Saturdays; heavy patient volume likely”
- “Ortho + Invisalign highlighted on homepage”

6) How to execute daily (simple loop)
- Pick 1 metro/day (or 2 if smaller).
- Pull 40–60 leads from Google Maps across 2–3 queries.
- Pull 10–20 additional from Yelp if needed.
- For each lead: capture website + phone + contact email/form; add one personalization note.
- QA check and mark Verified/Unverified.
- Load into CRM and set Next_Action = Email 1.
