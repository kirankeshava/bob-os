A) CRM / IMPORT SHEET COLUMNS (Header Row)
Copy this header row into row 1 of a sheet:
- Company Name
- Vertical (Chiro / Med Spa / Dental)
- City
- State
- Website
- Google Maps URL
- Main Phone
- Owner/Manager Name
- Role (Owner/Manager/Office Manager)
- Email
- Source (Google Maps / Yelp / Directory)
- Cluster (PHX-AZ or DAL-TX)
- Status (New / Emailed / Called / Replied / Demo Booked / Demo Held / Closed Won / Closed Lost / Do Not Contact)
- Last Touch Date
- Next Step
- Next Step Date
- Notes (no-show hints, booking link, #locations, etc.)

Logging rule (minimum viable): Every time you email/call/text, update Status + Last Touch Date + Next Step. Keep it brutally simple so volume stays high.

How to pull leads fast (no paid tools):
1. Open Google Maps.
2. Search the query, open each result, capture Company Name, Website, Phone, and (if site has it) Email + Owner/Manager.
3. If no email on Maps, open website → Contact page. If still none, use a generic front-desk form only if there’s no email (note “no email; use call”).
