## Lead List Build Spec (free tools only)
### What counts as a qualified lead (quick filters)
- Has a working phone number and website
- Books appointments (clear “Request Appointment / Book Online”)
- 5–50 operatories vibe (most general dentistry fits)
- Has a front desk/office manager role (not a solo concierge practice)

### Where to source leads (no spend)
1) **Google Maps** search queries (copy/paste into a sheet)
   - “dentist near me”, “family dentistry”, “general dentistry”, “cosmetic dentist”, “dental clinic”
2) **Yelp** (same categories)
3) **State dental association directories** (often searchable)
4) Practice websites → “Team” / “Contact” pages for office manager names

### Data fields to capture (minimum viable)
- Practice name
- Location (city, state)
- Website
- Phone
- Contact name (Owner or Office Manager if available)
- Role (Owner / Office Manager / Front Desk)
- Email (from site contact page; if none, use contact@domain as placeholder for initial call)
- Notes (hours, #reviews, multi-location indicator)

### 200-lead sourcing workflow (2–3 metros)
1) Choose **3 metros** with dense dental inventory (e.g., Phoenix, Dallas, Tampa)
2) For each metro, collect ~70 practices from Google Maps/Yelp
3) Visit each website; capture office manager name if listed
4) Add to CRM + tag by metro
5) Prioritize multi-location groups first (faster path to $10k)
