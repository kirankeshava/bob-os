D) IMPLEMENTATION SOP (Agency → Client launch)
====================================================

Objective: Launch a client account in 48–72 hours with instant SMS response, qualification, and booking/routing.

1) Client Intake (15–30 min)
Collect:
• Business name, hours, service area/ZIPs
• Service categories (e.g., repair, install, emergency)
• Booking preference: calendar booking link OR “call us” route
• Routing rules (who gets notified: owner, dispatcher, sales)
• Qualifying criteria (must-have ZIPs, job minimums, timeline)
• Compliance: confirm consent language exists on form/lead ad

2) Lead Source Mapping (15–45 min)
Identify where leads come from:
• Website form(s)
• FB Lead Ads
• Landing pages
• CRM (optional)
Ensure each source passes:
• Name
• Phone
• Lead source
• Service requested (if available)

3) Qualification Script Setup (30–60 min)
Use a short, structured script (3–6 questions max).
Guidelines:
• Keep it conversational
• One question per message
• Ask only what changes routing/booking
• End with a clear next step (book now / we’ll call you)

4) Booking/Routing Setup (30–60 min)
Choose ONE primary conversion action:
A) Auto-book appointment
• Use the client’s scheduling link
• Offer 2–3 time windows
B) Route to sales/dispatcher
• Notify by SMS/email
• Include lead answers + click-to-call

5) Status Tagging (15–30 min)
Standard lead statuses:
• New
• Contacted
• Responded
• Qualified
• Unqualified
• Booked
• No response
• Bad number
Define what triggers each status (e.g., “Qualified” after Q3 answered).

6) QA / Test (20–30 min)
Run 5 test leads:
• Normal qualified lead
• Out-of-area lead
• After-hours lead
• No-response lead
• Wrong service lead
Confirm:
• Message timing
• Question logic
• Booking/routing works
• Status tags populate

7) Go Live (5 min)
Turn on for all lead sources.
Notify client:
• What they will receive
• How to respond
• SLA expectations (respond within 5–10 minutes after a qualified handoff)

8) First-Week Optimization (15–30 min)
After 3–7 days:
• Review drop-off question (where leads stop responding)
• Simplify script
• Adjust routing by service/ZIP/urgency
