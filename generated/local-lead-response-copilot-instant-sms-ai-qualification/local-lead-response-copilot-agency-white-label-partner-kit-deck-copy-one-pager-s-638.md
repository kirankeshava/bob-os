That’s +2 extra jobs/month.

C) IMPLEMENTATION SOP (AGENCY → CLIENT) — STEP-BY-STEP
========================================
Goal: Launch each client in 48 hours with a tested qualification + booking flow.

1) CLIENT INTAKE (15 minutes)
Collect:
• Business name, phone, website
• Service types offered (top 3)
• Service area (ZIPs/cities) + any exclusions
• Business hours + after-hours handling
• Booking preference: (A) calendar booking link, or (B) warm transfer to office
• Disqualifiers: out-of-area, renters vs owners, minimum job size, etc.
• Offer/angle: “same-day service,” “free estimate,” etc.

2) LEAD SOURCE CONNECTION (30–60 minutes)
Choose one or more:
• Website form (Webhook/Zapier-style post, or form integration)
• Facebook Lead Ads (via automation connector)
Required data fields (minimum):
• Name
• Phone
• Service requested (if available)
• ZIP/City (if available)

3) NUMBER + MESSAGE BRANDING (10 minutes)
• Provision/assign texting number (or use existing)
• Set business name, tone, and compliance language
• Add “Reply STOP to opt out” line as needed

4) QUALIFICATION SCRIPT SETUP (30 minutes)
Create a short decision tree:
• Q1: Confirm service/issue (buttons where possible)
• Q2: Location (ZIP)
• Q3: Urgency/timeframe
• Q4: Preferred appointment time
Rules:
• If out of service area → tag “Not Service Area” + polite close
• If spam/invalid → tag “Spam/Invalid”
• If qualified → proceed to booking/transfer

5) BOOKING / ROUTING (20 minutes)
Option A — Calendar booking:
• Provide booking link (Calendly/GoHighLevel/etc.)
• Define appointment types + buffer times
Option B — Warm transfer:
• Provide destination phone number(s)
• Define routing hours + fallback message if missed

6) QA CHECKLIST (must pass before launch)
• Test lead submitted → SMS received in <30 seconds
• All questions flow correctly
• Qualified path results in booking link or transfer
• Tags/status appear correctly
• Notifications sent to client/agency
• Opt-out works

7) GO-LIVE + WEEK 1 OPTIMIZATION
• Launch
• Review first 20 conversations
• Update copy to reduce drop-off (shorter questions, clearer options)
• Confirm booked appointments are real + show rate

8) ESCALATION / SUPPORT
• Agency point of contact emails: agent_bob_replit+lead-copilot@agentmail.to
• Priority support (if qualified under minimum commitment) includes: 48-hour setup SLA + weekly tuning.
