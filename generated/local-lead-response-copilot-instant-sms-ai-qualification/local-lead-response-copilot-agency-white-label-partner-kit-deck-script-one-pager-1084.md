## D) Agency Implementation SOP (Step-by-Step)

### Goal
Launch a new client account in 48–72 hours with reliable SMS response, qualification, and booking/routing.

### Roles
- **Agency AM:** client intake, approvals, KPI review
- **Agency Implementer (or us):** build flow, test, go-live
- **Client Owner/Manager:** approves questions, handles hot transfers

### Intake Checklist (must-have)
1) **Business info**
   - Business name, service area, hours
2) **Lead sources**
   - Website forms, FB lead ads, etc.
3) **Routing rules**
   - Who gets notified? (name, phone, email)
   - Hot lead criteria (e.g., “today/tomorrow” jobs)
4) **Booking method**
   - Calendar link / appointment types / hours
5) **Qualification questions (3–6 max)**
   Recommended structure:
   - What service do you need?
   - ZIP code or area?
   - When do you want service? (today/this week/quote)
   - Any key details (size, issue, budget range if relevant)
   - Best email (optional)
6) **Compliance/consent language**
   - Confirm form includes SMS consent (agency responsibility)

### Build Steps
1) Create client workspace/account name
2) Connect lead source(s) to trigger instant SMS
3) Configure qualifying flow + answer capture
4) Configure booking link OR routing notifications
5) Define lead status tags:
   - New
   - Contacted
   - Engaged
   - Qualified
   - Booked
   - Not interested
   - Unreachable
   - Spam/invalid
6) Set business hours behavior:
   - After-hours message + next-day booking option

### QA Test (required)
- Submit 3 test leads:
  1) High-intent (same-day)
  2) Medium-intent (this week)
  3) Low-intent (just browsing)
- Verify:
  - SMS received within 60 seconds
  - Answers recorded
  - Correct status tagging
  - Booking works OR notifications fire

### Go-Live
- Turn on production triggers
- Monitor first 20 leads closely
- Adjust:
  - Question count (reduce friction)
  - Hot-lead thresholds
  - Booking availability

### Common failure modes (and fixes)
- **Low reply rate:** shorten message, reduce questions, add “reply 1/2/3” buttons if supported
- **Bad lead quality:** add ZIP/service filter early
- **Too many hot transfers:** tighten criteria (“today/tomorrow”)
- **Client says ‘leads didn’t work’:** show KPI report + identify breakdown (contact vs qualified vs booked)
