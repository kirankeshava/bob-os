C) IMPLEMENTATION SOP (AGENCY-FRIENDLY)
============================================================

SOP: Onboard a New Client Location in 30–60 Minutes

Goal
Get a location live quickly, with a clean qualification flow and reliable routing.

Inputs you need from the client
1) Business name + service area (cities/zips)
2) Primary services (top 3)
3) Business hours + after-hours handling
4) Booking preference:
   • Call now to office
   • Call to on-call tech
   • Calendar booking link
5) Lead sources:
   • Website form(s)
   • Facebook Lead Ads
   • Google LSA form, etc.
6) Disqualifiers:
   • Out-of-area zips
   • Wrong service types
   • Minimum job size

Step-by-step
1) Connect lead source
   • Add webhook / integration from form tool OR connect FB lead forms.
   • Confirm required fields pass through: name, phone, service requested, zip.

2) Configure phone + branding
   • Assign a dedicated sending number (local area code when possible).
   • Set business display name in message intro.

3) Build the qualification script (keep it short)
   Recommended 3–4 questions:
   Q1: “What service do you need help with?”
   Q2: “Is this urgent (today/24hrs) or flexible?”
   Q3: “What’s your zip code?”
   Q4 (optional): “Can you share a photo?”

4) Routing rules
   • If zip in service area AND service is supported → mark QUALIFIED.
   • If urgent → escalate immediately.
   • If out of area/wrong service → mark DISQUALIFIED with polite close.

5) Escalation & booking
   Choose one:
   • Call transfer: “Want me to connect you now?”
   • Calendar: “Here’s a link to pick a time…”
   • Human handoff: notify CSR via SMS/email with the transcript.

6) Test end-to-end
   • Submit a test lead.
   • Confirm: instant SMS sent, Q flow works, status logged, notifications sent.
   • Fix any missing fields or routing.

7) Go-live checklist
   • Confirm business hours behavior
   • Confirm opt-out language in footer (e.g., “Reply STOP to opt out”)
   • Confirm client understands expectations: fast replies win.

Ongoing optimization (weekly for first month)
• Remove/shorten any question that reduces replies.
• Tighten service/zip rules to reduce junk.
• Adjust booking language to increase conversion.
