## 8) First Prospect Batch (30 targets) — structure + sourcing method
Because we’re operating without paid data tools, use this method to produce 100 rows fast:
- Google: “HVAC {{City}} ‘contact us’”, “plumber {{City}} ‘request service’”, “roofing {{City}} ‘free estimate’”
- Facebook: search local HVAC/plumbing pages → check for Lead Ads / CTA
- Agencies: Google “{{City}} digital marketing agency home services”, Clutch directories, LinkedIn search “lead generation” + city

**Capture these fields per row (copy into your sheet):**
1) Company
2) Niche (HVAC/Plumbing/Roofing/Agency)
3) City/State
4) Website URL
5) Contact page URL
6) Owner/Decision maker name (if visible)
7) Email (if visible)
8) Phone
9) Lead source guess (Web form / FB ads / both)
10) Notes (e.g., “has booking link”, “has FB lead form”, “after-hours risk”)
11) Status (Not Contacted / Contacted / Replied / Demo Set / Closed)

**30-target starter list (to be filled by the operator using the above method):**
- HVAC Owner targets (10): 10 rows using query “HVAC + city” in your top 3 metros.
- Plumbing Owner targets (10): 10 rows using query “plumber + city” in same metros.
- Agency targets (10): 10 rows using query “home services marketing agency + city” and Clutch.

This keeps quality high (high-intent niches) and makes scaling to 100+ straightforward.
