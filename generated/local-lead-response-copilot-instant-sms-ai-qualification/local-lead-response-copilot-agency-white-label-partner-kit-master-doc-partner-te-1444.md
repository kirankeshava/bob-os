## 3) Implementation SOP (Agency → Client Rollout)

### Goal
Launch each client in **24–72 hours**, with consistent QA and measurable KPIs.

### A) Client Intake (15–25 min)
Collect:
1) Business name + location(s)
2) Services offered + service area/zip codes
3) Lead sources (website form, FB Lead Ads, etc.)
4) Hours of operation + after-hours handling preference
5) Booking preference:
   - Option 1: Call booking link (Calendly/CRM)
   - Option 2: Appointment request to office
6) Qualification questions (choose 3–6): job type, urgency, zip, budget range, property type, photo upload link
7) Lead routing contacts (who gets notifications)
8) Compliance note: Confirm customer consent language on forms (SMS consent checkbox recommended)

### B) Build (45–90 min)
1) Configure trigger for new lead submission
2) Create SMS opener (friendly + immediate)
3) Add qualification tree:
   - Keep it short (3–6 questions)
   - Use multiple choice where possible
4) Configure routing:
   - Qualified → booking link or request + notify
   - Unqualified → polite close + tag
   - No response → follow-up sequence
5) Configure status tags

### C) QA (20–30 min)
Run test leads for:
- Business hours vs after-hours
- Qualified path
- Unqualified path
- No-response path
Confirm:
- Median response time is logged
- Status tags apply correctly

### D) Launch (same day)
- Turn on for live leads
- Monitor first 20 leads closely
- Adjust 1–2 questions max (avoid constant changes)

### E) Weekly Optimization (15 min)
- Review top drop-off question
- Reduce friction (shorten question, offer choices)
- Ensure booking handoff is frictionless
