3) IMPLEMENTATION SOP (Agency Onboarding + Launch)
====================================================

Purpose
Launch Local Lead Response Copilot for a new client in <48 hours with consistent QA.

A. Client Intake (15 minutes)
1) Confirm offer + service area + hours of operation.
2) Define qualification rules (examples):
   • ZIP codes served
   • Minimum job size ($)
   • Urgency/timeframe
   • Property type (residential/commercial)
3) Define “Booked” destination:
   • Calendar link OR call transfer OR internal sales rep.
4) Confirm lead sources:
   • Website form(s)
   • FB lead ads
   • Call tracking form submissions (if any)
5) Collect compliance notes (TCPA/consent language present on forms).

B. Setup (30–60 minutes)
1) Create client workspace + tags:
   • New Lead
   • Contacted
   • Qualified
   • Booked
   • No Response
   • Not Qualified
2) Configure lead intake:
   • Connect form/FB webhook (or provide endpoint).
3) Configure the SMS script:
   • Brand name
   • First message within 0–60 seconds
   • 3–6 qualification questions
   • Handoff message to booking
4) Configure notifications:
   • Email/SMS to client team when “Qualified” or “Booked”
5) Configure booking:
   • Calendar link + rules (hours, buffer)

C. Recommended Qualification Question Set (copy/paste)
Q1: “Quick question — what service do you need help with?”
Q2: “What’s the address or ZIP code for the job?”
Q3: “When are you looking to get this done?” (Today/This week/This month/Just researching)
Q4: “Is this residential or commercial?”
Q5 (optional): “Ballpark budget approved for this?” (range)
Q6: “Best email to send confirmation?”
Routing:
• If within service area + timeframe acceptable + job size acceptable → Booked link.
• If not in area OR not a fit → polite decline + tag Not Qualified.
• If no response after N nudges → tag No Response.

D. QA Checklist (10 minutes)
1) Submit 2 test leads (one qualified, one unqualified).
2) Confirm response time < 60 seconds.
3) Confirm tags are correct.
4) Confirm booking works.
5) Confirm client team notifications trigger.

E. Post-Launch Optimization (weekly, 15 minutes)
1) Review: contact/qualified/booked rates.
2) Improve first message and Q2–Q3 wording.
3) Adjust qualification thresholds if volume too low/high.
