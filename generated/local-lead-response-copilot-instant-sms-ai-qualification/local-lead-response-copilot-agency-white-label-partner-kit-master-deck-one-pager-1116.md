D) IMPLEMENTATION SOP (AGENCY OPERATIONS)
====================================================
Objective: Launch each client in 24–72 hours with instant response + qualification + booking.

Roles
- Agency AM: owns client comms + approvals
- Build Operator (you or us): configures flow
- Client Owner/Office: confirms questions, hours, booking rules

Step 1 — Client Intake (15 minutes)
Collect:
1) Business name + service area + hours
2) Primary offer(s): “Water heater replacement,” “same-day pest,” etc.
3) Lead sources:
   - Website form (URL + fields)
   - Facebook Lead Ads (page access or webhook destination)
   - CRM (if applicable)
4) Booking preference:
   - Booking link OR “call office now” number
   - After-hours rules
5) Qualification questions (choose 3–5)
Default question bank (choose 3–5):
- What service do you need help with?
- What city/ZIP is the job in?
- Is this urgent (today/this week/not urgent)?
- Is this for residential or commercial?
- Do you own the property?
- Preferred day/time for an appointment?

Step 2 — Build & Configure (30–60 minutes)
- Write SMS opener (friendly + clear):
  “Hi {{first_name}}—this is {{business_name}}. Got your request. A couple quick questions so we can help fast.”
- Configure branching:
  - If outside service area → polite decline + tag Not Qualified
  - If urgent → notify ASAP + offer booking
  - If qualified → book or hand off to office
- Set lead statuses:
  New → Contacted → Qualified → Booked → Not Qualified → No Response

Step 3 — QA Test (10 minutes)
- Submit a test lead
- Confirm:
  - SMS triggers instantly
  - Questions display correctly
  - Notifications send to correct team
  - Booking link works
- Fix copy/logic issues

Step 4 — Launch
- Turn on routing for live leads
- Inform client what to expect:
  - “You’ll see more live conversations—respond within X minutes if we hand off.”

Step 5 — Week-1 Optimization
- Review first 20–50 leads
- Tune:
  - Question order
  - Qualification thresholds
  - After-hours messaging
  - Office hand-off rules

Step 6 — Monthly Reporting + Renewal Defense
- Send KPI report (template below)
- Highlight wins:
  - Median response time
  - Contact lift
  - Booked appointments
  - Sample conversation screenshots (if allowed)
