# Agency Partner Onboarding Checklist (Internal + Client-Facing)

Use this checklist to standardize every deployment and keep setup time under 30–60 minutes per account.

## A) Pre-Sale Qualification (Partner → Your Prospect)
1. Confirm lead source(s): Website form / FB lead ads / Google LSA / call tracking / other
2. Confirm target response time expectation: “<60 seconds” goal
3. Confirm booking method: Calendar link / receptionist transfer / dispatch system
4. Confirm service type + geo: home services, local high-intent (e.g., HVAC, roofing, plumbing)
5. Confirm volume estimate: leads/day and peak hours

## B) Client Intake (Required Fields)
**Business basics**
- Business name
- Primary contact name + phone
- Service areas (cities/zip)
- Services offered (top 5)
- Business hours + after-hours rules

**Offer & qualification**
- Minimum job size / budget threshold (if any)
- Exclusions (what they don’t do)
- “Good lead” definition (3 bullets)

**Booking & routing**
- Booking link (Calendly/GoHighLevel/etc.) OR
- Call transfer number + rules (who answers; when)
- If appointment required: availability windows

**Lead sources / integrations**
- Website form URL + field names OR webhook details
- FB page + lead form name (if using FB)
- CRM destination (if any): email, webhook, Zapier

**Brand & compliance**
- Brand voice: friendly/professional/direct
- Required disclaimers (if any)
- Opt-out text requirement (default: “Reply STOP to opt out”)

## C) Build (Provider or Partner Ops)
1. Create account profile (business hours, service area, services)
2. Set first-message template (speed-to-lead + name + reason)
3. Configure 3–5 question qualification flow:
   - Location
   - Service needed
   - Timeline
   - Budget/job size (optional)
   - Best time to call / booking prompt
4. Configure routing:
   - If qualified → booking link + notify team
   - If urgent → immediate call handoff
   - If not a fit → polite close + optional referral
5. Set lead status tags: New, Contacted, Qualified, Booked, Not a fit, No response

## D) QA Test (Must Do)
1. Submit 3 test leads (one good, one unclear, one bad)
2. Verify:
   - SMS sends within target time
   - Qualification logic routes correctly
   - Notifications reach correct email/phone
   - Booking link works
   - STOP opt-out works

## E) Launch Day Checklist
1. Confirm “go live” date/time
2. Confirm who monitors replies for first 48 hours
3. Confirm escalation path (if client says “not working”)
4. Provide client a 1-paragraph explanation:
   - What the texts will look like
   - What counts as qualified
   - What to do when a booking occurs

## F) Week 1 Optimization (Fast Iteration)
1. Review KPI snapshot after 48–72 hours:
   - Response time, contact rate, qualified rate
2. Adjust:
   - First message clarity
   - Question order (reduce friction)
   - Booking CTA (earlier vs later)
3. Add “no response” follow-up nudges (1–2 messages)

## G) Monthly Reporting (Partner → End-Client)
Send a short report using the KPI template:
- Median response time
- Contact rate
- Qualified rate
- Booked rate
- Lead status breakdown
- 3 recommendations for next month
