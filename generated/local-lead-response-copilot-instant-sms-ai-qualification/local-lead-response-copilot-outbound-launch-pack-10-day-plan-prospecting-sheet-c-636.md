- LinkedIn URL

- Status (Prospecting / Touched / Replied / Demo Booked / Closed-Won / Closed-Lost)
- Last Touch Date
- Next Action Date
- Next Action

**Where to source (free)**
1) **Google Maps**: search “HVAC near me”, “plumber [city]”, “roofing [city]”. Capture top 20 with websites.
2) **Facebook Ad Library**: search by niche + city; if running lead ads, tag “FB Ads”.
3) **Clutch / UpCity / Google ‘[city] lead generation agency’**: find agencies with “Facebook ads” services.

**Qualification tags**
- “High intent” if: has booking link, mentions “same-day”, heavy review volume, or active ads.
- “Needs speed” if: web form only, no chat, no visible instant response.
