## 4) Implementation SOP (Agency → Client onboarding)

### A) Intake checklist (client-facing)
Collect:
1) Business name + service area
2) Lead sources: website form URL(s), FB lead form(s), landing pages
3) Primary phone for routing + backup
4) Booking method: call scheduling link OR office booking process
5) Qualification questions (3–6 max): job type, address/zip, urgency, budget range (optional), preferred time
6) Compliance notes (hours, opt-out language)

### B) Build steps (internal)
1) Create account + label with agency + client
2) Connect lead source(s)
3) Write SMS opener + 3–6 question flow
4) Define statuses + routing rules
5) Configure booking/call handoff message
6) Add opt-out line (e.g., “Reply STOP to opt out.”)

### C) QA steps
1) Run 5 test leads (different paths)
2) Confirm timestamps captured (for response time)
3) Confirm status changes map correctly
4) Confirm booking links work

### D) Launch checklist
- Turn on live traffic
- Monitor first 20 leads
- Adjust wording for clarity + fewer steps

### E) Ongoing optimization (monthly)
- Review KPI report
- Identify where drop-off happens (contact vs qualified vs booked)
- Tighten first message + reduce friction
